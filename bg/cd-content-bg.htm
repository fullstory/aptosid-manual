Съдържание на Live ISO

sidux only provides Debian dfsg-free software on the live-ISO. За инсталиране на НЕ-свободни приложения и софтуер (firmware)

ISO-образът и изцяло базиран на Debian Sid (към момента 2009-07-xx), обогатен и стабилизиран със собствени пакети и скриптове на sidux. It comes with the latest enhanced sidux kernel, which is based on the most recent vanilla kernel 2.6.3X.xx. ACPI и DMA са включени по подразбиране.

Release ISO variations

sidux offers 5 up-to-date live-ISO entry point release variations to Debian Sid and installing does not usually take longer than 10-20 minutes.

The variations are:
  1. KDE lite 32 bit , live-ISO approximately 600MB: contains a bare minimum set of applications for the KDE4 desktop. Adding more applications to suit your long term requirements is easily done via apt.
  2. KDE lite 64 bit , live-ISO approximately 600MB: contains a bare minimum set of applications for the KDE4 desktop. Adding more applications to suit your long term requirements is easily done via apt.
  3. XFCE 32 bit, live-ISO approximately 500MB: contains a fully featured Desktop Environment (not minimal), and has all the applications you need to be productive immediately. It has a smaller footprint than KDE4.
  4. XFCE 64 bit , live-ISO approximately 500MB: contains a fully featured Desktop Environment (not minimal), and has all the applications you need to be productive immediately. It has a smaller footprint than KDE4.
  5. KDE-full, a combined 32 bit + amd64 bit, live-ISO approximately 2.5GB: contains a full suite of KDE4 applications with 16 languages supported and includes amongst other things, office productivity suites like OpenOffice.org and web browsers like iceweasel (Firefox)

Minimum System requirements for KDE4 and XFCE

AMD64

  • CPU requirements:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • any x86-64/ EM64T capable CPU or newer
    • newer 64 bit capable AMD Sempron and Intel Pentium 4 CPUs (watch for the "lm" flag in /proc/cpuinfo or use infobash -v3).
  • RAM requirements:
    • KDE: ≥512 MB RAM (≥768 MB RAM recommended), ≥1 GB RAM for liveapt.
    • XFCE: ≥256 MB RAM, ≥512 MB RAM for liveapt.
  • VGA graphics card capable of at least 640x480 pixel resolution.
  • Optical disk drive or USB media.
  • ≥3 GB HDD space, ≥10+ GB recommended.

i686

  • CPU requirements:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (not K5/ K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, not C3 Samuel or Ezra)/ C7
    • any x86-64/ EM64T capable CPU or newer
    • the full i686 command set is required.
  • RAM requirements:
    • KDE: ≥384 MB RAM (≥768 MB RAM recommended), ≥512 MB RAM for liveapt.
    • XFCE: ≥192 MB RAM, ≥512 MB RAM for liveapt.
  • VGA graphics card capable of at least 640x480 pixel resolution.
  • Optical disk drive or USB media.
  • ≥3 GB HDD space, ≥10+ GB recommended.

Applications and Tools

Наборът от програми и инструменти в sidux-ISO покрива почти всички ежедневни нужди на десктоп-потребителя.

Choice of 2 major Desktop Managers to access programs/applications:
KDE 4.2.x (en + de), a top of the range desktop manager
Xfce4 4.6.1 (en + de), a medium weight desktop manager.

Fluxbox 1.1.1 (en + de) is included on both variations as a lightweight desktop manager option.

For surfing the Internet, depending on the release variation, there is Konqueror and iceweasel.

For surfing the Internet, depending on the release variation, there is Konqueror and iceweasel.

Office Productivity applications (again depending on the variation and to name only a few), range from Koffice, Openoffice.org, Abiword, Gnumeric through file managers like Dolphin and Thunar.

To configure Networking and Internet connections there is Ceni and for WIFI see our WIFI Roaming documentation.

Non-free driver information is here.

For partitioning a hard drive, we have provided cfdisk and GParted and gparted has an ntfs resize facility.

Diagnostic tools like Memtest86+, an Advanced Memory Diagnostic Tool, are provided.

All ISO variations contain a full set of command line interface tools. A complete manifest of each of the release variants is included on each download mirror: sidux Mirrors, Downloading and Burning.
