Indholdet på Live CD'en

På grund af de juridiske aspekter finder du udelukkende dfsg-free software på Sidux live-CD'en, se venligst dette link for yderligere information om non-free sources for firmware

Denne ISO er fuldstændigt baseret på Debian Sid (up to date as of 2007-02-21) tunet og stabiliseret med sidux' egne paker og scripts. Den kommer med kernel 2.6.20.1, som er baseret på den nyeste vanilla kernel sammen med adskillige patches for forbedret hardware support og tillader initial testing af kvm for pacifica eller vanderpool kompatible CPUer (AMD64 for socket AM2, Intel Core2).

ACPI og DMA er aktiveret som standard (kan deaktiveres med acpi=off og nodma respektivt i grub bootline)

i686 optimeret - ikke til brug med ældre CPUer!

192 MB RAM nødvendig, 256 MB RAM anbefales

Programmer og værktøjer

Sidux-CD'en indeholder en mængde forskellige programmer og værktøjer der skulle dække næsten ethvert dagligdags behov for den almindelige bruger.

KDE 3.5.5 Den grafiske brugerflade, der med et enkelt klik giver dig adgang til programmer og værktøjer

Til at surfe på nettet finder du konqueror en web browser med ekstra feature: indbygget ftp klient og ssh (fish): sjældent set iblandt web browsere, Du finder også iceweasel

Til opsætning af netværk og internet forbindelse finder du ndiswrapper sammen med wpa/wlan og se desuden vores interne WIFI dokumentation.

Hvis du vil partitionere en harddisk, så finder du GParted og også cfdisk. Gparted kan nu også ændre størrelsen på dine nfts-drev!

Diagnostiske værktøjer som Memtest86+ -Advanced Memory Diagnostic Tool
