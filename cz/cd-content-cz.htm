Obsah Live ISO

aptosid poskytuje na Live-ISO pouze svobodný software podle zásad Společenské smlouvy Debianu. Prosím projděte si tento odkaz pro dodatečné informace o nesvobodných zdrojích pro ovladače..

ISO je zcela založeno na Debian sid (aktualizováno k datu 2011-12-31), obohaceno a stabilizováno balíčky a skripty pro aptosid. Je vybaven nejnovějším upraveným jádrem aptosid, které je založeno na nejaktuálnějšm vanilla jádru 3.1.6. ACPI a DMA jsou aktivovány v základu.

Kompletní souhrn každé z vydaných variant je zahrnut na každém zrcadle pro stahování: Zrcadla pro stahování a vypálení.

Vydané varianty ISO

aptosid nabízí 5 aktuálních vstupních variant live-ISO k Debian sid a instalace obvykle netrvá déle než 10-20 minut.

Varianty jsou:
  1. KDE lite 32 bit , live-ISO přibližně 600MB:
    obsahuje minimum aplikací pro prostředí KDE4. Přidání dalších aplikací pro vaše dlouhodobé potřeby se dá jednodušše provést pomocí apt.
  2. KDE lite 64 bit , live-ISO přibližně 600MB:
    obsahuje minimum aplikací pro prostředí KDE4. Přidání dalších aplikací pro vaše dlouhodobé potřeby se dá jednodušše provést pomocí apt.
  3. XFCE 32 bit, live-ISO přibližně 500MB:
    obsahuje plně vybavené prostředí (ne minimální) a také všechny aplikace, které potřebujete, abyste byli ihned plně produktivní. Má menší datovou stopu než KDE4.
  4. XFCE 64 bit , live-ISO přibližně 500MB:
    obsahuje plně vybavené prostředí (ne minimální) a také všechny aplikace, které potřebujete, abyste byli ihned plně produktivní. Má menší datovou stopu než KDE4.
  5. KDE-full, kombinované 32 bit + amd64 bit, live-ISO přibližně 2.5GB:
    obsahuje plnou sestavu KDE4 aplikací s 16 podporovanými jazyky a zahrnuje také další součásti, kancelářský balík jako LibreOffice a internetový prohlížeč iceweasel (Firefox).

Aplikace a nástroje

Minimální systémové požadavky pro KDE4 a XFCE

AMD64

  • Požadavky na CPU:
    • AMD64
    • Intel Core2
    • Intel Atom Series (2xx/ 3xx, N4xx/ D4xx/ D5xx; the N2xx series does not support EM64T)
    • VIA Nano
    • jakýkoliv x86-64/ EM64T umožňující CPU nebo novější
    • novější 64 bit umožňující AMD Sempron a Intel Pentium 4 procesory (zkontrolujte "lm" vlajku (flag) v /proc/cpuinfo nebo použijte infobash -v3
  • Poždavaky na RAM:
    • KDE: ≥512 MB RAM (≥1 GB RAM doporučeno), ≥1 GB RAM pro liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA grafická karta umožňující alespoň rozlišení 640x480.
  • Optická mechanika nebo USB medium.
  • ≥3 GB místa na HDD, ≥10+ GB doporučeno.
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)
    • UEFI+GPT (maximum HDD size <8 ZB)

i686

  • Požadavky na CPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (ne K5/ K6)
    • Intel Atom Series (Z5xx/ Z6xx, N2xx)
    • VIA C3-2 (Nehemiah, ne C3 Samuel nebo Ezra)/ C7
    • jakýkoliv x86-64/ EM64T umožňující CPU nebo novější
    • plná sestava příkazů pro i686 je nezbytná.
  • Požadavky na RAM:
    • KDE: ≥512 MB RAM (≥1 GB RAM doporučeno), ≥1 GB RAM pro liveapt.
    • XFCE: ≥512 MB RAM.
    • With i686, only 3.1 to 3.4 GB RAM is accessible, amd64 (x86-64/ EM64T) is strongly recommended for 64 bit capable hardware.
  • VGA grafická karta umožňující alespoň rozlišení 640x480.
  • Optická mechanika nebo USB medium.
  • ≥3 GB místa na HDD, ≥10+ GB doporučeno.
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)

Aplikace a nástroje

aptosid-ISO zahrnuje rozličné programy a nástroje, které pokrývají téměř všechny potřeby uživatele stolního počítače pro každodenní život.

Choice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE 4.3.x (en + de), spička při výběru správců prostředí
Xfce4 4.6.1.x (en + de), středně náročný správce prostředí.

Fluxbox 1.1.1-x (en + de) je zahrnut v obou variantách jako volba lehkého správce prostředí

Pro prohlížení internetu je k dispozici v závislosti na variantě Konqueror nebo iceweasel.

Office Productivity applications (again depending on the ISO variation range from LibreOffice, Abiword, Gnumeric, through to file managers like Dolphin and Thunar, (to name only a few).

Ko konfiguraci sítě a internetové připojení slouží Ceni a pro nastavení WIFI se podívejte do WiFi Roaming dokumentaci.

Informace o nesvobodných ovladačích najdete zde.

Pro vytvoření a úpravu diskových oddílů nabízíme cfdisk a GParted a gparted nabízí možnost změny velikosti ntfs.

Diagnostické nástroje jako Memtest86+, pokoročilý diagnostický nástroj paměti, jsou také obsaženy.

Všechny varianty ISO obsahují kompletní sestavu nástrojů pro příkazovou řádku. Kompletní přehled variant vydání je obsažen na každém zrcadle pro stahování: Zrcadla pro stahování a vypálení.
