Indholdet på Live CD'en

Af juridiske årsager finder du udelukkende dfsg-fri software på sidux live-CD'en, se venligst dette link for yderligere information om non-free sources for firmware

Denne ISO er fuldstændigt baseret på Debian Sid (opdateret 2009-02-xx) tunet og stabiliseret med sidux egne pakker og scripts. Den leveres med kernel 2.6.28.slh.xx, der er baseret på den seneste vanilla kernel sammen med sidux enhanced patches.

ACPI og DMA er aktiveret som standard.

Systemkrav:

  • amd64:
    • AMD64
      Intel Core2
      nyere 64 bit kapabel AMD Sempron og Intel Pentium 4 CPU'er (kig efte "lm" flaget i /proc/cpuinfo eller brug infobash -v3).
    • ≥256 MB RAM, ≥ 512 MB RAM for liveapt.
    • VGA grafikkort i stand til at yde en opløsning på mindst 640x480 pixel.
    • Optisk disk drev eller USB media.
    • ≥mindst 3 GB harddisk plads, ≥10 GB anbefales.
  • i686:
    • Intel Pentium pro/ Pentium II
      AMD K7 Athlon (ikke K5/ K6)
      VIA C3-2 (Nehemiah, ikke C3 Samuel eller Ezra)/ C7
      enhver x86-64/ EM64T egnet CPU eller nyere; det fuldstændige i686 kommando-set er nødvendigt.
    • ≥192 MB RAM, ≥ 512 MB RAM for liveapt.
    • VGA grafikkort i stand til at yde en opløsning på mindst 640x480 pixel.
    • Optisk disk drev eller USB media.
    • ≥mindst 3 GB harddisk plads, ≥10 GB anbefales.
Programmer og værktøjer

sidux-CD'en indeholder en mængde forskellige programmer og værktøjer der skulle dække næsten ethvert dagligdags behov for den almindelige bruger.

KDE 3.5.10 (en + de) Den grafiske brugerflade, der med et enkelt klik giver dig adgang til programmer og værktøjer

Til at surfe på nettet finder du konqueror; en web browser med ekstra feature: indbygget ftp klient og ssh (sftp): sjældent set iblandt web browsere! Du finder også iceweasel

Til netværk og internettet finder du Kom online med Ceni sammen med wpa/wlan og se desuden vores interne WIFI Roaming dokumentation. Og behovet for non-free drivere omtales her.

Hvis du vil partitionere en harddisk, så finder du GParted og også cfdisk på CD'en. Gparted kan nu også ændre størrelsen på dine nfts-drev!
