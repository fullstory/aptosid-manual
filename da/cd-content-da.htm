Indholdet på Live CD'en

Af juridiske årsager finder du udelukkende dfsg-fri software på Sidux live-CD'en, se venligst dette link for yderligere information om non-free sources for firmware

Denne ISO er fuldstændigt baseret på Debian Sid (opdateret 2008-03-17) tunet og stabiliseret med sidux egne pakker og scripts. Den leveres med kernel 2.6.24.3, der er baseret på den seneste vanilla kernel sammen med sidux enhanced patches.

ACPI og DMA er aktiveret som standard (kan deaktiveres med acpi=off og nodma respektivt i grub bootline)

i686 optimeret - ikke til anvendelse med ældre CPUer! (AMD64, Intel Core2, nyere 64 bit kapabel AMD Sempron og Intel Pentium 4 CPUs (kig efter "lm" parametren i /proc/cpuinfo eller brug infobash -v3) og i686 (Pentium pro/ Pentium II, AMD K7 Athlon eller nyere)

192 MB RAM nødvendig, 256 MB RAM anbefales

Programmer og værktøjer

Sidux-CD'en indeholder en mængde forskellige programmer og værktøjer der skulle dække næsten ethvert dagligdags behov for den almindelige bruger.

KDE 3.5.9 (en + de) Den grafiske brugerflade, der med et enkelt klik giver dig adgang til programmer og værktøjer

Til at surfe på nettet finder du konqueror; en web browser med ekstra feature: indbygget ftp klient og ssh (sftp): sjældent set iblandt web browsere! Du finder også iceweasel

Til opsætning af netværk og etablering af internet forbindelse finder du ndiswrapper sammen med wpa/ wlan, se desuden vores interne WIFI dokumentation. Der er nu også driver understøttelse for mac80211, detaljerne finder du på sidux hjemmeside og her.

Hvis du vil partitionere en harddisk, så finder du GParted og også cfdiskpå CD'en. Gparted kan nu også ændre størrelsen på dine nfts-drev!

Diagnostiske værktøjer som Memtest86+ -Advanced Memory Diagnostic Tool

Complete manifests of each of the release variants are included on each download mirror.
