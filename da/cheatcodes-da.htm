Cheat codes

Når der er mulige værdier angivet i "værdi"-feltet i tabellen, skal disse angives sammen med cheatcode med et = tegn. For eksempel, hvis 1280x1024 er den ønskede cheatcode for skærmopløsning, så skal du skrive; screen=1280x1024 i grub boot prompt'en, eller for xmodules; xmodule=vesa, eller for acpi; acpi=off

Cheatcode Værdi Beskrivelse
acpi off slå ACPI helt fra
oldboot slå ACPI fra, pånær dele nødvendige for boot
force tving ACPI til
apm off power-off
desktop kde vælg windowsmanager
fluxbox
gnome
icewm
xfce
dpi auto or XX indstil ønskede DPI for skærmen (e.g. min 15" 1600x1200 bærbare er 1600/(15*(4/5)) = 133 så dpi=133
failsafe minimal auto detektion
fromiso sti/til/sidux.iso Se venligst booting "fromiso"
hostname hvad-end-dit-hostname-er hostname, ændrer hostname på live-cd systemet
hsync 80 (zB) horisontal skærm-opdateringshastighed (kHz)
irqpoll brug IRQ poll
lang au, be, bg, br, ch, cn, cz, da, de, es, fi, fr, gr, gb, il, hr, ie, ga, it, jp, nl, pl, pt, ru, si, sk, tr, tw, us Indstil dit standard sprog. Det vil også påvirke valget af standard locale, tastatur layout (console og X), tidszone og debian mirror.
Hvis de relevante i18n pakker er installeret vil indstillingerne blive anvendt.
Formern "lang=ll-cc" respekteres, hvor både ll og cc kan have ethvert af de understøttede 2-tegns værdier. "ll" vil blive brugt til sprog indstilling og "cc" til tastatur og tidszone indstilling (for eksempel "lang=fr-be"). Derved kan indstillingerne for sprog og geografisk lokalisation indstilles separat.
md5sum test CD'ens integritet
mem 128MB (f.eks.) anvendt ram størrelse
noacpi deaktiverer acpi (Advanced Configuration and Power Interface)
noagp deaktiverer agp
noapic deaktiverer apic (Advanced Configuration and Power Interface)
noddc deaktiverer automatisk Direct Digital Calibration (forsøger at konfigurere dine skærm-indstillinger)
nodhcp deaktiverer automatisk brug af Dynamic Host Configuration Protocol (forsøger automatisk at sætte din ethernet forbindelse op).
nodma deaktiverer brugen af Direct Memory Access på dine drev
noeject prompt ikke for at udskyde cd'en efter nedlukning
nofloppy deaktiverer floppy
nointro skips visningen af index.html på live desktop startup
noisapnpbios deaktiverer ISA Plug'n Play
nomce ingen meddelelser om irrelevante fejl
noml nulstil alle modelines, og spørg hvilke der skal anvendes
nocpufreq aktiver ikke Speedstep/Powernow
nosmp deaktiverer Symetric Multi Processor understøttelse
notouchpad deaktiverer touchpad understøttelse
nowheel tvinger PS/2 protokol for PS/2 mus
pci noacpi anvend ikke ACPI for PCI enheder
quiet ingen output på skærmen
screen 800x600 indstil skærm opløsningen, 1024x768, 1600x1200 etc
toram kopier CD til RAM og kør derfra
vga normal se efterfølgende liste over vga codes: VGA Cheatcodes
vsync 60 vertikal skærm-opdateringshastighed (Hz)
xmodule ati anvender generisk ATI-driver
fbdev anvender generisk Framebuffer-driver
i810 anvender generisk Intel-driver
mga anvender generisk Matrox-driver
nv anvender generisk Nvidia-driver
radeon anvender generisk ATI-RADEON-driver
savage anvender generisk S3-Savage-driver
vesa anvender generisk Vesa-driver
xhrefresh 80 horisontal skærm-opdateringshastighed (kHz)
xvrefresh 60 vertikal skærm-opdateringshastighed
