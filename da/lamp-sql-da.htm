Mysql Setup

Undersøg om du har alle disse nødvendige linier uncommented (uden # foran) i din apache2.conf (eller i din conf.d/php.ini) fil:

#extension=php_mysql.so

Så det bliver

extension=php_mysql.so

Konfigurationsfilen for mysql findes i: /etc/mysql/my.cnf

Som standard opretter mysql bruger som root og kører uden password. For at ændre root password

mysql> USE mysql;
mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE user='root';
mysql> FLUSH PRIVILEGES;
For at oprette en ny bruger

For at opnå adgang til mySQL databasen fra et PHP script må du aldrig bruge root password, du skal oprette en bruger til dette formål. Alternativt kan du tilføje brugere til mysql databasen ved hjælp af et kontropanel som phpMyAdmin, for nemt at oprette og tilføje database rettigheder til brugere.

For at få adgang til web interface skriver du følgende i din web browser:

http://ipaddress/phpmyadmin/

Når du bliver spurgt om brugernavn og password, skriver du database root som brugernavn og password (I standardopsætningen er der ikke noget password for database root).

http://www.mysql-apache-php.com

http://httpd.apache.org/docs/1.3/misc/security_tips.html
