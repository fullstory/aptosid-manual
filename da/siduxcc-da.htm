sidux Kontrol Centret

sidux kontrol centret er en grafisk brugerflade til at administrere sidux. Du finder den som et selvstændigt program i KDE's hovedmenu->sidux. Den er indtil videre oversat til følgende sprog: Croatian, Danish, Dutch, English, French, German, Greek, Italian, Japanese, Polish, Portuguese, Romanian, Russian and Spanish. It is available in KDE and GTK versions.

siduxcc indeholder 7 moduler med de følgende funktioner:

1. Display modul
  • Ændr opløsning og opdateringsfrekvens
  • Indstil Farvedybde og DPI
  • Konfigurer X-Server
  • Installer grafikkort drivere
Kmenu>System>sidux Control Centre>Display
2. Internet modul
  • Indstil sidux standard web-browser
Kmenu>System>sidux Control Centre>Internet
3. Kernel modul
  • Installer ny kernel
  • Fjern gammel kernel
  • Installer kernel moduler
Kmenu>System>sidux Control Centre>Kernel
4. Netværks modul
  • Konfigurer netværkskort
  • Indstil værtsnavn
  • Indstil Nameservers
  • Installer Wlan-Cards (Ndiswrapper)
  • Installer Wlan-Cards (fwdetect)
Kmenu>System>sidux Control Centre>Network
5. Tjenester modul
  • Administrer almindelige tjenester (Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Samba, SSH...)
  • Start tjenester ved system-opstart
Kmenu>System>sidux Control Centre>Services
6. Informations modul
  • Viser dig information om dit system
Kmenu>System>sidux Control Centre>System info

sidux hermes

sidux-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match. There is also a gtk version.

sidux-hermes-qt4

It is recommended to install the qt4 version. It has the most features and it works on all desktops (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 has options like kernel information and the ability to download files ready to update (dist-upgrade to be carried out in init 3 later). 

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Use the gtk2 version if you want to have an application without qt dependencies.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contains the backend of all hermes versions. It will be automatically installed when you install the qt4 or gtk2 versions.

apt-get install sidux-hermes
