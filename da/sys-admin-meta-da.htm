Meta-Pakker

The meta-packages installer is installed through

apt-get install install-meta

It is available then by going to Kmenu>Utilities>metapackage-installer

The list so far includes:

  • Disk: Scan partition and repair tools
  • Education:
  • Games:
  • Language: Core language packages, OpenOffice.org and iceweseal language packages for Chinese, Japanese, Korean (CJK), Croation, Greek, Polish, and Russian

  • Network: Tor and Privoxy
  • Non_free : various drivers
  • Older_Computer: Lightweight apps and window managers

Meta-Installer og Proxies

Hvis din PC er forbundet til internettet via en HTTP-Proxy, kan du have problemer med at afvikle meta-installeren. meta-installeren skal eksekveres som root.

Du skal ændre dine proxy indstillinger i filen apt.conf. Der er to metoder til dette: Method 1 er bedst, og den er permanent:

Metode 1

Definer proxy'en i apt.conf filen, rediger som root med kwrite /etc/apt/apt.conf og tilføj det følgende:

Acquire{
HTTP::proxy "http://192.168.1.1:3128";
FTP::proxy "http://192.168.1.1:3128";
}

Methode 2

Åbn en konsol og brug sux for at blive root og indstil http_proxy og ftp_proxy miljø variablerne, og start så meta-installer'en på den samme terminal som den hvor du lige har sat proxy variablerne, ellers vil de ike have nogen effekt... Du skal gøre dette hver gang du vil anvende meta-installer'en:

export http_proxy="http://192.168.1.1:3128"
export ftp_proxy="ftp://192.168.1.1:3128/"
