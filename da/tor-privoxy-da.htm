Privoxy

Privoxy er en web proxy med avanceret filteregenskaber til beskyttelse af privatlivet, modificering af web page data, cookie management, adgangskontrol, fjernelse af reklamer, bannere, pop-ups og andet irriterende Internet junk. Privoxy har en meget fleksibel konfiguration og kan tilpasses til den enkelte brugers behov og smag. Privoxy kan anvendes både på stand-alone systemer og på multi-bruger netværk. Privoxy er baseret på Internet Junkbuster (tm).

Installation af privoxy:

apt-get update
apt-get install privoxy

Standard installationen burde udgøre et udemærket sted at starte for de fleste. Der vil utvivlsomt opstå situationer hvor du vil ønske at justere konfigurationen, men det kan du gøre når behovet opstår.

Anvendelse af privoxy og sidux meta-installer se venligst her.

I privoxy konfigurationen kan du få brug for at af-kommentere nogle linier (uncomment), afhængigt af dine præferencer. Dette vil variere fra bruger til bruger.

Tor

Tor er et netværk af virtuelle tunneler der tillader brugere og grupper at øge deres sikkerhed og privacy på internettet. Tor tillader også software udvikler at skabe nye kommunikationsværktøjer med indbygget sikkerhed og privacy. Tor er grundlaget for en række applikationer der tillader organisationer og individer at dele information over det offentlige netværk uden at kompromittere deres privatliv.

Du gøres opmærksom på, at hastigheden kan blive negativt påvirket når du anvender Tor.

For at installere Tor:

apt-get update
apt-get install tor

The default torrc file should work out of the box for most Tor users. You will need to configure Privoxy to use Tor on the internet, see: Tor and internet browsing.

You should also be aware that internet throughput speeds will be negatively affected.

For the Iceweasel brower there is a Torbutton add-on available.

The Tor documentation offers comprehensive on all aspects of Tor.

Vidalia is a controller GUI for the Tor software and allows you to start and stop Tor, view the status of Tor at a glance,and monitor Tor's bandwidth usage.

apt-get update
apt-get install vidalia
