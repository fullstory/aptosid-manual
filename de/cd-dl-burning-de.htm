Die sidux-CD herunterladen

Download-Links und Mirrors findet man unter sidux.com

Mit Bittorrent herunterladen

Bittorrent ist ein Filesharing-Protokoll, das auf der sidux-CD enthalten ist. Es ermöglicht, Dateien gleichzeitig herunter- und für andere hochzuladen. Die Adressen der aktuellen Torrents sind auf sidux.com zu finden.

Den Download eines Torrents kann man in der Konsole zum Beispiel so durchführen:

# btdownloadcurses --max_upload_rate 12 ftp://dummy/sidux/sidux-200xxxxx/sidux-200xxxxx.iso.torrent

In der Konsole sollte das Herunterladen bald beginnen.

Achtung: In dem Verzeichnis, aus dem der Befehl ausgeführt wird, muss genug Platz frei sein, also ca. 700 MB.

Nach dem Beenden des Herunterladens bitte das Programm weiterlaufen lassen, so dass andere die Datei von dir herunterladen können. Das wird unsere Server entlasten und das Herunterladen für andere schneller machen. Eine gute Einleitung zum Thema gibt es hier: Brian's BitTorrent FAQ and Guide.

Bitte nicht vergessen, die MD5-Summe zu prüfen.

Sollten Sie einen FTP-Server mit genügend Traffic zur Verfügung stellen können, können Sie gerne einen unserer Administratoren kontaktieren. Alternativ kann man auch im Forum oder dem IRC-Channel seine Absicht bekannt geben.

MD5-Summe und Integritätsprüfung von heruntergeladenen Dateien

Eine MD5-Summe ist die Check-Summe einer Datei. Diese Check-Summe wird zur Integritätsprüfung der zugehörigen Datei benutzt. Dabei wird die momentane MD5-Summe der Datei mit einer bekannten früheren Summe verglichen. So kann festgestellt werden, ob die Datei verändert oder beschädigt wurde, was bei heruntergeladenen Dateien ratsam ist.

Wenn die MD5-Summe der heruntergeladenen Datei mit der Summe, die in der MD5-Datei angegeben ist, übereinstimmt, ist sicher, dass die Datei korrekt heruntergeladen wurde. Unter Linux erhält man die MD5-Summe einer Datei mit:

$ md5sum zu_prüfende_datei

Das dauert ein wenig und die Summe wird dann auf der Konsole ausgegeben. Mit dem md5summer (486 KB) kann die MD5-Summe auch unter Windows geprüft werden.

Die ISO-Images von sidux werden immer mit der entsprechenden MD5-Summe zum Download angeboten und sollten vor dem Brennen immer geprüft werden. Dies sollte in einem Terminal durchgeführt werden. Man wechselt in das Verzeichnis, in dem sich sowohl die ISO-Datei als auch die MD5-Summe befinden, und kann mit dem Befehl:

$ md5sum -c sidux-Name.iso.md5

die MD5-Summe checken lassen. Wenn die beiden Prüfsummen nicht übereinstimmen, erhält man vom Programm die Meldung:

sidux-Name.iso: Fehlschlag md5sum: Warnung: 1 von 1 berechneten Prüfsumme stimmen nicht überein.

Ist aber die heruntergeladene Datei korrekt, dann endet das Programme ohne jegliche Meldung.

WICHTIGE INFORMATION: sidux, als Linux-LIVE-CD, basiert auf einer hochkomprimierenden Technologie, ursprünglich entwickelt von Knoppix, nun vorliegend in einer weiter entwickelten Form, und aus diesem Grund muss besonders auf die Brennmethode des ISO-Abbildes geachtet werden. Wir empfehlen hochwertige CD-Medien (oder DVD+R), das Brennen im DAO-Modus (disk-at-once) und nicht schneller als achtfach (8x).

Eine Live-CD mit Windows brennen

Selbstverständlich kann die CD auch in Windows gebrannt werden. Die heruntergeladene Datei muss als ISO-Abbilddatei gebrannt werden. Falls Winrar (oder ein anderes Archivierungsprogramm) mit einer ISO-Datei verknüpft ist, könnte dieses Programm die ISO-Datei als eine Archivdatei ansehen. Aus der ISO-Datei muss eine CD gebrannt werden. Die ISO-Datei soll nicht als Daten-CD gebrannt werden!! Und sie muss im Modus Disk-At-Once gebrannt werden.

Es gibt verschiedene gute Optionen, ISO-Dateien in Windows zu brennen.

Open-Source-Brennsoftware für Windows

cdrtfe: kompatibel mit Windows 9x/ME/2000/XP (getestet mit Win95, Win98SE, Win2000, WinXP). Nur für Win9x/ME: funktionierende ASPI-Layer (z. B. Adaptec ASPI 4.60)

Closed-Source- und proprietäre Brennsoftware für Windows
  • Nero (nicht nero express!)
  • CDR-WIN Eine Testversion kann aus dem Internet geladen werden, die aber mit reduzierter Geschwindigkeit brennt, was jedoch für diese Aufgabe ausreichend sein sollte. Im englischsprachigen Menu muss folgender Punkt gewählt werden: "Burn a ISO 9660 Image File to CD".
  • Eine weitere gute Freeware-Brennsoftware ist der CD/DVD Burner XP pro
  • Burncdcc von terabyteunlimited kann nur ISO-Abbilddateien brennen.

Die CD mit Linux brennen

Wer bereits Linux auf dem Rechner hat, kann die CD mit jedem installierten Brennprogramm erstellen. Bei sidux ist K3b das Standard-Brennprogramm. Dort muss man im Menüpunkt "Extras" -> "CD-Abbilddatei brennen..." anklicken, das zu brennende ISO-File (z.B. sidux-Name.iso) auswählen und den Brennmodus DAO (Disk At Once) einstellen.

K3b berechnet zuerst die MD5-Summe des ISO-Files (dauert einen Moment). Stimmt die angezeigte Prüfsumme mit der angegebenen Zeichenfolge der sich im selben Ordner befindlichen MD5-Datei (z.B. sidux-Name.iso.md5) überein, war der Download erfolgreich und die Datei kann mit einem Klick auf "Start" gebrannt werden.

Klickt man auf die berechnete Prüfsumme, erscheint daneben ein Symbol. Klickt man wiederum darauf und gibt in das Feld die Prüfsumme aus der MD5-Datei ein, so werden die beiden Prüfsummen verglichen.

Eine aktuelle sidux-CD unter Verwendung eines XDELTA-Patch herstellen

xdelta ist ein Programm, das unter anderem aus einem ISO-Image und einer Differenzdatei ein neues ISO-Image erstellt. Es ist also wie geschaffen, kleinere Fehlerkorrekturen und Fixes ohne großen Downloadaufwand anzubieten.

Sind zum Beispiel die Differenzdateien von angenommen RC1 nach RC3 (PATCH-RC1-RC2.XDELTA und PATCH-RC2-RC3.XDELTA) nur 16,5 MB bzw. 5,4 MB groß, während das ISO selbst 553 MB groß ist. (Dies ist ein fiktives Beispiel.)

Voraussetzung:

  • ein ISO-Abbild auf der Festplatte, z.B. sidux.iso
  • Eine Differenzdatei, z. B. PATCH-RC2-RC3.XDELTA

Beide Dateien müssen im gleichen Ordner sein.

Anwenden eines Patches auf ein ISO-Abbild als User in der Konsole:

$ cd /pfad/zum/iso
$ xdelta patch PATCH-RC2-RC3.XDELTA

Das Programm extrahiert die notwendigen Dateinamen aus dem PATCH. Es führt auch eine md5sum-Überprüfung durch. Achtung! Es prüft nicht, ob genug Speicherplatz vorhanden ist.

Im folgenden direkten Link findet man ein Programm, um xdelta unter Windows zu nutzen und eine ISO-Datei zu patchen. Hier der direkte Link zum Programm: xdelta.exe. Auf derselben Seite gibt es auch Informationen, allerdings nur auf Englisch.
