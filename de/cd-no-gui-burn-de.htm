Eine CD/DVD ohne GUI brennen

Man benötigt zum Brennen einer CD/DVD nicht notwendigerweise eine grafische Benutzeroberfläche (GUI).

Probleme, die beim Brennen auftreten, haben ihre Ursache normalerweise in den Frontends wie K3b, nicht so häufig in den Backends wie growisofs, wodim oder cdrdao.

Wie finde ich meinen Brenner:

Für ATAPI Geräte:

wodim:

$ wodim -dev=ATAPI -scanbus

....
scsidev: 'ATAPI'
devname: 'ATAPI'
scsibus: -2 target: -2 lun: -2
Warning: Using ATA Packet interface.
....
scsibus0:
0,0,0 0) 'AOPEN ' 'CD-RW CRW2440 ' '2.02' Removable CD-ROM
0,1,0 1) '_NEC ' 'DVD_RW ND-3540A ' '1.01' Removable CD-ROM
0,2,0 2) *

Das heißt, wodim kann ATAPI:0,0,0 oder ATAPI:0,1,0 verwenden. Im folgenden Beispiel wird ATAPI:0,0,0 für CDs verwendet, ATAPI:0,1,0 für DVDs.

cdrdao Geräte-Check:

$ cdrdao scanbus
Cdrdao version 1.2.1 - (C)  Andreas Mueller
ATA:1,0,0 AOPEN , CD-RW CRW2440 , 2.02
ATA:1,1,0 _NEC , DVD_RW ND-3540A , 1.01

Man kann auch die von wodim ausgegeben Spezifikationen für cdrdao verwenden:

ATAPI:0,0,0 oder ATA:1,0,0

Wir werden in unseren Beispielen die wodim-Namen verwenden.

Für SCSI-Geräte:

$ wodim -scanbus

scsibus0:
0,0,0 0) 'RICOH ' 'CD-R/RW MP7060S ' '1.80' Removable CD-ROM
0,1,0 1) *
0,2,0 2) *

Hier wird 0,0,0 für SCSI-Geräte verwendet.

Nützliche Beispiele:

Informationen über leere CDs/DVDs:

$ wodim -dev ATAPI:0,0,0 -atip

oder

$ cdrdao disk-info --device ATAPI:0,0,0$

oder, wie oben angekündigt:

cdrdao disk-info --device ATA:1,0,0 )

Einen wiederbeschreibbaren Rohling löschen:

$ wodim -v -dev ATAPI:0,0,0 blank=fast -force

oder

$ cdrdao blank --device ATAPI:0,0,0 --blank-mode minimal

Eine CD kopieren:

$ cdrdao copy --fast-toc --device ATAPI:0,0,0 --buffers 256 -v2

Eine CD "on the fly" kopieren:

$ cdrdao copy --fast-toc --source-device ATAPI:0,1,0 --device ATAPI:0,0,0 \ --on-the-fly --buffers 256 --eject -v2

Eine Audio-CD mittels wav-Dateien mit 12facher Geschwindigkeit brennen:

$ wodim -v -eject -pad -dao speed=12 dev=ATAPI:0,0,0 defpregap=0 -audio *.wav

Eine CD aus einem bin/cue-Abbild brennen:

$ cdrdao write --speed 24 --device ATAPI:0,0,0 --eject filenam.cue

CD aus einem ISO-Abbild brennen:

$ wodim -v -eject speed=12 dev=ATAPI:0.0.0 driveropts=burnfree -overburn \-ignsize data "iso_File_name"

Eine ISO-Abbildatei aus einem Ordner und allen Unterordnern erstellen:

$ genisoimage -o myImage.iso -r -J -l directory

Man kann growisofs verwenden, um eine DVD zu brennen, z. B. eine ISO-Datei:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso

Mehrere Dateien auf DVD brennen:

$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...

Wenn auf der DVD noch Platz ist, kann man Dateien hinzufügen:

$ growisofs -M /dev/dvd -R -J anotherfile andanotherfile...

Um eine Sitzung zu schließen:

$ growisofs -M /dev/dvd=/dev/zero
