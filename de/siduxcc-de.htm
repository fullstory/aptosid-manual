sidux Kontrollzentrum

Das sidux-Kontrollzentrum (siduxcc) ist ein Frontend zur Administrierung von sidux. Es befindet sich im KDE-Kontrollzentrum (kcontrol). Unterstützte Sprachen sind Dänisch, Deutsch, Englisch, Französisch, Griechisch, Italienisch, Japanisch, Polnisch, Portugiesisch, Kroatisch, Niederländisch, Rumänisch, Russisch und Spanisch. Es ist als KDE- und GTK-Version verfügbar.

Man findet das sidux-Kontrollzentrum im KDE-Menü an diesem Ort:

Kmenu > System > sidux Kontrollzentrum

siduxcc enthält 7 Module mit folgenden Funktionen:

1. Anzeige
  • Auflösung und Frequenz einstellen
  • Farbtiefe und DPI verändern
  • X-Server konfigurieren
  • Grafikkartentreiber installieren
2. Dienste
  • Dienste verwalten
  • Dienst beim Hochfahren starten
3. Internet
  • sidux-Browser: Auswahl des Programms für sidux-Browser
4. Kernel
  • Neuen Kernel installieren
  • Alte Kernel entfernen
  • Kernelmodule installieren
5. Netzwerk
  • Netzwerkkarten konfigurieren
  • Hostnamen festlegen
  • Nameserver festlegen
  • Wlan-Karten installieren (Ndiswrapper)
  • Wlan-Karten installieren (fwdetect)
6. Software
  • Aktualisierbare Pakete
  • Start siduxcc-hermes/meta-installer
7. System Info
  • Zeigt System-Informationen

sidux hermes

sidux-hermes (dist-upgrade-Wächter) ist eine Anwendung im Systemabschnitt der KDE-Kontrollleiste, welche vor Problemen bei einem Dist-Upgrade warnt. Sie scannt auf sidux.com nach dem Schlüsselwort "warning" und zeigt mit einem roten Licht an, falls aktuell eine Warnung ausgesprochen ist. Es gibt auch eine GTK2-Version.

sidux-hermes-qt4

qt4 ist die zur Installation empfohlene Version. Sie besitzt den größten Funktionsumfang und funktioniert mit allen Desktop-Umgebungen (KDE3, KDE4, Gnome, Xfce4).

Mit sidux-hermes-qt4 kann man Informationen über den Kernel abfragen und auch Pakete auf den Computer laden, die zur Aktualisierung anstehen (die Aktualisierung kann zu einem späteren Zeitpunkt im Runlevel 3 durchgeführt werden).

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Die qtk2-Version kann verwendet werden, wenn man keine qt-Abhängigkeiten installieren möchte.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes ist das Kernpaket (Backend) für alle Versionen von Hermes. Es wird automatisch mit der qt4- bzw. gtk2-Version gemeinsam installiert.

apt-get install sidux-hermes
