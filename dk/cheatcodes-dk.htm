Cheat codes

If the possible values are listed in the "Values" field of the table, they must be appended to the "cheatcode" with an = sign. For example, if 1280x1024 was the desired value for the screen cheatcode, then you would type screen=1280x1024 into the grub boot prompt, or for xmodules xmodule=vesa, or for acpi, acpi=off

Cheatcode Værdi Beskrivelse
acpi off slå ACPI helt fra
oldboot slå ACPI fra, pånær dele nødvendige for boot
force tving ACPI til
apm off power-off
desktop kde vælg windowsmanager
fluxbox
gnome
icewm
xfce
dpi auto or XX indstil ønskede DPI for skærmen (e.g. min 15" 1600x1200 bærbare er 1600/(15*(4/5)) = 133 så dpi=133
failsafe minimal auto detektion
fromhd /dev/hda1 (hda2,...) boot fra cd-image på harddisken
fromiso some/path/sidux.iso som fromhd, men anvender sidux iso image. kræver ikke en cd at boote fra, kun den korrekte kernel og initrd/minirt
hostname whatever-your-host-name-is hostname, ændrer hostname på live-cd systemet
hsync 80 (zB) horisontal skærm-opdateringshastighed (kHz)
irqpoll brug IRQ poll
md5sum test CD'ens integritet
mem 128MB (f.eks.) anvendt ram størrelse
noacpi deaktiverer acpi
noagp deaktiverer agp
noapic deaktiverer apic
noaudio deaktiverer lyd

nodma deaktiverer brugen af Direct Memory Access på dine drev
noeject do not prompt to eject the cd on shutdown
nofloppy deaktiverer floppy
nointro skips visningen af index.html på live desktop startup

nomce ingen meddelelser om irrelevante fejl
noml nulstil alle modelines, og spørg hvilke der skal anvendes
nocpufreq aktiver ikke Speedstep/Powernow
nosmp deaktiverer Symetric Multi Processor understøttelse
nosound deaktiverer lyd understøttelse
notouchpad deaktiverer touchpad understøttelse
nowheel tvinger PS/2 protokol for PS/2 mus

screen 800x600 indstil skærm opløsningen, 1024x768, 1600x1200 etc
toram kopier CD til RAM og kør derfra
vga normal se efterfølgende liste vga codes: VGA Cheatcodes
