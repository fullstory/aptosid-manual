Γράφοντας ένα CD/DVD χωρίς γραφικό περιβάλλον (GUI)

ΠΟΛΥ ΣΗΜΑΝΤΙΚΟ: Το sidux, ως ένα Linux LIVE-CD, βασίζεται σε μια τεχνολογία υψηλής συμπίεσης, (την οποία αρχικά εισήγαγε το Knoppix αλλά δεν είναι η ίδια με το Knoppix) και εξ'αιτίας αυτού, χρειάζεται ιδιαίτερη φροντίδα κατά την εγγραφή του ISO image. Χρησιμοποιήστε μόνο CD [ή DVD+RW] υψηλής ποιότητας και γράψτε σε DAO-mode (disk-at-once) και σε ταχύτητα όχι μεγαλύτερη από 8x.

Δεν χρειάζεστε γραφικό περιβάλλον για να γράψετε CD / DVD

τα προβλήματα στην εγγραφή συνήθως εμφανίζονται με το γραφικό frontend (k3b), όχι με τα backends (growisofs, wodim ή cdrdao).

Ποιές συσκευές να χρησιμοποιήσετε

Για ATAPI συσκευές μπορείτε να μάθετε με τα:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/hdc'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/hdd'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Άλλες εναλλακτικές είναι:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

και

$ wodim -prcap

Ακολουθούν μερικά χρήσιμα παραδείγματα:

Πληροφορίες σχετικά με ένα άδειο CD/DVD:
$ wodim dev=/dev/hdc -atip

ή

$ cdrdao disk-info --device ATA:1,0,0
Σβήσιμο ενός επανεγγράψιμου δίσκου:
$ wodim -blank=fast -v dev=/dev/hdc

ή

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Δημιουργία κλώνου (Clone) ενός cd:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Δημιουργία κλώνου (Clone) ενός cd χωρίς την διαμεσολάβηση εικόνας (image) στον σκληρό δίσκο (on the fly):
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Δημιουργία cd ήχου από αρχεία wav σε ταχύτητα 12x:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/hdc defpregap=0 -audio *.wav
Εγγραφή cd από αρχεία bin/cue:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Εγγραφή ISO image:
$ wodim dev=/dev/hdc driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso
Δημιουργία ISO image με όλα τα αρχεία (και υποφακέλους) ενός φακέλου.

Αυτό μπορεί να γίνει με την παραπάνω εντολή (δημιουργία ISO image):

$ genisoimage -o myImage.iso -r -J -l directory

Αν έχετε συσκευή εγγραφής DVD (DVD burner), μπορείτε επίσης να χρησιμοποιήσετε το growisofs για εγγραφή σε DVD, όπως την εγγραφή ενός ISO image σε DVD:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Εγγραφή πολλαπλών αρχείων σε DVD:
$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...
Αν έχει μείνει ελεύθερος χώρος στο DVD, μπορείτε να προσθέσετε εκ των υστέρων περισσότερα αρχεία:
$ growisofs -M /dev/dvd -R -J anotherfile andanotherfile...
Για να "κλείσετε" (finalize) το DVD, χρησιμοποιήστε:
$ growisofs -M /dev/dvd=/dev/zero
