Τα φόρουμ του sidux

Τα φόρουμ του sidux προσφέρουν μια θέση για ανάρτηση ερωτήσεων και για λήψη απαντήσεων στις ερωτήσεις αυτές, παρακαλώ ψάξτε στα φόρουμ πριν ανοίξετε ένα νέο θέμα, δεδομένου ότι άλλοι μπορεί να είχαν υποβάλει ήδη την ίδια ερώτηση. Τα φόρουμ είναι στα Γερμανικά και Αγγλικά.

Το sidux Wiki

Το sidux Wiki είναι ελεύθερο να χρησιμοποιηθεί και να διαμορφωθεί από όλους και για όλους τους χρήστες του sidux.

Ελπίζουμε για συνεισφορές από χρήστες Linux όλων των επιπέδων γνώσης, δεδομένου ότι αυτό το wiki προορίζεται να βοηθήσει τους χρήστες όλων των επιπέδων. Τα λεπτά που δίνετε στο wiki μας μπορούν να γλυτώσουν, άλλους χρήστες (και ίσως τους ίδιους σας του εαυτούς) που έχουν ανάγκη από λύσεις, ώρες προβλημάτων και έρευνας. Το Wiki είναι μεταφρασμένο στα Γερμανικά και στα Αγγλικά.

IRC Πρωτόκολλα συμπεριφοράς

Ποτέ μην χρησιμοποιείτε το IRC σαν διαχειριστής (root). Είστε ευπρόσδεκτοι στη συνομιλία IRC ως χρήστες, ποτέ σαν διαχειριστής: αν έχετε κάποιο πρόβλημα, πείτε το αμέσως.

Konversation

Ο εύκολος τρόπος είναι να επιλέξετε το sidux-irc εικονίδιο επάνω στην επιφάνεια εργασίας ή να χρησιμοποιήσετε το ήδη ρυθμισμένο Konversation στο Kmenu.

Αν προτιμάτε διαφορετικό chat client ρυθμίστε την επιλογή του διακομιστή (server) ως εξής.

irc.oftc.net
port 6667

Ενεργοποιήστε το irc στο sidux.com

Πηγαίνετε στο sidux.com και επιλέξτε το Chat (cgi) ή Chat (java) στην λίστα του μενού. Αυτό θα σας ανοίξει ένα irc chat client μέσα στον περιηγητή (browser) σας ο οποίος θα σας πάει στο #sidux.

Βάλτε το ψευδώνυμο της επιλογής σας στο κουτάκι ψευδώνυμου (nickname) και το #sidux στο κουτάκι του καναλιού (channel) και επιλέξτε 'login' και/ή ακολουθείστε τις οδηγίες

!paste

Επικόλληση Debian (debian paste)

Το !paste θα γίνει ο φίλος σας στο κανάλι, διαφορετικά θα σας διώξουν για flooding. Έτσι εάν καλείστε να χρησιμοποιήσετε !paste, για να δώσετε το περιεχόμενο της κονσόλας σας, γράψτε !paste στο σημείο που συνομιλείτε (ή κάποιος μπορεί να έχει ήδη παράσχει την πληροφορία για να μπορείτε να ενεργοποιήσετε την επικόλληση (paste)), και αυτό θα επιστραφεί:

"paste" είναι 
http://paste.debian.net/:παρακαλώ μην
επιλέξτε κανέναν τονισμό σύνταξης, κάνετε την επικόλλησή (paste)
σας, κατόπιν δώστε τον σύνδεσμο (link) για την επικόλληση σας στο #sidux.

Απλά επιλέγετε τη σύνδεση με ένα κλικ, που θα σας πάει σε έναν ιστό χώρο, επικολλήσετε το περιεχόμενο της κονσόλας που έχει ζητηθεί μέσα στο παράθυρο, κατόπιν πηγαίνοντας πίσω στην κορυφή, του παραθύρου επιλέξτε send.

Έπειτα θα δείτε ότι η είσοδός σας έχει προστεθεί στη βάση δεδομένων με διάφορες επιλογές για να συνδεθεί η επικόλληση με το IRC, που είναι η διεύθυνση που το κανάλι απαιτεί να δώσετε. (Παράδειγμα: Για να συνδεθείτε με την είσοδό (entry) σας: http://paste.debian.net/524). Η επικόλληση (paste) Debian μπορεί να ρυθμιστεί να δείχνει τις πληροφορίες για ένα συγκεκριμένο χρονικό διάστημα (μέχρι 72 ώρες). Έχετε επίσης την επιλογή να διαγράψετε τα paste / στοιχεία.

sidux-paste

Το sidux-paste θα επιτρέψει σε σας να επικολλήσετε την έξοδο (output) αρχείων από το τερματικό και tty, επομένως είναι ιδανικό εάν αντιμετωπίζεται προβλήματα στο init 3. Το sidux-paste χρησιμοποιεί το http://rafb.net για σύνδεσμο και η έξοδος (output) είναι διαθέσιμη για 24 ώρες.

Μπορείτε να είστε ως απλός χρήστης (user) ή διαχειριστής (root) για να επιτραπεί το sidux-paste, εντούτοις, σημειώστε ότι μερικά αιτήματα για την παραγωγή (output) απαιτούν την πρόσβαση διαχειριστή (root).

sidux-paste <file>
ή
sidux-paste -c "command1 --options parameters ; command2"
Παράδειγμα του sidux-paste <αρχείο>
sidux-paste /boot/grub/menu.lst
 Για το paste δείτε εδώ: http://rafb.net/p/n8miQp85.html

Η διεύθυνση http://rafb.net/p/n8miQp85.html είναι αυτό που χρειάζετε να δώσετε στο #sidux στο IRC

Παράδειγμα του sidux-paste -c
sidux-paste -c "fdisk -l; cat /etc/fstab; blkid"

Εργαλεία που χρειάζεστε σε μη γραφικό περιβάλλον (tty) and init 3

Συνήθως είστε σε μη γραφικό περιβάλλον/init 3, επειδή πρόκειται να κάνετε μια dist-upgrade (βελτίωση) ή επειδή κάτι φοβερό έχει πάει στραβά με το σύστημά σας.

gpm

Ένα καλό εργαλείο να έχετε σε μη γραφικό περιβάλον είναι το gpm . Αυτό, θα επιτρέψει σε σας να χρησιμοποιήσετε το ποντίκι σας για να αντιγράφεται και να κολλάτε μεταξύ των τερματικών/κονσόλων.

gpm είναι ήδη ρυθμισμένο στο sidux, αλλιώς κάντε: 

gpm -t imps2 -m /dev/input/mice

Εξασφαλίστε ότι το /etc/gpm.conf είναι υπαρκτό

/etc/init.d/gpm restart

Τώρα είστε σε θέση να έχετε ένα ποντίκι σε μη γραφικό περιβάλλον (tty)

File browsing and text editing

Το Midnight Commander είναι ένας διαχειριστής αρχείων για την γραμμή εντολών and text editor, είναι προ-εγκατεστημένος στο σύστημα και απλός στην χρήση.

Εκτός από τις συνηθισμένες εντολές από το πληκτρολόγιο, το πρόγραμμα ανταποκρίνετε, λόγω του gpm και σε κλικς από το ποντίκι.

Το mc ανοίγει το σύστημα αρχείων ενώ το mcedit ανοίγει ένα κενό αρχείο ή μπορείτε απευθείας να ανοίξετε ένα αρχείο.

Παράδειγμα για το πώς να ανοίξετε απευθείας ένα αρχείο:

mcedit /etc/apt/sources.list.d/debian.list

You can now edit the file to change parameters and save the alteration to have immediate effect.

Διαβάστε:

man mc

IRC Πρωτόκολλα συμπεριφοράς

Ποτέ μην χρησιμοποιείτε το IRC σαν διαχειριστής (root). Είστε ευπρόσδεκτοι στη συνομιλία IRC ως χρήστες, εάν βρίσκεστε επί του προβλήματος, πείτε το αμέσως.

irc σε μη γραφικό περιβάλλον ή/και στο init 3

Η προεπιλογή στο sidux είναι το irssi.

Όταν στο init 3, μπορείτε να μεταπηδήσετε σε έναν άλλο τερματικό/κονσόλα από:

# CTRL-ALT-F2
$ siduxbox login:  <όνομαχρήστη> <password> (όχι ως χρήστης root)
Κατόπιν γράψετε
$ sidux-irc (αυτό ανοίγει το irssi)

ή εάν έχετε εγκαταστήσει έναν άλλο πελάτη (client) όπως το weechat:

# CTRL-ALT-F2
$ siduxbox login:  <όνομαχρήστη> <password> (όχι ως χρήστης root)
Κατόπιν γράψετε
$ weechat-curses

Τώρα θα συνδεθείτε στο irc.oftc.net στο port 6667. Μόλις συνδεθείτε, αλλάξτε το ψευδώνυμό σας:

/nick γράψτε_το_ψευδώνυμο_τις_επιλογής_σας

Για να επισκεφτείτε το κανάλι του sidux

/join #sidux

Εάν επιθυμείτε να συνδεθείτε σε έναν άλλο κεντρικό υπολογιστή (Server):

/server server.όνομα

Στην κατώτατη μπάρα, θα δείτε αριθμούς να εμφανίζονται εάν υπάρχει δραστηριότητα στα κανάλια, για να μεταπηδήσετε σε εκείνο το κανάλι, ALT-1 ALT-2 ALT-3 ALT-4 κτλ..

Για να βγείτε

/exit

Εάν κάνετε μια dist-upgrade βελτίωση, παραδείγματος χάριν, για να ελέγξετε την πρόοδο

για να επιστρέψει στο d-u παραδείγματος χάριν
$ CTRL-ALT-F1
για να γυρίσετε πίσω στο irc
# CTRL-ALT-F2

Infobash

Το infobash είναι ένα script πληροφοριών συστήματος, το οποίο είναι ανεξάρτητο από κάποιο συγκεκριμένο πελάτη IRC. Το script πληροφοριών συστήματος μπορεί να επιδείξει όλα τα είδη πληροφοριών που αφορούν το υλικό και το λογισμικό σας στους ανθρώπους στο κανάλι, έτσι μπορούν να σας βοηθήσουν να εντοπίσετε τα προβλήματα ή να θαυμάσουν τις προδιαγραφές του συστήματος σας και την έκδοση πυρήνα, όποτε θέλετε εσείς μέσα από την κονσόλα.

Για να χρησιμοποιήσετε το infobash script με το Konversation, γράψτε μέσα στο chat box:

/cmd infobash -v3

Για να χρησιμοποιήσετε το infobash script με άλλους πελάτες (clients) γράψτε μέσα στο the chat box:

/exec -ο infobash -v3
ή
/infobash -v3

Γράψετε σε μια κονσόλα:

infobash -v3

Διαβάστε το πλήρες σεμινάριο (tutorial) του infobash

man infobash
