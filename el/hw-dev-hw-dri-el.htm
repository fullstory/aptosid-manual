  • sgfxi / smxi

    Χρησιμοποιήστε το πολύ-εργαλείο διαχείρισης συστήματος που δημιουργήθηκε με την συμβολή της κοινότητας sidux, smxi/sgfxi, που είναι συντομογραφία του sidux-maintenance. Για να διαβάσετε περισσότερα αναφερθείτε στο smxi και sxfxi Σύνταξη και σετ εντολών.

    1. Οι ανοικτού κώδικα Xorg οδηγοί για nVidia, ATI, Radeon, Intel, είναι προ-εγκατεστημένοι στο sidux. Αν τους έχετε αντικαταστήσει με κλειστού κώδικα οδηγούς μπορείτε να επιστρέψετε (revert) ή να επανέλθετε (roll back) σε αυτούς με το sgfxi

      Οι οδηγοί για τον X. Org X διακομιστή (server) (δείτε xserver-xorg για περαιτέρω περιγραφή) εξασφαλίζουν υποστήριξη σε 2d για NVIDIA Riva, TNT, GeForce και Quadro κάρτες και των ATI Mach, Rage, Radeon και FireGLcards παράλληλα με atimisc 'r128' και 'radeon' υπό-οδηγούς. Επίσης εξασφαλίζει υποστήριξη για τους ελεύθερους οδηγούς 3d της Intel.

      Για να τους εγκαταστήσετε:

συνδεθείτε ως (υπερχρήστης) root
και μετά γράψτε init 3

    Συμβουλές για υλικό με μη-ελεύθερες (non-free) ανάγκες

    Λόγω της περιπλοκότητας του νόμου, το sidux live-cd παρέχει μόνο dfsg-ελεύθερο λογισμικό.

    Λίστα πηγών (sources list)

    Επιδή το sidux iso περιέχει μόνο dfsg ελεύθερο λογισμικό, εσείς πιθανών θα θέλετε να προσθέσετε contrib/ non-free (μή ελεύθερο λογισμικό) στο /etc/apt/sources.list.d/debian.list ή στο /etc/apt/sources.list Να είστε σίγουροι ότι είστε συνδεμένοι στο διαδίκτυο.

    # See sources.list(5) for more information, especially
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.
# The /etc/apt/sources.list.d/ directory provides a way to add sources.list
# entries in separate files that end with sources.list file.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
    Κάρτες γραφικών
    • ATi Radeon γραφικά: 3d επιτάχυνση για παλιότερες κάρτες μέχρι και την r35x θα πρέπει να δουλέψει, νεότερες Radeon X1xxx κάρτες χρειάζονται μη ελεύθερους οδηγούς για την επιτάχυνση της απόδοσης η οποία δεν είναι ακόμα συμβατά με το X.org 7.2. Αναλόγως του chipset "xmodule=vesa" ή "xmodule=fbdev" σαν κλεπτοκώδικας (cheatcode) μπορεί να χρειαστεί για νεότερα Radeon X1xxx μοντέλα.
    Firmware
    • Για βοήθεια αναγνώρισης του firmware που πρέπει να εγκαταστήσετε, τρέξτε σαν χρήστης:
    $ fw-detect

    Αυτό θα σας δώσει μια λίστα των συνιστώμενων και τις εντολές που πρέπει να εκτελέσετε. Θα επιστρέψει 'null' (τίποτα) αν δεν χρειάζεστε κάτι ή αν το firmware είναι ήδη εγκατεστημένο.

    • Atheros/ "madwifi" wlan: 
      m-a a-i madwifi
    • Atmel AT76c50x 11 MBit/s wlan: 
      apt-get install atmel-firmware
    • Broadcom, b43/ b43 legacy wlan: 
      apt-get install b43-fwcutter
    • Broadcom Blutonium Bluetooth chipset (BCM2033): 
      apt-get install bluez-firmware
    • Eagle USB ADSL modem: πάρτε το firmware από http://eagle-usb.org/ueagle-atm/non-free/ και τοποθετείστε το στο /lib/firmware/.
    • DVB firmwares για παραλλαγή ολοκληρωμένου χαρακτηριστικού DVB TV cards (ως επί το πλείστον οι κάρτες δεν το χρειάζονται αυτό): πάρτε το χρήσιμο σε σας firmeware (ελέγξτε dmesg) από http://www.linuxtv.org/downloads/firmware/ και τοποθετείστε το στο /lib/firmware/.
    • IVTV firmware:πάρτε το firmware από http://dl.ivtvdriver.org/ivtv/firmware/ και τοποθετείστε το στο/lib/firmware/.
    • Intel ipw2100 and ipw2200, 11/54 MBit/s wlan: 
      apt-get install firmware-ipw2x00
(obsolete ipw2100-firmware and ipw2200-firmware ought to be purged)
    • Intel ipw3945 (54 Mbit/s wlan)/ ipw 4965 (draft-n) iwlwifi, 54 Mbit/s wlan Παρακαλώ διαγράψτε τα iwlwifi-ucode, iwlwifi-3945, firmware-iwl3945 και firmware-iwl4965, εάν είναι ήδη εγκατεστημένα:.: 
      apt-get install firmware-iwlwifi
    • Intel ipw4965/ iwlwifi, draft-n wlan: 
      apt-get install firmware-iwlwifi
    • Intersil prism54, 54 MBit/s wlan: πάρτε το firmware από http://prism54.org/firmware/ και τοποθετείστε το στο /lib/firmware/.
    • nVidia γραφικά: η 3D επιτάχυνση δεν είναι δυνατή με ελεύθερους οδηγούς ακόμα, παρακαλώ χρησιμοποιήστε get-sidux-binary-gfx ή sgfxi για να πάρετε scripts εγκατάστασης για αυτές τις κάρτες.
    • RaLink rt61 και rt73 54 MBit/s wlan,: 
      apt-get install firmware-ralink

      Παρακαλώ διαγράψτε τα rt61-fwcutter, rt73-fwcutter, firmware-rt61 and firmware-rt73, εάν είναι ήδη εγκατεστημένα

    • Ricoh R5U870 OEM webcams:Πάρτε το firmware από http://mediati.org/r5u870/r5u870-0.11.0.tar.gz και αντιγράψτε τα r5u870_*.fw αρχεία στο /lib/firmware/.
    • Texas Instruments ACX100 (22 Mbit/s)/ ACX111 (54 MBit/s) wlan: 
      m-a a-i acx100
      Πάρτε το firmware από http://www.kazer.org/acx-firmware-20060207.tar.bz2 και τοποθετείστε το στο /lib/firmware/.
    • ZyDAS zd1201 11 MBit/s wlan: Πάρτε το firmware από http://surfnet.dl.sourceforge.net/sourceforge/linux-lc100020/zd1201-0.14-fw.tar.gz και τοποθετείστε το στο /lib/firmware/.
    • ZyDAS zd1211 54 MBit/s wlan: 
      apt-get install zd1211-firmware

    Δοκιμάζουμε να εξασφαλίσουμε πακέτα για νομική επαναδιανομή του firmware από δικές μας μη-ελεύθερες πηγές, αλλά δεν το επιτρέπουν όλοι οι προμηθευτές αυτό.

    Γράφοντας στην NTFS-κατάτμηση με ntfs-3g

    Προειδοποίηση: Αν και το ntfs-3g έχει δηλωθεί να είναι ‘σταθερό’ ποτέ μην το χρησιμοποιήσετε χωρίς να έχετε σώσει τα αρχεία σας σε εξωτερικό μέσο, και φυσικά ποτέ συστήματα παραγωγής! Αν το κάνετε, θα είναι δικό σας σφάλμα αν χάσετε τα δεδομένα σας, για αυτό χρησιμοποιήστε το με δικιά σας ευθύνη

    Ανοίξτε ένα τερματικό και γράψτε τις ακόλουθες εντολές: Βλέπετε Κάνοντας κατατμήσης στον σκληρό σας δίσκο – Ονομασίες δίσκων

    sux
apt-get update && apt-get install ntfs-3g
umount /media/xdxx
mount -t ntfs-3g /dev/disk/by-uuid/xxyyzz[etc] /media/xdxx
Για να βγείτε έξω από την κονσόλα γράψτε: exit

    Τώρα η NTFS-κατάτμηση θα πρέπει να είναι προσαρτημένη (mounted) rw και θα πρέπει να μπορείτε να αποθηκεύσετε τα δεδομένα σας σε αυτή. Αλλά ξανά προ ειδοποιήστε Χρησιμοποιήστε το σε επείγουσες καταστάσεις, δεν συνιστάται για χρήση επί καθημερινής βάσεως.
