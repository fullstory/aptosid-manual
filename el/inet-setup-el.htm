Ρύθμιση WiFi Roaming με wpa

Πιθανόν να χρειαστείτε μη-ελεύθερο firmware να είναι υπαρκτό πάνω σε ένα USB-stick για να το εγκαταστήσετε στο λειτουργικό σύστημα. Παρακαλώ δείτε μη-ελεύθερα firmware debs πάνω σε ένα stick.

Γενικά

Το wpa-roaming είναι μια μέθοδος με την οποία μπορείτε να περιηγήσετε και να συνδεθείτε με ασύρματα δίκτυα με ή και χωρίς γραφικό περιβάλλον.

Με τις πιο κάτω ρυθμίσεις, όταν δεν είναι συνδεδεμένο ένα σύρμα ethernet, η wlan0 κάρτα σας συνδέει με το ασύρματο δίκτυο της επιλογής σας ή με ένα διαθέσιμο ανοικτό ασύρματο δίκτυο ή με ένα προκαθορισμένο ασύρματο δίκτυο. Εάν συνδέσετε ένα σύρμα ethernet, το ενσύρματο δίκτυο σταματάει την πρόσβαση στο ασύρματο δίκτυο και η eth0 κάρτα σας συνδέει με το ενσύρματο δίκτυο. Με την αποσύνδεση του σύρματος το ασύρματο δίκτυο θα είναι και πάλι διαθέσιμο.

Ρύθμιση του δικτύου

Σαν root κάντε τις κατάλληλες αλλαγές στο /etc/network/interfaces αρχείο για να μοιάζει όπως το πιο κάτω (τα ονόματα των διεπαφών μπορεί να είναι διαφορετικά):

# The loopback network interface
auto lo
iface lo inet loopback


#Added by user
allow-hotplug wlan0
iface wlan0 inet manual
  wpa-roam /etc/wpa_supplicant/wpa-roam.conf

#this line must always be here
iface default inet dhcp

Τώρα χρειάζεστε ένα wpa-roam.conf για το wpa_supplicant

cp /usr/share/doc/wpasupplicant/examples/wpa-roam.conf /etc/wpa_supplicant/wpa-roam.conf

Χρησιμοποιήστε ένα επεξεργαστή κειμένου για να ανοίξετε το αρχείο

<editor> /etc/wpa_supplicant/wpa-roam.conf

Αφαιρέστε το # από την γραμμή 30. Αυτό πρέπει να το κάνετε γιατί αλλιώς οι ρυθμίσεις δεν θα φυλάγονται σε αρχείο:

update_config=1

Για ένα laptop ή desktop που χρειάζεται απλά να συνδεθεί αμέσως με ένα ασφαλή δίκτυο, αφαιρέστε το # για WEP ή WPA-WPA2PSK αναλώγος:

WEP παράδειγμα:

network={
	ssid="debian" #Example WEP Network
	key_mgmt=NONE
	wep_key0=6162636465
	wep_tx_keyidx=0
}

WPA παράδειγμα:

network={
	ssid="aptosid_Worldwide" #Example WPA Network
	psk="mysecretpassphrase"
}

Το επόμενο βήμα ασφαλίζει την πρόσβαση στο wpa-roam.conf . Αυτό είναι απαραίτητο γιατί μέσα στο αρχείο υπάρχουν κλειδιά ιδιοτηκών δικτύων:

chmod 600 /etc/wpa_supplicant/wpa-roam.conf

Ενεργοποιείστε την ασύρματη σύνδεση

ifup wlan0

Τώρα ελέγξτε αν είστε συνδεδεμένος στο δίκτυο:

wpa_cli status

Θα πρέπει να δείτε κάτι παρόμοιο με το παρακάτω:

Selected interface 'wlan0'
bssid=94:0c:6d:aa:f4:42
ssid=aptosid_Melbourne
id=3
pairwise_cipher=CCMP
group_cipher=CCMP
key_mgmt=WPA2-PSK
wpa_state=COMPLETED
ip_address=192.168.1.102

Αν δεν έχει ip_address= αριθμούς, τότε δεν είστε συνδεδεμένος οπότε ελέγξτε ξανά τα αρχεία ρυθμίσεων, σταματώντας πρώτα το wlan0:

wpa_action wlan0 stop

Αν χρειάζεστε εξειδικευμένα αρχεία ρύθμισης δείτε εδώ

Ενεργοποίηση αλλαγής μεταξύ ενσύρματων και ασύρματων δικτύων

Δείτε πρώτα Αλλαγή μεταξύ ενσύρματου και ασύρματου δικτύου γιατί αν δεν είναι ρυθμισμένο σωστά δεν θα γίνει η αλλαγή και η σύνδεση στο δίκτυο.

Αφού ρυθμίσετε το ifplugd το τελικό αρχείο ρύθμισης θα πρέπει να μοιάζει κάπως έτσι:

auto lo
iface lo inet loopback

# governed by ifplugd ... do not use allow-hotplug or auto options
iface eth0 inet dhcp

#Added by user
allow-hotplug wlan0
iface wlan0 inet manual
  wpa-roam /etc/wpa_supplicant/wpa-roam.conf

#this line must always be here
iface default inet dhcp

Με την βοήθεια του IDString και του Priority μπορείτε να επιλέξετε σε πιο δίκτυο συνδέεται ο υπολογιστής κατά την εκκίνηση. Η μεγαλύτερη προτεραιότητα (priority) είναι 1000 και η μικρότερη είναι 0. Πρέπει να προσθέσετε και το id_str στο /etc/network/interfaces.

Η σύνταξη του /etc/network/interfaces.

Το πρώτο είναι για σύνδεση με dhcp και το δεύτερο για στατική διεύθυνση IP.

# id_str="home_dhcp"
iface home_dhcp inet dhcp

#this line must always be here
iface default inet dhcp

# id_str="home_static"
iface home_static inet static

        address 192.168.0.20
        netmask 255.255.255.0
        network 192.168.0.0

        broadcast 192.168.0.255
        gateway 192.168.0.1
Παράδειγματα

Αν θέλετε να συνδέεστε αυτόματα στο ασύρματο δίκτυο του σπιτιού σας όταν είστε σπίτι, δώστε IDString "home" και προτεραιότητα "15". Εάν ταξιδεύτε και θέλετε το laptop να συνδέετε σε οποιοδήποτε ανοικτό διαθέσιμο δίκτυο, δώστε IDString "stalk" και προτεραιότητα "1" (πολύ μικρή). Παρακαλώ πάντα ελέγχετε αν η σύνδεση σας είναι νόμιμη και αποσυνδεθείτε όταν είναι ξεκάθαρο ότι το δίκτυο δεν είναι δωρεάν.

Παράδειγμα για το /etc/network/interfaces:

# /etc/network/interfaces -- configuration file for ifup(8), ifdown(8)

# The loopback interface
# automatically added when upgrading
auto lo
iface lo inet loopback

allow-hotplug eth0
iface eth0 inet dhcp

allow-hotplug wlan0
iface wlan0 inet manual
  wpa-roam /etc/wpa_supplicant/wpa-roam.conf

#this line must always be here
iface default inet dhcp

iface home inet dhcp
iface stalk inet dhcp

Παράδειγμα /etc/wpa_supplicant/wpa-roam.conf (SSID και passwords έχουν αλλαχθεί ή είναι απλά επεξηγιματικά):

ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
        ssid="my_ssid"
        scan_ssid=1
        psk=123ABC  ##here comes the passphrase in hexadecimal code!!
#       psk="password_in_ascii"   ##you dont need to
        key_mgmt=WPA-PSK
        pairwise=TKIP
        group=TKIP
        auth_alg=OPEN
        priority=15
        id_str="home"
}

network={
        ssid=""
        scan_ssid=1
        key_mgmt=NONE
        auth_alg=OPEN
        priority=1
        disabled=1   ## no automatic connection, one needs wpa_cli or wpa_gui
        id_str="stalk"
}

Με "disable=1" δεν θα συνδεθεί αυτόματα σε ένα δίκτυο (ανοικτά ασύρματα δίκτυα). Πρέπει να ενεργοποιήσετε το roaming μέσω wpa_gui ή wpa_cli. Για αυτόματα roaming αφαιρέστε ή βάλτε μπροστά ένα # στην γραμμή με το "disabled".

WEP κρυπτογράφηση

Εάν θέλετε να προσθέσετε δίκτυα κρυπτογραφιμένα με WEP στο wpa-roam.conf σας μόνιμα:

network={
	ssid="example wep network"
	key_mgmt=NONE
	wep_key0="abcde"
	wep_key1=0102030405
	wep_tx_keyidx=0
}

Σημειώσεις

1. Εύκολο στην επαναχρησιμοποίηση

Όταν το ρυθμίσετε είναι εύκολο να χρησιμοποιήσετε τα ίδια αρχεία ρυθμίσεως και σε άλλους υπολογιστές. Απλά αντιγράψτε το /etc/network/interfaces (αλλάξτε τα ονόματα των διεπαφών αν χρειάζεται) και /etc/wpa_supplicant/wpa-roam.conf στον άλλο υπολογιστή. Δεν χρειάζεται να "εγκαταστήσετε" κάτι άλλο μετά από αυτό.

2. Backup

Είναι καλό να κάνετε backup τα /etc/network/interfaces και /etc/wpa_supplicant/wpa-roam.conf αλλά κρυπτογραφίστε το backup γιατί περιέχει ευαίσθητες πληροφορίες.

A good method to safely backup and encrypt the config files is with tar and gpg. As root:

tar -cf backup_name.tar /etc/network/interfaces /etc/wpa_supplicant/wpa-roam.conf && gpg -c backup_name.tar

Two files have now been created in $ HOME:
backup_name.tar
backup_name.tar.gpg

Delete backup_name.tar from your system as you only need the backup_name.tar.gpg file.

To decrypt the backup_name.tar.gpg file:

gpg -d backup_name.tar.gpg
3. Κρυμμένα SSID

Κρυμμένα SSID ανυχνεύονται όταν το scan_ssid=1 βρίσκεται μέσα σε ένα network block (network={...}).

Ασφάλεια για ασύρματα modem/router

Όπου έχετε τον έλεγχο του ασύρματου modem/router, υπάρχουν μερικές βασικές πολιτικές ασφαλείας που θα σας βοηθήσουν να προστατέψετε την δική σας μεριά του δικτύου.

Επιλογή πρωτόκολλου
  • WPA2-PSK είναι η καλύτερη επιλογή.
  • Σαν πρωτόκολλο κρυπτογράφισης επιλέξτε το AES.
  • Το συνθηματικό θα πρέπει να είναι δύσκολο να βρεθεί.
Συνθηματικά / Κωδικοί

Για ένα συνθηματικό/κωδικό ασφαλή ο οποίος δεν μπορεί να μαθευτή απ' έξω χρησιμοποιήστε το pwgen σε ένα τερματικό (διαβάστε επίσης: man pwgen):

$ pwgen -s 63 1
VltnfGmGKXovVv2rmrCFFXBZ55Mij5bA6WytVJnVoKUqRn6dfjldG6MBrRo0Cdi
  • -s = ασφαλής
  • 63 = αριθμός χαρακτήρων
  • 1 = δημιούργησε μόνο έναν τυχαίο κωδικό

Χωρίς το -s επιστρέφονται κωδικοί τους οποίους μπορείτε να προφέρετε, αυτό όμως είναι μάλλον κάτι που δεν θέλετε:

$ pwgen 8 3
Sooxae2s Niew9ugh Hi7eeloo

Όταν δημιουργηθεί το συνθηματικό/κώδικας φυλάξτε το σε ένα αρχείο κειμένου πάνω σε ένα USB-stick και εφαρμόστε το και στους υπόλοιπους υπολογιστές που χρησιμοποιούν το ασύρματο δίκτυο σας. Μην φυλάξετε το συνθηματικό/κωδικό πάνω στον υπολογιστή σας.

Παράδειγμα router με τις πιο πάνω συμβουλές:
Version: WPA2-PSK
Encryption: AES
PSK Password: VltnfGmGKXovVv2rmrCFFXBZ55Mij5bA6WytVJnVoKUqRn6dfjldG6MBrRo0Cdi
