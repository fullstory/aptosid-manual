Αναζήτηση On-Line

Option 1

Υπάρχουν μερικοί περιορισμοί και ιδιαιτερότητες στον τρόπο χρήσης της μηχανή αναζήτησης με τα Ελληνικά. Για off-line αναζητήσεις, χρησιμοποιήστε το Recoll όπως περιγράφεται πιο κάτω.
Περιορισμοί/Ιδιαιτερότητες για την Ελληνική γλώσσα

Η αναζήτηση λέξης στα Αγγλικά επιστρέφει τα αναμενόμενα αποτελέσματα, αλλά αν θα χρησιμοποιήσετε μια Ελληνική λέξη, είναι σημαντικό η λέξη να είναι γραμμένη σωστά και να είναι τονισμένη γιατί αλλιώς θα σας επιστραφούν λάθος αποτελέσματα.

Χρησιμοποιήστε το Ταίριαξε οποιαδήποτε λέξη στην αναζήτηση σας γιατί η αναζήτηση με Ταίριαξε όλες τις λέξεις μπορεί να μην σας επιστρέψει αποτελέσματα. Επιπλέον μια αναζήτηση για μια Ελληνική λέξη μπορεί να επιστρέψει σαν αποτέλεσμα σύνδεσμους χωρίς περιγραφή. Οι σύνδεσμοι αυτοί δεν περιέχουν την λέξη που χρησιμοποιήσατε στην αναζήτηση αλλά μόνο κομμάτια της λέξης. Τέλος το μαύρισμα (highlighting) της λέξης που χρησιμοποιήσατε στην αναζήτηση δεν υποστηρίζετε και η ταξινόμηση με βάση την σχετικότητα δεν είναι πάντα σωστή.

Πληκτρολογήστε μια ή περισσότερες λέξης στο πεδίο αναζήτησης και κάντε κλικ Αναζήτηση (ή πατήστε Enter).

Επιλογές Αναζήτησης

Εάν είναι επιλεγμένο το Ταίριαξε όλες τις λέξεις , θα επιστραφούν μόνο σελίδες οι οποίες περιέχουν όλες τις λέξεις του πεδίου αναζήτησης. Με το Ταίριαξε οποιαδήποτε λέξη , θα επιστραφούν όλες οι σελίδες οι οποίες περιέχουν τουλάχιστον μια από τις λέξεις του πεδίου αναζήτησης.

Εναλλακτικά μπορείτε να βάλετε ένα συν (+) μπροστά από μια ή περισσότερες λέξης για να σας επιστραφούν σελίδες οι οποίες περιέχουν όλες εκείνες τις λέξεις. Λέξεις με ένα πλην (-) μπροστά τους αλλάζουν το αποτέλεσμα έτσι ώστε να επιστρέφονται μόνο σελίδες που δεν περιέχουν εκείνες τις λέξεις.

Σημειώστε ότι η αναζήτηση εκφράσεων δεν υποστηρίζεται, πχ δεν έχει νόημα να βάζετε τις λέξεις προς αναζήτηση μέσα σε εισαγωγικά "όπως έτσι".

Τα αποτελέσματα ταξινομούνται με βάση την σχετικότητα με τις πιο σχετικές σελίδες να μπαίνουν πρώτες. Η σχετικότητα εξαρτάται από τον αριθμό και την θέση των λέξεων που ταιριάζουν μέσα στην σελίδα.

Option 2

Για on line αναζήτηση γράψτε ή επικολλήσετε (paste) τα ακόλουθα στο πεδίο αναζήτησης του browser σας, (όπου xxxx είναι η λέξη η οποία θέλετε να αναζητήσετε):

xxxxx site:manual.aptosid.com/el

Αναζήτηση Off-line

Το Εγχειρίδιο Λειτουργικού Συστήματος aptosid έχει δημιουργίσει μερικές ειδικές ρυθμίσεις με την βοήθεια του προγραμματιστή του Recoll οι οποίες είναι μέρος του πακέτου του recoll. Το Recoll, το οποίο είναι ένα προσωπικό εργαλείο αναζήτησης πλήρης κειμένου (personal full text search tool) για Unix/Linux. Βασίζεται σε ένα πολύ δυνατό Xapian back-end, για το οποίο παρέχει ένα πλούσιο σε χαρακτηριστικά αλλά εύκολο στη χρήση front-end με ένα QT γραφικό περιβάλλον. Ένα μεγάλο ευχαριστώ στην ομάδα του Recoll.

Το Recoll ονομάζεται Personal Search Tool (Προσωπικό εργαλείο αναζήτησης) στο Debian Μενού του KDE.

Βήμα 1

Αυτή την στιγμή το εγχειρίδιο είναι διαθέσιμο μέσω του apt για τις παρακάτω γλώσσες:
da, de, el, en, es, fr, hr, it, ja, nl, pl, pt-br, ro, ru.

apt-get update
apt-get install aptosid-manual-xx (όπου -xx είναι η γλώσσα σας (el για ελληνικά))
-->
Βήμα 2
apt-get install recoll
Βήμα 3
  • Την πρώτη φορά που θα ξεκινήσει το Recoll θα σας ρωτήσει αν θέλετε να χτίσετε (build) την βάση δεδομένων (database), πατήστε cancel. Μετά από αυτό θα ανοίξει ένα παράθυρο ρυθμίσεων.
  • Στις Global parameters - Στο Top directories κουτάκι, πατήστε στο σύμβολο του + (συν) και πηγαίνετε στο usr/share/aptosid-manual , κάντε κλικ στην γλώσσα σας και μετά κλικ στο ok. Μετά κάντε κλικ στο σύμβολο του - (πλην) και διαγράψτε οποιοδήποτε άλλο φάκελο συμπεριλαμβανομένου και του ~ (tilde, περισπωμένη).
  • Τέλος File>Update 2045Index και αναζητήστε αυτό που θέλετε μέσα στο aptosid-εγχειρίδιο.

Σημείωση: Στο κείμενο που σας επιστρέφετε σαν αποτέλεσμα της αναζήτησης, λείπουν οι τόνοι και το σίγμα τελικό (ς) γίνεται σίγμα κανονικό (σ). Αυτό γίνεται για ελαχιστοποίηση του χρόνου αναζήτησης και δεν είναι σφάλμα του προγράμματος ή ορθογραφικό λάθος.

Εάν χρειαστεί να ρυθμίσετε ξανά (reset) το recoll λόγω ενημέρωσης του εγχειριδίου aptosid μέσω του apt, (σαν χρήστης $ μέσα στην κονσόλα):

recollindex -z

Έπειτα: Kmenu >Debian >Applications > Data Management > Personal Search Tool και στο Recoll File>Update Index.

Το 'Preview' θα αναδείξει τις λέξεις μέσα στο αρχείο, τις οποίες χρησιμοποιήσατε στην αναζήτηση και το 'Open' θα ανοίξει τον Konqueror. Χρησιμοποιείστε την Find on this page εντολή για μια μικρο αναζήτηση για την λέξη που έχετε αναζητήσει στο recoll.
