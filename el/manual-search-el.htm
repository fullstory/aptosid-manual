Αναζήτηση On-Line

Λόγω ορισμένων σοβαρών περιορισμών της μηχανής αναζήτησης, χρησιμοποιήστε συνηθισμένες αγγλικές λέξης για να ψάξετε μέσα στο ελληνικό εγχειρίδιο. Για παράδειγμα χρησιμοποιήστε partition αντί κατάτμηση . Για off-line αναζητήσεις, χρησιμοποιήστε το Recoll όπως περιγράφεται πιο κάτω.

Πληκτρολογήστε μια ή περισσότερες λέξης στο πεδίο αναζήτησης και κάντε κλικ Αναζήτηση (ή πατήστε Enter).

Επιλογές Αναζήτησης

Εάν είναι επιλεγμένο το Ταίριαξε όλες τις λέξεις , θα επιστραφούν μόνο σελίδες οι οποίες περιέχουν όλες τις λέξεις του πεδίου αναζήτησης. Με το Ταίριαξε οποιαδήποτε λέξη , θα επιστραφούν όλες οι σελίδες οι οποίες περιέχουν τουλάχιστον μια από τις λέξεις του πεδίου αναζήτησης.

Εναλλακτικά μπορείτε να βάλετε ένα συν (+) μπροστά από μια ή περισσότερες λέξης για να σας επιστραφούν σελίδες οι οποίες περιέχουν όλες εκείνες τις λέξεις. Λέξεις με ένα πλην (-) μπροστά τους αλλάζουν το αποτέλεσμα έτσι ώστε να επιστρέφονται μόνο σελίδες που δεν περιέχουν εκείνες τις λέξεις.

Σημειώστε ότι η αναζήτηση εκφράσεων δεν υποστηρίζεται, πχ δεν έχει νόημα να βάζετε τις λέξεις προς αναζήτηση μέσα σε εισαγωγικά "όπως έτσι".

Τα αποτελέσματα ταξινομούνται με βάση την σχετικότητα με τις πιο σχετικές σελίδες να μπαίνουν πρώτες. Η σχετικότητα εξαρτάται από τον αριθμό και την θέση των λέξεων που ταιριάζουν μέσα στην σελίδα.

Αναζήτηση Off-line

Το Εγχειρίδιο Λειτουργικού Συστήματος sidux (Operating System Manual) δημιούργησε μερικές ειδικές ρυθμίσεις σε συνδυασμό με τον προγραμματιστή του Recoll, το οποίο είναι ένα προσωπικό εργαλείο αναζήτησης πλήρης κειμένου (personal full text search tool) για Unix/Linux. Βασίζεται σε ένα πολύ δυνατό Xapian back-end, για το οποίο παρέχει ένα πλούσιο σε χαρακτηριστικά αλλά εύκολο στη χρήση front-end με ένα QT γραφικό περιβάλλον.

Το Recoll ονομάζεται Personal Search Tool (Προσωπικό εργαλείο αναζήτησης) στο Debian Μενού του KDE.

Για να ρυθμίσετε την τοπική αναζήτηση θα πρέπει να πηδήξετε μέσα από μερικά στεφάνια!

Βήμα 1

Αυτή την στιγμή το εγχειρίδιο είναι διαθέσιμο μέσω του apt για τις παρακάτω γλώσσες:
da, de, el, en, es, fr, hr, it, ja, nl, pt-br, ro, ru.

apt-get update
apt-get install sidux-manual-xx (όπου -xx είναι η γλώσσα σας (el για ελληνικά))
Βήμα 2
apt-get install recoll
Βήμα 3

Κατεβάστε τα recoll αρχεία που είναι συγκεκριμένα για το sidux (σαν χρήστης $) μέσα σε ένα προσωρινό φάκελο και αποσυμπιέστε τα:

wget http://bluewater.sidux.com/outgoing/sidux-recoll-configs.tar.gz

Δημιουργήστε ένα .recoll φάκελο μέσα στο $HOME, εάν δεν υπάρχει ήδη, (είναι ένας κρυμμένος (hidden) φάκελος).

Βάλτε τα ακόλουθα sidux-recoll-config(sidux-recoll-αρχεία-ρυθμίσεων) αρχεία μέσα στο .recoll
mimeconf
mimemap
mimeview
recoll.conf

Για τα βήματα 4 & 5 θα πρέπει να είστε διαχειριστής (root) για να μπορέσετε να αλλάξετε τον /usr/share/recoll φάκελο:
Βήμα 4

Βάλτε το αρχείο με όνομα rclsiduxman που βρίσκεται στον φάκελο filters του sidux-recoll-config μέσα στο /usr/share/recoll/filters.

mv sidux-recoll-configs/filters/* /usr/share/recoll/filters
Βήμα 5

Βάλτε το αρχείο με όνομα sidux-book.png που βρίσκεται στον φάκελο images του sidux-recoll-config μέσα στο /usr/share/recoll/images.

mv sidux-recoll-configs/images/* /usr/share/recoll/images
Βήμα 6

Σαν διαχειριστής (root) μέσα στην κονσόλα:

update-menus

Σαν χρήστης στο Kmenu, πηγαίνετε στο Kmenu >Debian >Applications > Data Management > Personal Search Tool .

Την πρώτη φορά που θα ξεκινήσει το Recoll θα σας ρωτήσει αν θέλετε να χτίσετε (build) την βάση δεδομένων (database), πατήστε cancel, έπειτα στο File>Update Index και ψάξτε το θέμα που χρειάζεστε στο εγχειρίδιο του sidux.

Εάν χρειαστεί να ρυθμίσετε ξανά (reset) το recoll λόγω ενημέρωσης του εγχειριδίου sidux μέσω του apt, (σαν χρήστης $ μέσα στην κονσόλα):

recollindex -z

Έπειτα: Kmenu >Debian >Applications > Data Management > Personal Search Tool και στο Recoll File>Update Index.

Αν το χρειαστείτε, μέσα στο Konqueror, χρησιμοποιείστε την Find on this page εντολή για μια αναζήτηση για την λέξη που έχετε αναζητήσει στο recoll.
