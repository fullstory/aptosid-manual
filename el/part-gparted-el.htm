ΠΡΟΣΟΧΗ: για την ονομασία δίσκων Παρακαλούμε αναφερθείτε στο UUID, ετικέτες τμημάτων και fstab, καθώς το sidux χρησιμοποιεί εξ' ορισμού UUID

Τα εργαλεία κατάτμησης μπορεί να απαιτήσουν κωδικό υπερχρήστη(root). Kαθώς δεν έχει οριστεί κανένας, πατήστε απλά 'enter', Δείτε το : Live Mode

Βασικά

Ένα τμήμα πρέπει να έχει σύστημα αρχείων. Το Linux υποστηρίζει διάφορα συστήματα αρχείων για χρήση. Υπάρχουν τα ReiserFs, Ext3, και για έμπειρους χρήστες τα XFS και JFS. Το Ext2 είναι εύχρηστο σαν σύστημα για αποθήκευση δεδομένων καθώς υπάρχει διαθέσιμος windows driver για data-swapping (ανταλλαγή δεδομένων).

Για φυσιολογική χρήση συνιστούμε το ext3 σύστημα αρχείων, αυτό είναι το καθορισμένο σύστημα αρχείων για το sidux και είναι πολύ καλά υποστηριζόμενο.

Η έκδοση του GParted είναι η 0.3.1.1. Οι ακόλουθες πληροφορίες έχουν αντληθεί από την επίσημη τεκμηρίωση του GParted και έχουν τροποποιηθεί ώστε να αποτελέσουν ένα οδηγό χρησιμοποιώντας το sidux. Aφορούν το GParted μέσα από το sidux, και δεν θα πρέπει να χρησιμοποιούνται ως ο απόλυτος οδηγός. E&OE

Για πλήρη τεκμηρίωση αναφερθείτε στο GParted

Χρησιμοποιώντας το GParted

Η δημιουργία και διαχείρηση τμημάτων δεν είναι κάτι που γίνεται τυπικά κάθε μέρα. Παρ όλα αυτά, διαβάστε μια φορά αυτό τον οδηγό, ώστε να είστε άνετοι με διάφορα σενάρια και μερικές από τις καρτέλες που θα εμφανιστούν.

  • Γενικές πληροφορίες

    Όταν εκτελείται το GPARTED , ανοίγει ένα παράθυρο και ανιχνεύονται οι δίσκοι.

    Αν πατήσετε στο μενού πάνω αριστερά, εμφανίζεται μια καρτέλα. Μπορείτε να επιλέξετε την ανανέωση των δίσκων του συστήματος. Yπάρχει επίσης μια συντόμευση στο πληκτρολόγιο για εύκολη ανανέωση των πληροφοριών. Με την δεύτερη επιλογή, μπορείτε να επιλέξετε τον σκληρό δίσκο του οποίου τα τμήματα θέλετε να τροποποιήσετε. Αυτό είναι χρήσιμο αν έχετε παραπάνω από έναν δίσκους. Η τρίτη επιλογή στο Gparted μενού χρησιμοποιείται για την εμφάνιση περισσότερων πληροφοριών.

  • Edit

    Ανοίγει ένα νέο παράθυρο από το οποίο μπορείτε να δείτε τα υποστηριζόμενα συστήματα αρχείων και μερικές επιλογές για επεξεργασία τμημάτων.

    Το μενού είναι το δεύτερο από αριστερά. Περιέχει 2 γκριζαρισμένες λειτουργίες οι οποίες είναι αρκετά χρήσιμες: Undo και Apply και επίσης σας επιτρέπει να έχετε πρόσβαση και σε άλλες περιοχές. Yπάρχουν επίσης και οι παρακάτω επιλογές:

  • View

    • Harddisk Information

      Το πεδίο Harddisk Information εμφανίζει λεπτομέρειες για τον σκληρό δίσκο, όπως το μοντέλο, το μέγεθος κλπ. Αυτό είναι και το πιο χρήσιμο για ένα σύστημα με πολλούς σκληρούς δίσκους, όπου οι πληροφορίες χρησιμοποιούνται για να επιβεβαιώσουν ότι ο σκληρός δίσκος που εξετάζεται είναι και αυτός ο οποίος θέλουμε.

    • Pending Operations

      Στο παράθυρο που βρίσκεται στη βάση βρίσκεται μια λίστα με λειτουργίες που εκκρεμούν. Οι πληροφορίες αυτές είναι χρήσιμες καθώς παρέχουν μια ένδειξη με τον αριθμό όσων λειτουργιών εκκρεμούν.

  • Device

    Το Device σας επιτρέπει να δώσετε μια ετικέτα σκληρού δίσκου (disk label) εάν η παρούσα ετικέτα είναι ακατάλληλη, μπορείτε να το αλλάξετε χρησιμοποιώντας αυτή την επιλογή.

  • Το μενού Partition

    Αυτό το μενού είναι από τα πιο σημαντικά. Σας επιτρέπει να εκτελέσετε πολλές λειτουργίες, μερικές από τις οποίες είναι επικίνδυνες.

    Το Delete επιλέγεται όταν θέλετε να διαγράψετε ένα τμήμα. Για να γίνει η διαγραφή, πρέπει πρώτα να επιλέξετε το τμήμα.

    Το Resize/Move είναι επίσης μια χρήσιμη λειτουργία.

  • Δημιουργώντας ένα νέο τμήμα

    Στην μπάρα εργαλείων, το κουμπί New σας επιτρέπει να δημιουργήσετε ένα νέο τμήμα, αν έχετε ήδη επιλέξει μια μη προσδιορισμένη περιοχή. Ένα νέο παράθυρο εμφανίζεται και σας αφήνει να επιλέξετε το μέγεθος που θέλετε, αν θα είναι πρωτεύoν(Primary), Extended ή λογικό(Logical) τμήμα καθώς και το σύστημα αρχείων.

  • Αν κάνετε λάθος

    Αν έχετε κάνει λάθος μπορείτε να χρησιμοποιήσετε το κουμπί Delete για να διαγράψετε το επιλεγμένο τμήμα ή αν δέν έχετε πατήσει apply για να εφαρμοστεί η επιλογή σας, μπορείτε να πατήσετε το κουμπί undo.

  • Αλλαγή μεγέθους/Μετακίνηση (resize/move)

    Όταν θέλετε να αλλάξετε το μέγεθος ενός τμήματος που έχετε επιλέξει, πατήστε το κουμπί Resize/Move : ένα νέο παράθυρο εμφανίζεται. Χρησιμοποιήστε το ποντίκι για να μειώσετε ή να αυξήσετε το μέγεθος του τμήματος ή αν προτιμάτε, χρησιμοποιήστε τα βέλη.

    Αφού η εντολή για αλλαγή μεγέθους έχει δοθεί, πατήστε το Apply καθώς μέχρι να το πατήσετε δεν έχει γίνει καμιά αλλαγή στο δίσκο.

    Η διάρκεια των λειτουργιών εξαρτάται από το νέο μέγεθος του τμήματος.

    Αφού επεξεργαστείτε με προσοχή τη λίστα με τα τμήματα, παρακαλούμε αποσυνδεθείτε από το sidux και επανεκκινήστε το σύστημα σε sidux ώστε να επαναγνωστεί η λίστα με τα τμήματα.

Αλλαγή μεγέθους σε τμήμα NTFS

Η αλλαγή μεγέθους σε τμήμα NTFS απαιτεί επανεκκίνηση του συστήματος! ΜΗΝ ΠΡΑΓΜΑΤΟΠΟΙΗΣΕΤΕ καμιά άλλη λειτουργία στο τμήμα πρίν την επανεκκίνηση, αλλιώς θα υπάρξουν σφάλματα.

  • Μετά την εκκίνηση σε M$ Windows, το σύστημα θα εμφανίσει μια ειδική οθόνη, και ένα μήνυμα που θα ρωτά για drive consistency : Checking file system on c :
  • Αφήστε το AUTOCHK να εκτελεστεί : Τα NT χρειάζονται να ελέγξουν το σύστημα αρχείων μετά την αλλαγή μεγέθους στο τμήμα.
  • Στο τέλος της διαδικασίας, ο υπολογιστής θα επανεκκινήσει αυτόματα για δεύτερη φορά. Αυτό σιγουρεύει ότι όλα πήγαν τέλεια.
  • Μετά την επανεκκίνηση, τα Windows XP θα είναι καλά, αλλά θα πρέπει πρώτα να αφήσετε το σύστημα να εκκινήσει και να περιμένετε για την οθόνη login!
