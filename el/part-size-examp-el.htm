Μεγέθη Κατατμήσεων (Partition) και Παραδείγματα

Για φυσιολογική χρήση συνιστούμε το ext3 σύστημα αρχείων, αυτό είναι το καθορισμένο σύστημα αρχείων για το sidux και είναι πολύ καλά υποστηριζόμενο.

Με τον Partition-Manager (Διαχειριστή κατατμήσεων) GParted οι σκληροί δίσκοι μπορούν να χωριστούν ή/και να διαμορφωθούν (format). Το πρόγραμμα έχει γραφικό περιβάλλον που είναι αρκετά απλό.

Το Gparted μπορεί επίσης να σμικρύνει (shrink) ή να μεταφέρει (move) κατατμήσεις και να διαχειριστή κατατμήσεις NTFS [μόλις κάνετε αλλαγές σε μια NTFS κατάτμηση θα πρέπει να επανεκκινήσετε τον υπολογιστή πριν κάνετε οποιεσδήποτε άλλες αλλαγές ]. Δείτε την πλήρη βιβλιογραφία για το gparted. Οι αλλαγές στα ntfs-partitions μπορούν επίσης να γίνουν με εμπορικά εργαλεία όπως το Partition Magic (TM) και το Acronis (TM).

ΠΑΝΤΑ ΝΑ ΚΑΝΕΤΕ BACK-UP ΤΑ ΔΕΔΟΜΕΝΑ ΣΑΣ!

Αν μια κατάτμηση φαίνεται να είναι προσαρτημένη, αποπροσαρτήστε την συσκευή, συμπεριλαμβανομένου και του swap, με δεξί κλικ πάνω στην κατάτμηση στο gparted ή μέσω ενός τερματικού, για παράδειγμα:

umount /dev/sda1

Η swap κατάτμηση μπορεί να αποπροσαρτηθεί μέσα σε ένα τερματικό με: 

swapoff -a

Θεωρητικά, 5 GB είναι υπεραρκετά για εγκατάσταση, αλλά μόνο για αυτό. Ένα λογικό ελάχιστο για μια minimum εγκατάσταση είναι τα 12 GB. Για τους νέους στο linux, προτείνουμε μόνο 2 κατατμήσεις για την αρχή (root/home και swap), γιατί έτσι απλοποιείται η πρώτη εγκατάσταση αρκετά και μετά μπορείτε να δημιουργήσετε έξτρα κατάτμηση για ένα ξεχωριστό /home και επιπλέον κατατμήσεις για αρχεία.

Πάντως θα πρέπει να έχετε μια swap κατάτμηση (όμοιο με το windows swapfile, αλλά πολύ πιο αποτελεσματικό). Για κανονική χρήση, η swap κατάτμηση θα πρέπει να είναι διπλάσια της μνήμης ram.

Για να ανταλλάξετε αρχεία με μια εγκατάσταση Windows θα πρέπει να χρησιμοποιήσετε vfat (fat32) ή ext2 μιας και υπάρχει οδηγός ext2 για MS Windows™. [Το XFS δεν υποστηρίζεται].

Όποιος δημιουργεί περισσότερες από μια κατατμήσεις, καλό είναι να γράψει κάτω τα ονόματα τους!

Αυτή η κατανομή προτείνεται όταν υπάρχει λίγος χώρος στον σκληρό δίσκο (π.χ. σε laptop). Τα Windows εγκαθίστονται σε μια ξεχωριστή 3-5 GB κατάτμηση χωρίς τίποτα άλλο. Όλα τα προγράμματα και αρχεία βρίσκονται στη δεύτερη μεγαλύτερη κατάτμηση. Όποιος έχει περισσότερο χώρο μπορεί να μεγαλώσει αυτή την κατάτμηση ή να χωρίσει τις windows-εφαρμογές και δεδομένα σε 2 κατατμήσεις.

Γενικά

Υπάρχουν πολλοί τρόποι να κάνετε partition τον σκληρό σας δίσκο. Αυτά τα παραδείγματα είναι αρκετά για αρχή. Έχει νόημα επίσης να προσθέσετε έναν επιπλέον σκληρό δίσκο ώστε να έχετε επιπλέον ταχύτητα, άνεση και ασφάλεια. Σε αυτή την περίπτωση τα MS Windows (TM) πάντα μπαίνουν στον πρώτο δίσκο/κατάτμηση!
