Ενεργοποιώντας υπηρεσίες (services) στο aptosid

insserv (Πόντος 2008-04 και μετά ): Για να ξεκινήσετε/σταματήσετε υπηρεσίες που είναι ήδη εγκατεστημένες:

Διαβάστε το /usr/share/doc/insserv/README.Debian, τις σημειώσεις έκδοσης (release notes) και τις man pages (σελίδες εγχειριδίου) προσεκτικά:

 $ man insserv
 $ man invoke-rc.d
 $ man update-rc.d
 ψάξτε στο google για LSB headers

Για 'ξεκίνημα' (start):

/etc/init.d/<υπηρεσία> start

Για 'σταμάτημα' (stop):

/etc/init.d/<υπηρεσία> stop

Για 'επανεκκίνηση' (restart):

/etc/init.d/<υπηρεσία> restart

Για να μην τρέχει μια υπηρεσία κατά την εκκίνηση (startup)

update-rc.d <υπηρεσία> remove
[θα αφαιρέσει όλους τους συνδέσμους εκκίνησης]

Για να σιγουρευτείτε ότι μια υπηρεσία θα τρέχει κατά την εκκίνηση [δεν χρειάζεται πάντα].

update-rc.d <υπηρεσία> defaults
[θα δημιουργήσει συνδέσμους εκκίνησης]

Για να δείτε ποιές υπηρεσίες τρέχουν:

ls /etc/rc5.d

S σημαίνει ότι η υπηρεσία θα ξεκινήσει.
K σημαίνει ότι η υπηρεσία δεν θα ξεκινήσει.

Παλαιότερες Εγκαταστάσεις (πριν την Πόντος 2008-04)

insserv σε συστήματα που εγκαταστάθηκαν πριν την Πόντος 2008-04

Προειδοποίηση: Παλαιότερες εγκαταστάσεις δεν θα πάρουν το insserv αυτόματα, λόγω των initscripts ή symlinks του χρήστη. ΔΕΝ συνιστάται η εγκατάσταση του insserv σε εγκαταστάσεις παλαιότερες της Πόντος. Το πακέτο αυτό θα πρέπει να χρησιμοποιείται με προσοχή γιατί λανθασμένες ή ανύπαρκτες LSB εξαρτήσεις υπαρχόντων initscripts μπορούν να οδηγήσουν σε σύστημα που να μην εκκινεί. Η εγκατάσταση του insserv σε ένα υπάρχον σύστημα μπορεί να χρειαστεί μερικές χειροκίνητες αλλαγές.

Διαβάστε τις σημειώσεις τεκμηρίωσης του Debian, και τις σημειώσεις έκδοσης πριν εγκαταστήσετε και ενεργοποιήσετε το insserv.

Boot-Up Manager (bum) - Γραφικό εργαλείο ρύθμισης υπηρεσιών

Εάν η λογική της διαδικασίας εκκίνησης ενός debian συστήματος δεν σας είναι κατανοητή, τότε δεν θα πρέπει να παίζετε με εικονικούς συνδέσμους (symlinks), δικαιώματα (permissions) κτλ. Το Boot-Up Manager (διαχειριστής εκκίνησης) θα σας βοηθήσει να αυτοματοποιήσετε τις ρυθμίσεις σας χωρίς να σπάσετε το σύστημα.

Το Boot-Up Manager είναι ένας γραφικός διαχειριστής των επιπέδων εκτέλεσης (runlevel) ο οποίος σας επιτρέπει να ρυθμίσετε τις υπηρεσίες init που θα τρέχουν κατά την εκκίνηση και επανεκκίνηση. Σας παρουσιάζει μια λίστα με όλες τις υπηρεσίες που μπορούν να τρέξουν κατά την εκκίνηση και εσείς μπορείτε να τις ενεργοποιήσετε/απενεργοποιήσετε.

apt-get install bum

Για να χρησιμοποιήσετε το Boot-Up Manager GUI:

$ sux
password:
bum

Λεπτομερείς τεκμηρίωση για το Boot-Up Manager (bum).

Τερματίζοντας μια υπηρεσία ή μια διεργασία

Το pkill είναι πολύ χρήσιμο γιατί μπορεί να διαβαστεί από άνθρωπο (human readable) και λειτουργεί σε mode χρήστη και mode διαχειριστή μέσα στο τερματικό ή το tty

pkill -n υπηρεσία

Εάν δεν είστε σίγουρη για το ακριβής όνομα μιας διεργασίας ή μιας υπηρεσίας που θέλετε να τερματίσετε το pkill <tab> <tab> θα σας επιστρέψει μια λίστα.

Μια καλή εναλλακτική είναι το htop. (το killall -9 είναι η τελευταία σας εναλλακτική).

Έχασα τον κωδικό υπερχρήστη

Δεν μπορείτε να ανακτήσετε τον χαμένο κωδικό αλλά μπορείτε να ορίσετε έναν καινούριο.

Αρχικά εκκινήστε από το Live-CD.

Ως διαχειριστής (root) προσαρτήστε (mount) το τμήμα root (για παράδειγμα το /dev/sdb2)

mount /dev/sdb2 /media/sdb2

Τώρα κάνετε chroot στο παλιό root τμήμα με και δώστε ένα νέο password:

chroot /media/sdb2 passwd

Αλλαγή κωδικού χρήστη

$ passwd

Για να αλλάξετε τον κωδικό 'root', σαν # root:

passwd

Για να αλλάξετε τον κωδικό του χρήστη όταν είστε διαχειριστής, σαν # root:

passwd <user>

Επιλογή προεπιλεγμένων γραμματοσείρων, rendering και μεγέθη

το aptosid χρησιμοποιεί (Debian) προεπιλεγμένες δωρεάν (ανοιχτού κώδικα) γραμματοσειρές οι οποίες αποδείχτηκαν πολύ ισορροπημένες, η δικές σας επιλογές γραμματοσειρών μπορούν να μειώσουν την ποιότητα του rendering. Υπάρχουν όμως πολλές ισχυρές επιλογές στο Debian (εκτός του KDE > systemsettings ) που μπορούν να βοηθήσουν στην παροχή άψογου rendering και με άλλες γραμματοσειρές. Έχετε υπόψη σας ότι κάθε γραμματοσειρά έχει μόνο μερικά πραγματικά τέλεια μεγέθη, άλλα μεγέθη μπορεί να μην παίζουν καλά.

Το παίξιμο με το μέγεθος dpi με την εντολή μπορεί να φανεί χρήσιμο:

fix-dpi-kdm

Θα πρέπει να δείχνει το DPI για το μέγεθος της οθόνης σας αλλά μπορείτε να παίξετε και με αυτό. Θα πρέπει να πάτε στο init 3 και πίσω στο init 5 για να το κάνετε να δουλέψει ή να επανεκκινήσετε τον υπολογιστή.

Μέτα από αλλαγή γραμματοσειράς ή DPI (μέσα στον X ή στον Firefox/Iceweasel), μπορεί να χρειαστούν μερικές ρυθμίσεις για να έχετε τα αποτελέσματα που θέλετε ιδίως μετά από αλλαγή από Bitmap γραμματοσειρές σε True Type γραμματοσειρές ή το αντίθετο μέσω:

dpkg-reconfigure fontconfig-config

Επιλέξτε native και autohinter στο automatic ή παίξτε μαζί του.

Εάν αυτό δεν δουλέψει μπορεί να χρειαστεί να ξανακάνετε το fonts cache σας με:

apt-get install --reinstall --yes -o DPkg::Options::=--force-confmiss -o DPkg::Options::=--force-confnew fontconfig fontconfig-config
Εφαρμογές βασισμένες στο GTK όπως ο Firefox/Icewasel

Γενικά εφαρμογές που είναι βασισμένες στο GTK προβληματικές με τις προεπιλογές του KDE. Αυτό μπορεί να λυθεί με:

apt-get install gtk2-engines system-config-gtk-kde gtk-qt-engine gtk2-engines-qtcurve

Μέσα στο System Settings > Appearance θα έχετε ένα νέο αντικείμενο μενού με όνομα GTK Styles and Fonts. Ρυθμίστε το 'GTK Styles' να χρησιμοποιεί 'Clearlooks' και ρυθμίστε τα 'GTK Fonts' να χρησιμοποιούν τα 'KDE fonts' ή πειραματιστείτε με τις διάφορες επιλογές όσπου να είστε ικανοποιημένος.

Αυτό ΠΙΘΑΝΟΝ να φτιάξει το rendering γραμματοσειρών σε gtk εφαρμογές

modprobe lp
echo lp >> /etc/modules
apt-get remove --purge cupsys cups

/etc/init.d/cups restart

http://localhost:631

Στην πραγματικότητα αυτό που θέλει να κάνετε είναι να αλλάξετε το όνομα χρήστη σε root και να δώσετε το root password.

Ήχος στο aptosid

Εκ προεπιλογής ο ήχος είναι απενεργοποιημένος στο aptosid.

Η KDE έκδοση χρησιμοποιεί Kmix και η XFCE χρησιμοποιεί Mixer.

Πολλές φορές αρκεί ένα κλικ στο εικονίδιο του ήχου και η αποεπιλογή του κουτιού 'Mute'.

Kmix

Στο Kmix θα πρέπει να ενεργοποιήσετε τις επιλογές που θέλετε για τις επιλογές καναλιών ήχου, Kmix > Setting > Configure Channels.Ή σε ένα τερματικό:

$ kmix
XFCE

Στο XFCE τρέξτε το mixer και προσθέστε ορισμένα κοντρόλ μέσω του Multimedia > Mixer και κλικ στο Select Controls box. Ή σε ένα τερματικό:

$ xfce4-mixer

Alsamixer

Αν προτιμάτε να χρησιμοποιήσετε το Alsamixer, αυτό βρίσκεται στο πακέτο alsa-utils:

apt-get update
apt-get install alsa-utils
exit

Ρυθμίστε τις επιλογές ήχου σαν $χρήστης από το τερματικό:

$ alsamixer
