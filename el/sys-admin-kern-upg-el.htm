Εγκατάσταση νέων πυρήνων

Οι πυρήνες sidux είναι μέσα στο apt. Παρακαλώ επιλέξτε τον κοντινότερο σε σας καθρέφτη (mirror) repo sidux και αλλάξτε τις πηγές σας:

Οι πυρήνες είναι διαθέσιμοι στις ακόλουθες μορφές:

  • sidux-686sidux-686 - πυρήνας για τη i686 οικογένεια επεξεργαστών με τον μονό/διπλό πυρήνα ή περισσότερες CPU
  • sidux-amd64πυρήνας για το 64bit sidux
Βήματα για να το κάνετε χειροκίνητα μέσα στην κονσόλα:

1. Στην κονσόλα μπείτε ως υπερχρήστης (root)

apt-get update

2. Για να εγκαταστήσετε την πιο πρόσφατη έκδοση του πυρήνα:

apt-get install linux-image-2.6-sidux-686 linux-headers-2.6-sidux-686

4. Τώρα επανεκκινήστε τον υπολογιστή σας και ξεκινήστε με τον νέο πυρήνα (πρέπει να αρχίζει ως πρώτος στην λίστα εξ ορισμού).

Εάν ο νέος πυρήνας σας βγάλει προβλήματα, μπορείτε να επανεκκινήσετε και να επιλέξετε έναν παλαιότερο πυρήνα.

Ενότητες (modules)

Για να βρείτε ποιές ενότητες χρειάζεστε, η ακόλουθη εντολή σας παραθέτει τον κατάλογο τρεχουσών διαθέσιμων ενοτήτων, αντιγράψτε αυτήν την γραμμή στο τερματικό σας:

apt-cache search 2.6.*.slh.*-sidux| awk '/modules/{print $1}'

Για να πάρετε μια πλήρη περιγραφή για κάθε ενότητα, αντιγράψτε αυτήν την γραμμή στην κονσόλα σας

apt-cache search 2.6.*.slh.*-sidux

Για να εγκαταστήσετε τις απαραίτητες ενότητες (για παράδειγμα virtual-ose, και -qc-usb):

apt-get install virtualbox-ose-modules-2.6.24-2.6.24.2.slh.7-sidux-686 (ΠΑΡΑΔΕΙΓΜΑ)
apt-get install qc-usb-modules-2.6.24-2.6.24.2.slh.7-sidux-686 (ΠΑΡΑΔΕΙΓΜΑ)

Για να ελέγξετε τις ενότητες που φορτώνονται στον πυρήνα:

ls /sys/module/
ή
cat /proc/modules

ΣΗΜΑΝΤΙΚΟ: εάν χρειάζεστε ορισμένες ενότητες οδηγών, πχ. για wlan, πρέπει να εγκαταστήσετε τις κατάλληλες .deb συσκευασίες. Για παράδειγμα, για τον Agere et131x ethernet adapter: 

apt-get install et131x-modules-*

dpkg -i et131x-modules*.deb χρησιμοποιείται μόνο εάν έχετε τα τοπικά debs ήδη, αλλά ακόμη και σε εκείνη την περίπτωση, εάν είναι στην apt-cache, το apt-get είναι μια καλύτερη επιλογή.

Δείτε επίσης μπαίνοντας online και ndiswrapper και για WIFI

Άλλοι μέθοδοι εγκατάστασης πυρήνα

siduxcc

Μπορείτε επίσης να εγκαταστήσετε τους πυρήνες με το γραφικό κέντρο ελέγχου siduxcc: το όποιος βρίσκεται KDE-Start-Menu>System>sidux Control Centre. Το siduxcc προσφέρει επίσης διάφορες άλλες γραφικές εφαρμογές διοίκησης συστημάτων

Αφαίρεση παλιών πυρήνων (kernel remover)

Μετά από την επιτυχή εγκατάσταση του νέου πυρήνα, οι παλαιοί πυρήνες μπορούν να διαγραφούν, εντούτοις όμως αυτό που συστήνεται είναι να τους κρατάτε για μερικές ημέρες σε περίπτωση που υπάρχουν προβλήματα και επομένως επανεκκινήστε τον υπολογιστή σας από έναν παλαιότερο πυρήνα όπως απαριθμείται στην grub screen

Οι παλαιοί πυρήνες μπορούν να αφαιρεθούν από το σύστημα. Μπορείτε να χρησιμοποιήσετε το script kernel-remover το όποιο είναι στο πακέτο sidux-kernelhacking. Αυτό το script πρέπει να εγκατασταθεί και σας επιτρέπει να ορίσετε ποιούς πυρήνες επιθυμείτε να αφαιρέσετε. 

apt-get update
apt-get install sidux-kernelhacking
kernel-remover

Εναλλακτικά, μπορείτε να χρησιμοποιήσετε το siduxcc, το οποίο χρησιμοποιεί το kernel-remover script.
