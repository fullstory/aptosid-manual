Meta-Packages

Κατά τη διάρκεια της εγκατάστασης στο σκληρό δίσκο μπορείτε επίσης να διαλέξετα μετα-πακέτα διαφόρων εφαρμογών και παιχνιδιών ώστε να συμπεριληφθούν στην εγκατάστασή σας.

Ο εγκαταστάτης των μετα-πακέτων είναι επίσης διαθέσιμος σε εγκατεστημένο σύστημα πηγαίνοντας στο Kmenu>Utilities>sidux-meta-installer

Η λίστα μέχρι στιγμής περιλαμβάνει:
Δίσκος: -Εργαλεία σάρωσης κατάτμησης και επιδιόρθωσης
Μόρφωση:
Παιχνίδια:
Γλώσσες: -Πακέτα γλωσσών για ρώσσικα, κροάτικα και ελληνικά
Πολυμέσα (Multimedia) - VDR:
Δίκτυο: -Tor και Privoxy
Κλειστού_Κώδικα (Non_free) : -Διάφοροι οδηγοί (drivers)
OpenOffice.org - (Lite ISO)
Παλαιοί_Υπολογιστές: -Ελαφριές (Lightweight) εφαρμογές και διαχειριστές παραθύρων (window managers)
Office: -Gnumeic + Abiword, koffice και OpenOffice.org
Server: -sidux LAMP stack
Διαχειριστές Παραθύρων:
- icewm- (lightweight desktop)
- Fluxbox- (lightweight desktop)
- fvwm-crystal- (lightweight desktop)
- Xfce4- (lightweight desktop)

Μετα-εγκαταστάτης και Proxies

Εάν ο υπολογιστής σας συνδέεται στο διαδίκτυο μέσω HTTP-Proxy, μπορεί να δυσκολευτείτε να τρέξετε τον μέτα-εγκαταστάτη. Ο μέτα-εγκαταστάτης πρέπει να εκτελεστεί από τον διαχειριστή (root).

Θα πρέπει να αλλάξετε τις proxy ρυθμίσεις σας μέσα στο apt.conf αρχείο. Υπάρχουν 2 μεθόδοι να χρησιμοποιήσετε: Η πρώτη μέθοδος είναι καλύτερη γιατί είναι μόνιμη:

Μέθοδος 1

Καθορίστε τον proxy στο apt.conf αρχείο, επεξεργαστείτε το ως root με το kwrite /etc/apt/apt.conf και προσθέστε τα ακοόλουθα:

Acquire{
HTTP::proxy "http://192.168.1.1:3128";
FTP::proxy "http://192.168.1.1:3128";
}

Μέθοδος 2

Ανοίχτε μια κονσόλα και χρησιμοποιείστε το sux για να γίνετε root και να ρυθμίσετε τις μεταβλητές περιβάλλοντος(environment variables) http_proxy και ftp_proxy και στη συνέχεια εκτελέστε τον μετα-εγκαταστάτη στην κονσόλα που ορίσατε αυτές τις μεταβλητές αλλίως δεν θα έχουν καμία επίδραση. Αυτό θα χρειάζετε να το κάνετε κάθε φορά που χρειάζεστε πρόσβαση στον μετα-εγκαταστάτη:

export http_proxy="http://192.168.1.1:3128"
export ftp_proxy="ftp://192.168.1.1:3128/"
