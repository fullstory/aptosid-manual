Release notes for sidux-2009-03 Momos on 2009-11-11

For "Μώμος" (Momos), development had to take massive upstream changes for sysvinit/ insserv, X.org— and device detection, as well as sidux infrastructure changes into account, but also concentrates on furthering the KDE4 integration into sidux and upgrading to kernel 2.6.31. Subsequently it is the first sidux release to ship without KDE3 or qt3 packages.

Kernel 2.6.31 doesn't only improve and stabilise hardware support for newer devices, it also provides initial support for USB 3.0 devices and the Creative Labs Sound-Blaster-X-Fi soundcard. Further advancements include an experimental driver for RaLink rt2770, rt2870 and rt3070 802.11b/g/n USB wlan cards (rt2800usb) and improved powersaving modes for mac80211 based wlan drivers. For the netbook arena new fan control mechanisms and broadened webcam support have become available.

As usual, a large number of individual functionality enhancements and bugfixes have been applied to the full package line up and also to the packages that are sidux specific, in order to improve the general distribution infrastructure.

Υποδείξεις για άτομα που κάνουν αναβάθμιση:

Το sidux προσπαθεί να παρέχει ένα απλό μονοπάτι αναβάθμισης του debian και των πακέτων του sidux, όμως μένουν μερικά πράγματα τα οποία θα μπορούσαν να βελτιωθούν κάνοντας ορισμένες αλλαγές χειροκίνητα.

crda

Το νέο regulatory domain framework για cfg80211 βασισμένες wlan κάρτες χρειάζεται το crda για να επιτρέψει την πρόσβαση σε ασύρματα κανάλια ψηλότερα του ch11 (2.4 GHz ή τα 5 GHz αντίστοιχα τους), χωρίς το crda μόνο τα "world domain" (κανάλια 1-11) είναι προσβάσιμα. Προηγούμενα hacks με την χρήση παραμέτρων module ("ieee80211_regdom={US,EU,JP}") δεν προσφέρονται πλέον και αποτρέπουν την φόρτωση του cfg80211, παρακαλώ αφαιρέστε χειροκίνητα τυχών module-init-tools overrides ("grep -R ieee80211_regdom /etc/modprobe.d/" για να τα βρείτε).

# apt-get update apt-get install crda

Ρυθμίσεις ζώνης ώρας (Timezone)

Παλαιότερες εκδόσεις του tzdata (αυτές που υπάρχουν στις παλαιότερες εκδόσεις του sidux) δεν ρύθμιζαν σωστά την ζώνη ώρας. Παρακαλώ ξανά ρυθμίστε την ζώνη ώρας – ασχέτως αν φαίνεται να δουλεύει – με:

dpkg-reconfigure tzdata

Απαρχαιωμένα πακέτα

Σαν μέρος τις συνεχούς ανάπτυξης του sidux, ένας αριθμός πακέτων έχει αντικατασταθεί από καλύτερα εναλλακτικά πακέτα ή είναι πλέον περιττά.

ndiswrapper

Due to upcoming packaging changes (#517130 and Bits from the kernel team) combined with basically missing upstream development for the last 2 years, Μώμος is likely to be the last sidux version which can ship precompiled ndiswrapper modules. Please investigate if native FOSS modules are available for your wlan hardware or consider replacing it with better supported alternatives.

umtsmon

Continuing development of umtsmon is stagnating, without plans for a kde4 transition or non-root operations. wvdial currently seems to be the best alternative, as it doesn't ship with a graphical user interface, it's also usable outside of X.

virtualbox-ose

Due to recent packaging changes (#517130, Bits from the kernel team and #551531) in virtualbox-ose, it is unfortunately no longer possible to provide prebuilt kernel modules for virtualbox-ose in Μώμος, subsequently seamless guest integration can no longer be provided on the ISOs; it will remain user-installable. For systems supporting hardware virtualisation (AMD pacifica virtualisation extensions "svm" or Intel IVT (vanderpool) virtualisation extensions "vmx"), kvm is the preferred replacement.

Η πιο κάτω εντολή μπορεί να βρει πακέτα τα οποία δεν είναι διαθέσιμα σε κανένα ενεργό apt αποθετήριο και μπορεί να σας υποδείξει απαρχαιωμένα πακέτα. Προσοχή όμως στα πακέτα που εγκαταστάθηκαν χειροκίνητα. Το apt-show-versions είναι ξεχωριστό πακέτο και δεν είναι κατ' ανάγκη προ εγκατεστημένο. 

$ apt-show-versions | awk '/No available version in archive/{print $1}'
