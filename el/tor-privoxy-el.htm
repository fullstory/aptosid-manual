Privoxy

Το Privoxy είναι ένα web proxy με προηγμένες δυνατότητες φιλτραρίσματος για την προστασία της ιδιωτικής ζωής (privacy), τροποποίησης των δεδομένων μιας ιστό σελίδας, διαχείρισης cookies, ελέγχου πρόσβασης και αφαίρεσης διαφημίσεων, banners, pop-ups και άλλων ενοχλητικών διαδικτυακών σκουπιδιών (junk). Το Privoxy έχει πολύ ευέλικτες ρυθμίσεις και μπορεί να ρυθμιστεί για να ικανοποιήσει τις προσωπικές σας ανάγκες και γούστα. Το Privoxy μπορεί να χρησιμοποιηθεί και σε stand-alone (όχι σε τοπικό δίκτυο) συστήματα και σε δίκτυα πολλών ατόμων (multi-user). Το Privoxy βασίζετε πάνω στο Internet Junkbuster (tm).

Εγκατάσταση του privoxy:

apt-get update
apt-get install privoxy

Μια εγκατάσταση προεπιλογής (default installation) θα πρέπει να παρέχει ένα λογικό σημείο εκκίνησης για τους περισσότερους. Αναμφίβολα θα υπάρξουν περιπτώσεις που θα θέλετε να προσαρμόσετε τις ρυθμίσεις αλλά αυτό μπορεί να αντιμετωπιστεί όταν το χρειαστείτε. Στις περισσότερες περιπτώσεις χρειάζεται λίγη ή καθόλου αρχική ρύθμιση.

Στο αρχείο ρύθμισης του privoxy, μπορεί να χρειαστεί να κάνετε uncomment (να φύγει το # από μπροστά) γραμμές. Αυτό θα εξαρτηθεί από τις προτιμήσεις σας και θα είναι διαφορετικό για κάθε χρήστη.

Tor

Το Tor είναι ένα δίκτυο από εικονικά τούνελ (virtual tunnels), που επιτρέπει σε ανθρώπους και ομάδες να βελτιώσουν την προστασία της ιδιωτικής ζωής (privacy) τους και την ασφάλεια τους στο διαδίκτυο. Επίσης επιτρέπει σε προγραμματιστές να δημιουργήσουν νέα εργαλεία επικοινωνίας με built-in χαρακτηριστικά προστασίας της ιδιωτικής ζωής. Το Tor παρέχει την βάση για μια γκάμα εφαρμογών που επιτρέπουν σε οργανισμούς και ιδιώτες να διαμοιράζονται πληροφορίες μέσω δημόσιων δικτύων χωρίς να θέτουν σε κίνδυνο την ιδιωτική τους ζωή (privacy).

Για να εγκαταστήσετε το Tor:

apt-get update
apt-get install tor

Το εκ-προεπιλογής torrc αρχείο θα πρέπει να δουλέψει χωρίς αλλαγές (out of the box) για τους περισσότερους χρήστες. Θα πρέπει να ρυθμίσετε το Privoxy για να χρησιμοποιεί το Tor στο internet, δείτε: Tor και περιήγηση διαδικτύου (στα Αγγλικά).

Θα πρέπει να ξέρετε ότι η ταχύτητα του διαδικτύου σας θα επηρεαστεί αρνητικά.

Για τον περιηγητή διαδικτύου Iceweasel, υπάρχει ένα Torbutton add-on.

Η τεκμηρίωση του Tor (στα Αγγλικά) παρέχει λεπτομερείς πληροφορίες για όλες τις πτυχές του Tor.

Το Vidalia είναι ένας γραφικός διαχειριστής του Tor και σας επιτρέπει να σταματήσετε και να ξεκινήσετε το Tor, να δείτε την κατάσταση (status) του Tor και να παρακολουθήσετε πόσο bandwidth χρησιμοποιεί το Tor.

apt-get update
apt-get install vidalia
