AntiVir PersonalEdition Classic για Linux

Οι ακόλουθες πληροφορίες προήλθαν από πόστ σε φόρουμ

Για να εγκαταστήσετε το antvir PE Linux παρακαλώ κάνετε τα ακόλουθα βήματα:

1. σαν διαχειριστής (root)
# cd /tmp

2. Παρακαλώ κατεβάστε από εδώ :
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. Εξάγετε το κατεβασμένο συμπιεσμένο αρχείο (tarfile) 
# tar xvpzf antivir-workstation-pers.tar.gz

4. Αλλαγή καταλόγου
# cd antivir-workstation-pers-2.1.9-18
(ή οτιδήποτε αριθμός έκδοσης ισχύει τότε)

5. Τρέξτε το install script και ακολουθήστε τις οδηγίες εγκατάστασης 
./install

6. Αφαιρέστε το παλαιό antivir.conf 
# mv /etc/antivir.conf /etc/antivir.conf.old

Παρακαλώ μην ξεχάσετε να ενημερώσετε (update) το antivir
# antivir --update

rkhunter

ο ανιχνευτής (scanner) rootkit rkhunter είναι ένα ανιχνευτικό εργαλείο το οποίο βοηθά στο να είστε σίγουρος ότι το σύστημα σας να είναι καθαρό από δυσάρεστες εφαρμογές. Αυτό το εργαλείο ανιχνεύει τα rootkits, τα backdoors και τους τοπικούς εκμεταλλευτές (local exploits) τρέχοντας τεστ όπως:
- MD5 hash compare
- Ψάχνει για προεπιλεγμένα αρχεία που χρησιμοποιούνται από rootkits
- Λανθασμένα δικαιώματα (permissions) σε δυαδικά (binaries) αρχεία
- Ψάχνει για ύποπτα strings στις ενότητες (modules) LKM και KLD
- Ψάχνει για κρυμμένα αρχεία
- Προαιρετική ανίχνευση μέσα σε plaintext και τα δυαδικά αρχεία
- 

apt-get update
apt-get install rkhunter
rkhunter --update

Το rkhunter θα ρωτήσει επίσης εάν επιθυμείτε την οργάνωση ενός cron για ανίχνευση σε τακτά διαστήματα

Για ανίχνευση χρησιμοποιώντας το rkhunter 
rkhunter -c

Παρακαλώ διαβάστε τις σελίδες εγχειριδίου για μια πλήρη επεξήγηση όλων των επιλογών:

man rkhunter

chkrootkit

το chkrootkit είναι ένα εργαλείο για να ελέγξετε τοπικά για τα σημάδια ενός rootkit.

apt-get install chkrootkit

Χρησιμοποιώντας το chkrootkit για ανίχνευση 

chkrootkit

ελέγχει για αυτούς τους τύπους απειλών:

ifpromisc.c
ελέγχει εάν η διεπαφή είναι στο promiscuous mode. 
chklastlog.c
ελέγχει για διαγραφές μέσα στο lastlog 

chkwtmp.c
ελέγχει για διαγραφές μέσα στο wtmp

chkproc.c
ελέγχει για σημάδια ενός LKM δούρειου ίππου (trojan)
strings.c
γρήγορη και βρώμικη (quick and dirty) αντικατάσταση σειρών (string)
chkutmp.c
ελέγχει για διαγραφές μέσα στο utmp
