Cheat codes

If the possible values are listed in the "Values" field of the table, they must be appended to the "cheatcode" with an = sign. For example, if 1280x1024 was the desired value for the screen cheatcode, then you would type screen=1280x1024 into the grub boot prompt, or for xmodules xmodule=vesa, or for acpi, acpi=off

Cheatcode Value Description
acpi off turn ACPI completely off. Advanced Configuration and Power Interface (ACPI) is an open industry specification co-developed by Intel (TM), Microsoft (TM) and Toshiba (TM), that defines an operating system-directed power management and to make advanced configuration architectures.
oldboot needed ACPI parts to boot remain active, others are turned off
force force ACPI on
apm off power-off
desktop kde choose your desktop
fluxbox
gnome
icewm
xfce
dpi auto or XX set desired Dots Per Inch for your display (e.g. my 15" 1600x1200 laptop is 1600/(15*(4/5)) = 133 so dpi=133)
failsafe minimal auto detection
fromiso some/path/sidux.iso like fromhd, but uses the sidux iso image. Also does not require a cd to boot from, only requires the correct kernel and initrd/minirt
hostname whateveryourhostnameis hostname, changes hostname of live-cd system
hsync 80 (e.g.) horizontal monitor refresh rate (kHz)
irqpoll use IRQ poll
md5sum test integrity of medium
mem 128MB ram size to be used
noacpi disable acpi Advanced Programmable Interrupt Controller (APIC).
noagp disable agp
noapic disable apic
noddc disable automatic Direct Digital Calibration (attempts to configure your monitor settings)
nodhcp disable automatic use of Dynamic Host Configuration Protocol (attempts to automatically setup your ethernet connections).
nodma disable the use of Direct Memory Access for your drives
noeject do not prompt to eject the cd on shutdown
nofloppy disable floppy
nointro skips the display of index.html on live desktop startup
noisapnpbios disable ISA Plug'n Play
nomce no message for an irrelevant error
noml discards all the modelines provided. asks display about settings to use
nocpufreq do not activate Speedstep/Powernow
nosmp disable Symetric Multi Processor support
notouchpad disable touchpad support
nowheel force PS/2 protokol for PS/2 mouse
pci noacpi do not use ACPI to route PCI interrupts
quiet don't list everything to console
screen 800x600 sets you screen resolution, 1024x768, 1600x1200 etc
toram copy CD to RAM and start from there
vga normal look here for vga codes: VGA Cheatcodes
vsync 60 (e.g.) vertical monitor refresh rate (Hz)
xmodule ati use given X-module
fbdev use given X-module
i810 use given X-module
mga use given X-module
nv use given X-module
radeon use given X-module
savage use given X-module
vesa use given X-module
xhrefresh 80 (e.g.) horizontal monitor refresh rate in X (kHz)
xvrefresh 60 (e.g.) vertical monitor refresh rate in X (Hz)
