Contenido del LiveCD

Debido a la complejidad de la ley, sidux solamente proveerá software dfsg-free en el liveCD, por favor verifica este enlace para información adicional sobre fuentes no-libres de firmware

  • El ISO es completamente basado en Debian Sid (actualizado el 2007-05-13) enriquecido y estabilizado con scripts y paquetes provisto por sidux.
  • Todo ISO viene con el kernel 2.6.21.1, which is based on the most recent vanilla kernel together with sidux enhanced patches.
  • ACPI y DMA están preseleccionada (puede ser desactivado con acpi=off y nodma respectivamente en la línea de comando de grub)
  • i686 optimisación - no apto para uso con CPUs más viejos!
  • 192 MB RAM requerido, 256 MB RAM recomendado
Aplicaciones y Herramientas
  • El liveCD de sidux contiene una variedad de programas y herramientas que cubre casi cualquier necesidad del usuario de escritorio en la vida cotidiana.
  • KDE 3.5.6 El escritorio gráfico del usuario de escritorio que con un clic del botón, permite acceso a programas y aplicaciones.
  • Para surfear en internet está el konqueror un navegador web con muchos extras: tiene un cliente FTP y ssh (fish) incorporado: muy fuera del ordinario en el mundo de los navegadores web, También esta incluido el Iceweasel
  • Para redes y conexiones a internet se encuentra: ndiswrapper junto con el wpa/wlan y ver nuestro documentación de WIFI internal. It now also includes driver support by mac80211 details are on main sidux site .
  • Si deseas particionar el disco duro, hemos incluido GParted y también el cfdisk. El Gparted ahora tiene la facilidad de redimensionar particiones NTFS!
  • Herramientas diagnosticas, como el Memtest86+, una herramienta avanzada de diagnosis y comprobación de memoria
