Quemando CD/DVD sin GUI

ESTE ES MUY IMPORTANTE: sidux, como un Live-CD de Linux, es basado en tecnología de alta compresión, y por eso, cuidado especial es requerido al momento de quemar el imagen ISO. Sólo usa media de CD [o DVD+RW] de alta calidad, y quemarlo en modo-DAO/SAO (disco/sesión-a-la-vez) y a la velocidad más bajo por el tipo de disco, idealmente 8x]

burniso

Si desea quemar un CD/DVD, no es necesario un GUI para hacerlo.

Los problemas quemando mayormente ocurre con el frontend (k3b, nautilus, arson) no tanto con el backend (growisofs, wodim o cdrdao).

sidux provee un script listo para quemar un ISO de sidux, llamada 'burniso'.

burniso se quema imagenes ISO en media, utilizando el modo Disk-At-Once y una velocidad preseleccionada de 8x, a través del wodim. 

apt-get install sidux-scripts

como $User:

$ cd /dir/que_contiene/su/ISO

$ burniso

Todos los imagenes ISO en el directorio corriente serán mostrado, y después de la selección de un ISO, empieza a quemar el imágen, entonces asegurar que tiene un media insertado.

Que dispositivos a usar, como $user:

Para dispositivos ATAPI podemos averiguar:

wodim:

$ wodim --devices
wodim: Sobrevista de drives acesibles (2 encontrado) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Otras alternativas son:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

y

$ wodim -prcap

Aquí hay algunos ejemplos útil:

Info sobre un CD/DVD en blanco:
$ wodim dev=/dev/scd0 -atip

ó

$ cdrdao disk-info --device ATA:1,0,0

ó, como se menciono arriba:

cdrdao disk-info --device ATA:1,0,0 )
Borrar un disco re-escribible:
$ wodim -blank=fast -v dev=/dev/scd0

ó

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Clonar un CD:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Clonar un CD sobre la marcha:
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Crear un CD de audio desde archivos wav, con velocidad 12x:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav
Quemar un CD desde archivos bin/cue:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Quemar un imagen ISO:
$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

Si recibes un error driveropts, es porque el burnfree está deprecado para algunas dispositivos, entonces útiliza:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

ó

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso
Crear un imagen ISO con todos los archivos (y subdirectorios) de un directorio.

Este puede ser quemado con el comando dado arriba (Quemar imagen ISO):

$ genisoimage -o myImage.iso -r -J -l directory

Con un quemador DVD, también puede usar growisofs para quemar en DVD, como quemar un imagen ISO en DVD:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Quemar archivos múltiples a un DVD:
$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...
Si hay espacio libre en el DVD, puede agregar más archivos:
$ growisofs -M /dev/dvd -R -J otro archivo y otro archivo...
Para finalizar la sesión, usa el comando:
$ growisofs -M /dev/dvd=/dev/zero
