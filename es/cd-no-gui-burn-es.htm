Quemando CD/DVD sin GUI

ESTO ES MUY IMPORTANTE: sidux, como Live-CD de Linux, está basado en tecnología de alta compresión, y por eso se requiere un cuidado especial en el momento de quemar la imagen ISO. Sólo use un CD [o DVD+RW] de alta calidad, y quémelo en modo-DAO/SAO (disco/sesión-a-la-vez) y a la velocidad más baja según el tipo de disco, idealmente a no más de 8x

burniso

Si desea quemar un CD/DVD, no es necesario un GUI (interfaz gráfica) para hacerlo.

Los problemas quemando mayormente ocurren con el frontend (k3b, nautilus, arson) no tanto con el backend (growisofs, wodim o cdrdao).

sidux provee un script listo para quemar un ISO de sidux, llamado 'burniso'.

burniso quema imagenes ISO en CDs o DVDs utilizando el modo Disk-At-Once y con una velocidad preseleccionada de 8x, a través del wodim. 

apt-get install sidux-scripts

como $User:

$ cd /dir/que_contiene/su/ISO

$ burniso

Todas las imágenes ISO en el directorio corriente serán mostradas, y después de haber seleccionado un ISO, empieza a quemar la imagen, por lo que tiene que estar seguro de que tiene un disco insertado.

Que dispositivos a usar, como $user:

Para comprobar qué opciones y capacidades en dispositivos ATAPI:

wodim:

$ wodim --devices
wodim: Detalle de drives acesibles (2 encontrado) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Otras alternativas son:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

y

$ wodim -prcap

Aquí hay algunos ejemplos útiles:

Info sobre un CD/DVD en blanco:
$ wodim dev=/dev/scd0 -atip

ó

$ cdrdao disk-info --device ATA:1,0,0

ó, como se menciona arriba:

cdrdao disk-info --device ATA:1,0,0 )
Borrar un disco reescribible:
$ wodim -blank=fast -v dev=/dev/scd0

ó

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Clonar un CD:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Clonar un CD sobre la marcha:
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Crear un CD de audio desde archivos wav, con velocidad 12x:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav
Quemar un CD desde archivos bin/cue:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Quemar una imagen ISO:
$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

Si recibe un error driveropts, es porque el burnfree no ha sido cargado por algunos dispositivos, entonces haga:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

ó

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso
Crear una imagen ISO con todos los archivos (y subdirectorios) de un directorio.

Puede ser hecho con el comando indicado arriba (Quemar imagen ISO):

$ genisoimage -o myImage.iso -r -J -l directory

Si tiene un quemador de DVD también puede usar growisofs para quemar DVD, igual como quemar un imagen ISO en DVD:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Quemar archivos múltiples a un DVD:
$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...
Si hay espacio libre en el DVD, puede agregar más archivos:
$ growisofs -M /dev/dvd -R -J otro archivo y otro archivo...
Para finalizar la sesión, use el comando:
$ growisofs -M /dev/dvd=/dev/zero
