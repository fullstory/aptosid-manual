Cheatcodes

If the possible values are listed in the "Values" field of the table, they must be appended to the "cheatcode" with an = sign. For example, if 1280x1024 was the desired value for the screen cheatcode, then you would type screen=1280x1024 into the grub boot prompt, or for xmodules xmodule=vesa, or for acpi, acpi=off

Cheatcode Valor Descripción
acpi off apagar ACPI completamente
oldboot partes de ACPI requerido para arrancar queda activa, otras están apagada
force forzar encendido de ACPI
apm off corriente apagado
desktop kde escoja su escritorio (desktop)
fluxbox
gnome
icewm
xfce
dpi auto or XX fija Puntos por pulgada deseado para su monitor (por ejemplo, mi laptop de 15" con 1600x1200 es 1600/(15*(4/5)) = 133 entonces dpi=133
failsafe detección automática mínimo
fromhd /dev/hda1 (hda2,...) arranca desde un imagen CD previamente copiado en la partición indicado (la partición debe contener un archivo sidux/sidux)
fromiso algún/ruta/sidux.iso como el fromhd, pero se uso el imagen ISO de sidux. Tampoco requiere un CD para arrancar, solamente el kernel correcto y initrd/minirt
hostname whatever-your-host-name-is hostname, cambio del nombre del hospedaje de el sistema Live-CD
hsync 80 (eg) tasa de renovación horizontal del monitor (kHz)
irqpoll usa polling de IRQ
md5sum comprobar la integridad de la media
mem 128MB (zB) cuanto RAM va a ser usado
noacpi desactivar acpi
noagp desactivar agp
noapic desactivar apic
noaudio desactivar audio
noddc desactivar el uso automática del Direct Digital Calibration (intenta configurar los propiedades el monitor)
nodhcp desactivar el uso automática del Dynamic Host Configuration Protocol (intenta de configurar automáticamente las conexiones a la tarjeta ethernet).
nodma desactivar el uso de Direct Memory Access para los dispositivos de almacenamiento
noeject No pedir la eyección del CD al cerrar la aplicación
nofloppy desactivar el floppy
nointro omitir el desplegué de index.html en el arranque del LiveCD
noisapnpbios desactivar ISA Plug'n Play
nomce no muestra mensajes para errores irrelevantes
noml descartar todos los modelines suministrados. pregunta la pantalla sobre los configuraciones de usar
nocpufreq no activa el Speedstep/Powernow
nosmp desactivar apoyo para el Symetric Multi Processor
nosound desactivar apoyo para apoyo del sonido
notouchpad desactivar apoyo para el touchpad
nowheel forzar el protocolo PS/2 para el mouse PS/2
pci noacpi no se usa ACPI para enrutar los interruptores PCI
quiet no se lista todo en la consola
resume /dev/hdaX permitir el modo de suspender
screen 800x600 fija la resolución de la pantalla, 1024x768, 1600x1200, etc.
toram copiar el CD a RAM y ejecuta desde allí
vga normal busca aquí para códigos vga: VGA Cheatcodes
vsync 60 tasa de renovación vertical del monitor(Hz)
xmodule ati usar el modulo dado
fbdev usar el modulo dado
i810 usar el modulo dado
mga usar el modulo dado
nv usar el modulo dado
radeon usar el modulo dado
savage usar el modulo dado
vesa usar el modulo dado
xhrefresh 80 tasa de renovación horizontal del monitor (kHz)
xvrefresh 60 tasa de renovación vertical del monitor
