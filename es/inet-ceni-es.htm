Conectándose en línea / como configurar la red con Ceni

Probablemente necesitará firmware no libre que esté en un lápiz USB para instalar el sistema operativo. Vea, por favor, Datos para hardware con necesidades no-libres.

Si tiene un servidor DHCP en su LAN al que está conectado su computador mientras se inicia, la configuración de red será hecha automaticamente. Si no, necesitará ejecutar el programa Ceni: haga clic en Kmenu>aptosid>Internet>Ceni. Se abrirá un terminal/consola donde le pedirá su clave de root, (en el LiveISO no hay password fijado).

Lo más rápido para acceder a Ceni es abrir un terminal, y escribir:

ceni

y le pide su clave de root.

WiFi, Uno de los puntos fuertes de Ceni es su fabulosa habilidad para configurar tarjetas inalambricas WiFi - Guía de Configuración Básica:

Conectando con un módem telefónico de 56k

KDE tiene un programa para conectarse mediante un módem telefónico llamado KPPP, Herramienta Telefónica para Internet, que se encuentra en el menú principal bajo la rúbrica Internet.

Esta aplicación tiene un manual de ayuda incorporado y contiene una guía para configurar el módem que le permitirá conectarse en línea.

Muros de Fuego (Firewalls)

Un muro de fuego usualmente no es necesario si está detras de un router bien configurado, sin embargo, juega un rol importante en la seguridad si necesita conectar a internet con un modem adsl usb o a través de un modem telefónico:

apt-get install guarddog
ó
apt-get install firestarter

Guarddog - es un utilidad de firewall del ambiente KDE amigable con el usuario. La mejor manera de empezar es, primero, leer unos tutoriales cortos acerca de cómo configurarlo.

Firestarter - Un firewall de fuente abierta para Linux con interfaz gráfica. [Cuando salga de la interfaz gráfica, el firewall sigue funcionando en el fondo.]
