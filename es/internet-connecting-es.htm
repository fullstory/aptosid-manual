Conectándose en línea / como configurar el red with Ceni

Si tiene us servidor DHCP en su LAN y su computador esta conectada a el mientras se inicia, las configuraciones de la red debe ser configurado automaticamente - sino, necesita ejecutar el Ceni, haga clic en Kmenu>sidux>Internet>Ceni. Este se abre un terminal/consola donde le pide su clave raiz, (en un liveCD no hay password fijado).

Lo más rápido para accesar a Ceni es de abrir un terminal, y tipear:

ceni

y se le pide su clave de raíz.

WiFi, Una de los fuertes de Ceni es su habilidad fabuloso de configurar tarjetas inalambricas WiFi - Basic setup guide:

También puede hacer todo esto desde el GUI utilizando siduxcc, así: KDE-Start-Menu>System>sidux Control Centre

Conectando con un módem telefónico de 56k

KDE tiene un programa como frente para un módem telefónico llamado KPPP, Herramienta Telefónica para el Internet, que se encuentra en el menú principal bajo el Internet.

Esta aplicación tiene un manual de ayuda incorporado y esta contiene una guía comprensiva para configurar el módem para permitirle conectarse en línea.

Muros de Fuego (Firewalls)

Un muro de fuego usualmente no es necesario si estas detras de un router properly configured, sin embargo, juega un rol importante en la seguridad si necesita conectar a internet con un modem adsl usb o a través de un modem telefonica:

apt-get install guarddog
ó
apt-get install firestarter

Guarddog - es un utilidad de muro de fuego amigable al usuario del ambiente KDE. La mejor manera de empezar es, primera, leer unas tutoriales cortos acerca de como configurarlo.

Firestarter - Un muro de fuego de fuente abierto para Linux con ambiente gráfica. [Cuando se salga del interfaz gráfica, el muro de fuego sigue funcionando en el fondo.]
