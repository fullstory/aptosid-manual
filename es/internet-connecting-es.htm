Conectándose en línea / como configurar el red with Ceni

If you have a DHCP server in your LAN and your computer is connected to it while booting up your network-settings should be configured automatically - otherwise you need to start Ceni Click Kmenu>sidux>Internet>Ceni. This will open a terminal/konsole whereupon it will ask you for your root password, ( on a live-Cd there is no password set)

The fast way to access Ceni is to open the terminal/konsole and type 

ceni

whereupon it will ask you for your root password.

One of the Ceni strengths is its fabulous ability to configure wireless cards:

You can also do this with the GUI control centre siduxcc: KDE-Start-Menu>System>sidux Control Centre

Conectando con un módem telefónico de 56k

KDE tiene un programa como frente para un módem telefónico llamado KPPP, Herramienta Telefónica para el Internet, que se encuentra en el menú principal bajo el Internet.

Esta aplicación tiene un manual de ayuda incorporado y esta contiene una guía comprensiva para configurar el módem para permitirle conectarse en línea.

CortaFuegos (Firewalls)

Un CortaFuego usualmente no es necesario si estas detras de un router properly configured, sin embargo, juega un rol importante en la seguridad si necesita conectar a internet con un modem adsl usb o a través de un modem telefonica:

apt-get install guarddog
ó
apt-get install firestarter

Guarddog - es un utilidad de cortafuego amigable al usuario del ambiente KDE. La mejor manera de empezar es leer unas tutoriales cortos acerca de como configurar su primera.

Firestarter - Un cortafuego de fuente abierto para Linux con ambiente gráfica. [Cuando se salga del interfaz gráfica, el cortafuegos sigue funcionando en el fondo.]
