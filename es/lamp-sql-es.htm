Configuración de Mysql

Asegurar que tiene los siguientes líneas, quizas quitar el marco de comentario, en el archivo de configuración apache2.conf (o en el archivo conf.d/php.ini):

extension=php_mysql.so

entonces, este será

extension=php_mysql.so

El archivo de configuración esta ubicada en: /etc/mysql/my.cnf

Por preselección, Mysql crear el usuario como root y ejecuta sin clave de raíz. Para cambiar el clave raíz:

mysql> USE mysql;
mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE user='root';
mysql> FLUSH PRIVILEGES;
Para crear un nuevo usuario

Nunce debe usar el clave raíz como un login normal, entonces crear un usuario para conectar al base de datos mysql a través de un script PHP. Alternativamente, puede agregar usuarios a un mysql base de datos a través de un panel de control como es o phpMyAdmin para facilmente crear o asignar permisos al base de datos a los usuarios.

Para accesar el interface web, tipear lo siguiente en el browser y oprima Enter:

http://ipaddress/phpmyadmin/

Cuando aparece el prompt para el nombre de usuario y el clave, entra el root del base de datos como usuario y el clave (por preselección no hay clave para el usuario root del base de datos)

Fuentes de información:
