Buscando fuera de línea

Conjunto con el desarrollador de Recoll, el Manual de sidux tiene unas configuraciones especiales como parte del paquete. Recoll es una herramienta de busqueda de todo texto personal para Unix/Linux. Es basada en una maquina Xapian muy fuerte, para lo cual provee un frente con bastante carácteristicas pero con una interfaz gráfica de QT, facil de usar. A very special thank you to Recoll.

Recoll se llama Personal Search Tool en el Menu de Debian de KDE.

1er Paso

En este momento el manual está disponible en los siguientes idiomas:
da, de, el, en, es, fr, hr, it, ja, nl, pl, pt-br, ro, ru.

apt-get update
apt-get install sidux-manual-xx (donde -xx es su idioma)
2do Paso
apt-get install recoll
3er Paso
  • En la primera ejecución, Recoll preguntará si desea construir un base de datos, haga clic en Cancel, entonces un dialogo de configuración de indice se aparece.
  • En los paramaters Globales - caja de Top directorios, haga clic en el simbolo + (más) y vaya a /usr/share/sidux-manual, haga clic en el idioma preferido y haga clic en OK. Proxima, haga clic en el simbolo - (menos), y borrar cualquier otras carpetas, incluyendo el ~ (tilde).
  • Finalmente, escoge File > Update Index y busca el topic deseada en el Manual de sidux.

Si necesita reiniciar recoll, debido a actualizaciones al manual con apt-get, (como usuario $ en una konsola):

recollindex -z

Entonces: Kmenu > Debian > Applications > Data Management > Personal Search Tool y en Recoll File > Update Index.
