Lista de fuentes y controladores firmware: no-libres

Lista de fuentes

Puesto que el ISO aptosid contiene solamente software libre dfsg, usted probablemente querrá añadir contrib non-free a su /etc/apt/sources.list.d/debian.list :

mcedit /etc/apt/sources.list.d/debian.list

Las líneas en su máquina deberían ser algo así, dependiendo del espejo que haya escogido, (vea Lista de los servidores Debian y del estado actual de los espejos):

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Después de modificar sus fuentes, necesita hacer apt-get update antes de buscar o instalar paquetes nuevos, (es una buena costumbre hacer apt-get update antes de instalar cualquier paquete).

Detección de firmware no-libre

Para instalar firmware no-libre, (se supone que contrib non-free se ha añadido a las fuentes):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

El firmware necesario se puede determinar con la siguiente enumeración de dispositivos/ firmware o usando el script fw-detect (que se encuentra en aptosid-scripts).

$ fw-detect

La salida de la orden fw-detect describe las órdenes necesarias para instalar y activar el firmware:

Ejemplo:

#Controlador detectado que necesita firmware para funcionar
#Siga estas instucciones para obtener el firmware correcto
# y active el controlador zd1211rw:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Para instalar firmware desde los repositorios git necesitará:

apt-get install git-core

Firmware no-libre en ficheros debs en una llave USB

Si necesita obtener por adelantado el firmware .deb, para ponerlo en una llave USB para transferir los archivos a otro ordenador, puede descargarlos, ya sea como un fichero zip o uno tar.gz desde http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/ y descomprimirlos en una carpeta llamada firmware, y verá un cierto número de ficheros .deb.

A continuación descargue http://packages.debian.org/sid/firmware-linux-free*.deb añádalo a todos los otros ficheros .deb en esa carpeta. Después páselos mediante la llave al ordenador que necesita el firmware non-free y haga:

dpkg -i firmware-linux-nonfree*.deb

NOTA: Estamos investigando si podemos proporcionar paquetes para algunos de estos dispositivos, pero su estado legal no es siempre fácil de entender.
