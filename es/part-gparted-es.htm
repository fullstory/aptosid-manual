Como particionar el DD (Disco Duro) (utilizando GParted)

AVISO: para nombramiento de discos Por favor referirse a UUID, "label" de Partición y fstab, en vista de que sidux ahora por preselección se utiliza el UUID

Herramientas de particiones puede requerir la clave de root (raíz). Si no ha sido fijado, sóla oprima 'Entra', Ver: Live Mode (Modo vivo)

Prologo

Una partición tiene que tener un sistema de archivos. Linux sabe como usar varios sistemas de archivos. Hay Ext3, Ext2, Reiserfs y por los más experimentados el XFS y JFS. Ext2 es útil como una formato de almacenamiento en vista de que hay una intérprete (driver) disponible para Windows para intercambiar datos. [Linux puede leer y escribir en sistemas FAT.]

Para uso normal, recomendamos el sistema de archivos ext3, que es el sistema de archivos pre-selecciónada por sidux y es bien mantenido.

La versión usada en este documento de GParted es el 0.3.1.1. La siguiente información ha sido extraído de la documentación Oficial de GParted y ha sido modificado para servir de guía al uso de Gparted con sidux, y no debe ser tomado como una guía definitiva. E&OE

Para la documentación completa, ver GParted

Utilizando GParted

Crear y manejar particiones no es algo que se realiza todos los días. Por eso, es una buena idea leer esta guía por lo menos una vez, para estar cómodo con los conceptos y algunas de los paneles que aparezcarán.

  • Información general

    Cuando se ejecuta GPARTED, una ventana se abriera y todos los dispositivos se escanearan.

    Si pinchas en el clave "menú" (arriba a la izquierdo), un menú saliente se presenta. Puede seleccionar para re-leer la desplegué de los dispositivos en su sistema. También, un atajo de teclado esta disponible para fácilmente re-leer la información de la pantalla. Con una segundo selección, puede escoger el disco duro cuyo particiones desea modificar. Este es útil si tiene más de una disco duro. La tercer opción baja el clave Gparted en el menú esta utilizado para obtener más información.

  • Editar

    Se abre una ventana desde lo cual puede ver los tipos de archivos soportados y algunas opciones para ediciones de particiones.

    El menú es el segundo de la izquierda. Se muestra dos funciones en gris, los cuales serán muy útil: Deshacer (Undo) y Aplicar (Apply) y también permite accesar/ver otras áreas. Estas opciones también puede ser vistos:

  • Ver (View)

    • Información de dispositivos

      El panel de Información del Disco Duro muestra detalles acerca del disco duro, tales como el Modelo, Tamaño, etcétera. Este panel es muy útil en un sistema con múltiples discos duros, donde esta información sera utilizado para confirmar que el disco duro siendo examinado es el disco que desea modificar.

    • Operaciones pendientes

      A la pie de la ventana es una lista de operaciones pendientes. Este información es útil porque provee una información de la partición que se harán cambios.

  • Dispositivo

    El dispositivo permite fijar una etiqueta al disco, si la etiqueta del disco que ya tiene no es apropiado, puede cambiarlo utilizando este opción.

  • Menú de la partición

    El menú es de alta importancia. Le permite realizar muchas operaciones, algunas son peligrosos.

    Borrar (Delete) es seleccionado si decide borrar una partición. Para realizar la borrada, primero tiene que seleccionar la partición.

    Redimensionar/Mover es una función útil.

  • Creando una nueva partición

    En la barra de herramientas, el botón Nuevo (New) permite crear una nueva partición, si ya, ha seleccionado una área no alocado. Una nueva ventana aparece, y puede escoger el tamaño deseado, si la partición sera una Primario, Extendido o una particón Logical, y la sistema de archivos.

  • ¿Si hace un error?

    Si ha cometido un error, puede usar el botón de Borrar (Delete) para borrar la partición escogido, o si todavía no han aplicado su decisión de seguir adelante, puede usar el botón de Deshacer (Undo).

  • Redimensionar/Mover

    Cuando desea redimensionar una partición que ha seleccionado, pinchar en el botón de Redimensionar/Mover (Resize/Move): y una nueva ventana emergente aparecer. Utiliza el mouse para reducir (o aumentar) la partición o si deseas, puede usar las flechas.

    Después de que ha dado el comando de Redimensionar, pinchar en Aplicar, por que ningún operaciones afecta el disco duro hasta que pinchar en Aplicar (Apply).

    La duración de las operaciones depende en el tamaño nuevo de la Partición.

    Después de modificar la tabla de particiones, por favor salga de sidux y re-arrancar su sistema en sidux de nuevo, para leer de nuevo la nueva tabla de particiones.

Redimensionando una partición NTFS

Después de redimensionar una partición NTFS, es necesario re-arrancar con MS Windows. NO haga otra operación en esa partición antes de arrancar o tendrá errores

  • Después del re-arranque en MS Windows, el sistema mostrará una pantalla especial, con una mensaje preguntando acerca de la consistencia de disco duro: Verificando el sistema de archivos en el disco C:
  • Deja que el AUTOCHK corra: NT necesita verificar su sistema de archivos después de la operación de redimensionamiento.
  • Después de este proceso, el computador, automáticamente se rearrancará por una segunda vez. Esto es para asegurar que todo funcione perfectamente.
  • Después del rearranque, el Windows XP estará bien, pero tiene que dejar que la sistema arranca completamente, llegando a la pantalla de hacer login a Windows!
