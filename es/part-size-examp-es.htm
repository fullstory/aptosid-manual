Tamaños de particiones y ejemplos

Para un uso normal, recomendamos el sistema de archivos ext3. Es el escogido por sidux y está bien mantenido y apoyado.

Con el Gestor de Particiones gparted los disco duros se puede particionar y/o formatear. Este programa tiene una interfaz gráfica amigable que se explica por sí misma.

Gparted también puede reducir o mover particiones y además puede manipular particiones NTFS [con una precaución: una vez alterada una partición NTFS, se tiene que reiniciar inmediatamente el sistema operativo antes de hacer cualquier otra operación.] Vea la documentación completa de gparted. Se pueden realizar cambios en particiones-ntfs con herramientas de software propietario tales como Partition Magic (TM) y Acronis (TM).

¡SIEMPRE, RESPALDE SUS DATOS!

Si una partición parece como montada, desmóntela, ya sea con un click derecho en el icono de la partición en el Gparted o a través de una termninal, por ejemplo:

umount /dev/sda1

La partición de intercambio puede ser desmontada en una terminal con: 

swapoff -a

En principio, 5 GB es más que suficiente para una instalación en un disco duro, pero no será muy agradable. Un minimo razonable para una instalación será de unos 12 GB. Para aquellos que sean nuevos en Linux, sugerimos solamente 2 particiones al principio: root y swap. Esto simplifica la primera instalación considerablemente. Luego establecerá más particiones para /home y otros datos.

Realmente debe tener una partición de intercambio (equivalente al swapfile de Windows pero es mucho más efectivo). Para uso normal, la partición swap debe ser 2 veces la cantidad de RAM instalada.

Para intercambiar datos con una instalación de Windows, debe utilizar vfat (fat32) o el ext2 en vista que ya existe un driver de MS Windows™ para leer ext2 [XFS no está soportado].

Quien quiera crear más de una partición deberá escribir sus nombres y características para tenerlos como referencia luego!

Aquí hay unos ejemplos sencillos para DD de diferentes tamaños y diferentes tipos de uso, todo redondeado.

Configuración de Pruebas - 60GB para un PC donde Windows requiere bastante espacio
Disco Tamaño Sist.Archivos Puntos Monturas/Sistema
sda1 20GB NTFS WindowsXP
sda5 30GB FAT32 Datos para Win y Linux
sda6 8GB ext3 /
sda7 500MB Swap Swap
TV y Video-PC - 200GB sin Windows para archivos grandes (ej. Video).
Disco Tamaño Sist.Archivos Puntos Monturas/Sistema
sda1 8GB ext3 /
sda5 6GB ext3 /home
sda6 5GB ext3 /var
sda7 178.5GB ext3 /video
sda8 1GB Swap Swap
Standard-PC con Windows 120GB para usar lo mejor de ambos mundos.
Disco Tamaño Sist.Archivos Puntos Monturas/Sistema
sda1 20GB NTFS WindowsXP
sda5 8GB ext3 /
sda6 20GB ext3 /home
sda7 70GB FAT32 Intercambio de datos WindowsXP-Linux
sda8 500MB Swap Swap
Juegos Linux - y Media-PC 160GB
Disco Tamaño Sist.Archivos Puntos Monturas/Sistema
sda1 18GB ext3 /
sda5 20GB ext3 /home
sda6 30GB ext3 /opt
sda7 90GB ext3 /data
sda8 2GB ext3 /var
sda9 500MB Swap Swap
Linux y Windows en 20 GB
Disco Tamaño Sist.Archivos Puntos Monturas/Sistema
sda1 5GB NTFS Win-System
sda2 10GB FAT32 Windows-Programas y Datos para Windows y Linux
sda5 5GB ext3 /
sda6 500MB Swap Swap

Esta reparto de espacio es recomendable cuando hay poco sitio en el DD (p. ej. en un ordenador portátil). Windows se instala solo en una partición con 3-5 GB. Todos los programas y archivos están en la segunda partición, que es más grande. Quien tenga más espacio puede hacer esta partición más grande o separar los programas Windows de los datos en 2 particiones.

En general

Hay muchas maneras de particionar el disco duro. Estos ejemplos deben ser suficientes para comenzar. Tiene sentido agregar un segundo disco duros para mejorar el rendimiento, comodidad y seguridad. En cualquier caso, recuerde que Windows siempre tiene que estar instalado en el primer disco duro!
