Meta-Paquetes

El instalador de meta-paquetes se instala de esta manera:

apt-get install install-meta

Es disponible entonces por ir a Kmenu > Utilities > metapackage-installer

The meta-packages installer should only be used if the system has been recently dist-upgraded, so that whenever you run it, it only installs the packages needed and doesn't initiate a partial dist-upgrade at an inopportune time.

Meta-Instalador y Proxies

Si el PC esta conectada al Internet via HTTP-Proxy, puede que tiene dificultad de ejecutar el meta-instalador. El meta-instalador tiene que ejecutarse con privilegios de root.

Necesita alterar la configuración de proxy en el archivo apt.conf. Hay 2 maneras de hacerlo: Método 1 es mejor, porque es permanente:

Método 1

Definir el proxy en el archivo apt.conf, editarlo como root con kwrite /etc/apt/apt.conf y agregar los siguiente:

Acquire{
HTTP::proxy "http://192.168.1.1:3128";
FTP::proxy "http://192.168.1.1:3128";
}

Método 2

Abrir una consola con sux para volverse raiz para fijar los variables del entorno, http_proxy y ftp_proxy, entonces llama al meta-installer en el terminal donde fijaste los variable de proxy, sino, no tendra efector. Tendra que hacer esto cada vez que desea acesar el Meta-instalador:

export http_proxy="http://192.168.1.1:3128"
export ftp_proxy="ftp://192.168.1.1:3128/"
