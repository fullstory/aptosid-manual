Privoxy

Privoxy es un proxy web con capabilidades avanzada de filtros para proteger privacidad, páginas emergentes, manejar cookies, controlar acceso, y remover publicidad, banners, ventanas emergentes (popups) y otra basura ofensivo de Internet. Privoxy tiene una configuración muy flexible y puede ser configurada para necesidades y gustos individuales. Privoxy tiene aplicación tanto en sistemas independientes y en redes multiusuarios. Privoxy es basada en el Internet Junkbuster(tm).

Para instalar el Privoxy:

apt-get update
apt-get install privoxy

Un punto de partido razonable para la mayoria sera con la instalación preseleccionada. Sin duda, hay ocasiones cuado desearas ajustar la configuración inicial, pero este puede ser resuleto cuando se presenta la necesidad. Poco a ningún configuración sera requerida en la mayoria de casos.

El uso de Privoxy y el Meta-instalador de sidux referirse aquí.

El manual completo de Privoxy con información de configuraciones avanzadas.

Tor

Tor es un red de tuneles virtuales que permite individuales y grupos mejorar su privacidad y seguridad en Internet. También habilitar que desarrolladores de software pueden crear nuevas herramientaas de comunicaciones con caracteristicas de privacidad incorporada. Tor proporciona la fundación para un rango de aplicaciones que permite organizaciones y individuales compartir informacion sobre redes publicas sin compromisar su privacidad.

Debe estar consciente que la velocidad de transmision por internet será afectado negativamente.

Para instalar Tor:

apt-get update
apt-get install tor

La documentación de Tor ofrece información comprehensivo sobre todos los aspectos de Tor.
