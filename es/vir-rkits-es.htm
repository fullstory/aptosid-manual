AntiVir PersonalEdition Classic para Linux

Lo siguiente información viene de un posteo en el foro

Para instalar el antiviro PE Linux, favor sigue los siguientes pasos:

1. como raíz
# cd /tmp

2. Favor, bajarlo desde aquí:
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. extraer los archivos desde el tar:
# tar xvpzf antivir-workstation-pers.tar.gz

4. cambiar el directorio:
# cd antivir-workstation-pers-2.1.9-18
(o cualquier número de versión es aplicable al momento)

5. Ejecuta el script de instalación: y sigue los instrucciones de instalación:
./install

6. remover el viejo antivir.conf:
# mv /etc/antivir.conf /etc/antivir.conf.old

Por favor, no olvide de actualizar:
# antivir --update

rkhunter

rkhunter es un escaneador para rootkits para asegurar que el sistema está libre de aplicaciones maliciosos. Esta herramienta busca rootkits, puertos traseros y explotas locales a través de pruebas como esta:
- MD5 hash comparación
- Buscada por archivos utilizada por rootkits
- Permisiones equivocadas para archivos binarios
- Búsqueda de cadenas sospechosas en los módulos LKM y KLD
- Búsqueda de archivos ocultos
- Escanea opcionalmente dentro de archivos de texto y binarios

Rootkit Hunter fue lanzada como proyecto bajo licencia GPL y es de libre uso para todos.

apt-get update
apt-get install rkhunter
rkhunter --update

rkhunter además preguntas si deseas configurar un cron para escanear regularmente.

Para escanear utilizando el rkhunter
rkhunter -c

La página web de rkhunter

chkrootkit

chkrootkit es una herrramienta para buscar localmente por indicios de un rootkit.

apt-get install chkrootkit
Como escanear con el chkrootkit:
chkrootkit

chkrootkit busca para estos tipos de definiciones:

ifpromisc.c
revisa si el interfaz esta en un modo promiscuos.

chklastlog.c
revisa por eliminaciones de lastlog

chkwtmp.c
revisa por eliminaciones en el wtmp

chkproc.c
revisa para señales de troyanos LKM

chkdirs.c
revisa por señales de troyanos en LKM

strings.c
reemplazo rápido y poco elegante de cadenas

chkutmp.c
revisa por eliminaciones en el utmp
