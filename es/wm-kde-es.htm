KDE se traba

En tal situación, no siempre necesita el botón de reseteo. Eso puede dañar el sistema de archivos o llegar a una perdida de datos. De cualquier forma el sistema de archivos no será limpio después de un apagado duro (sistema de archivos no limpio)

Primero, intenta cambiar a una consola de texto Ctrl-Alt-F1 o rearrancar el servidor-X con Ctrl-Alt-Retroceso (backspace), (si ninguna de estos dos funciona, todavía hay esperanza):

La tecla PETSIS/SYSREQ (Impr-Pant/Print-Scr, a la derecha arriba del teclado) puede ayudar rearrancar limpiamente una sistema congelada.

La siguiente secuencia de combinaciones de tecladas puede dar buen resultado:
* ALT+PETSIS+R (debe regresar control del teclado)
* ALT+PETSIS+S (pide un sync)
* ALT+PETSIS+E (Envía "term" a todos procesos menos "init")
* ALT+PETSIS+I (Envía muere a todos procesos menos "init")
* ALT+PETSIS+U (sistemas de archivos están montada leer solamente, que prevea el fsck al rearrancar)
* ALT+PETSIS+B (rearranca la sistema, sin los pasos previo, sera como un apagón [hard reset]).

Es mejor de dar cada paso unas segundos para completar, terminando todos los procesos puede llevar algún tiempito. Las letras a recordar puede ser ayudada con este mnemotécnico: "Raising Skinny Elephants Is Utterly Boring".

Problemas con el Login bajo KDE

El contenido del directorio /tmp es normalmente vaciado en cada arranque, y algunos directorios y socketas requerido por el servidor-X también están borrado.

Normalmente, durante el proceso del arranque, el script x11-common de X-Org trata de recrear estas cosas.

Puede suceder que estos scripts no sean llamados durante el proceso del arranque. En este caso, puede recrear los enlaces necesitado con lo siguiente:

# X-ORG: # dpkg-reconfigure x11-common

Como activar o cambiar los temas de KDE

Todos estos cambios tiene que ser realizado como root.

Activar temas

Corrientemente, los temas-KDM no están activado al arranque en Debian.

Agregue
UseTheme=true
línea en el [X-*-Greeter]-stanza en su archivo  /etc/kde3/kdm/kdmrc.
Cambia temas

Si activado, KDM utiliza su tema de círculos, para cambiar esto, solamente cambia este línea con:

Theme=/usr/share/apps/kdm/themes/circles

a

Theme=/usr/share/apps/kdm/themes/
Desactivar temas

Si desea desactivar los temas solamente cambia a UseTheme=false.

NOTA: Tiene que reinicializa KDM después de realizar cambios a kdmrc.

Una manera simple de usar los temas de KDM es instalar kdetheme, como root:

 apt-get install kdmtheme

kdmtheme es una gestionador para temas de KDM. Este provee un Modulo de Control KDE (KCM) que permite la fácil instalación, desinstalación y cambio a las temas KDM.
