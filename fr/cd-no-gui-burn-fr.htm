Gravure de CD/DVD sans Interface Graphique

FAITES BIEN ATTENTION ! sidux, en tant que live-cd Linux, est basé sur un système de compression très puissant, et pour cette raison, une attention minutieuse doit être dédiée à la gravure soignée de l'image du CD.
N'utilisez que des supports CD ou CD-RW de qualité supérieure, et gravez à vitesse limitée, pas plus de 8x en mode DAO (Disk-At-Once) exclusivement.

burniso

Vous n'avez pas besoin d'une interface graphique pour graver un CD / DVD.

La plupart du temps les soucis de gravure se produisent avec l'interface (k3b) plutôt qu'avec les outils qu'elle pilote (growisofs, wodim ou cdrdao).

sidux fournit un script de gravure ISO prêt à l'emploi appelé 'burniso'.

burniso grave les fichiers ISO sur disque, en utilisant wodim en mode DAO (Disk-At-Once) à une vitesse de 8x. 

apt-get install sidux-scripts

En tant que $Utilisateur :

$ cd /répertoire/contenant/votre/ISO

$ burniso

Toutes les images ISO dans le dossier courant apparaîtront et la gravure commencera après selection d'une d'entre elles. Soyez donc certain que votre media est inséré et prêt.

Quels périphériques utiliser, en tant que $utilisateur :

Pour vérifier les options du lecteur CD/DVD ainsi que les capacités des périphériques ATAPI :

wodim :

$ wodim --devices
wodim: Vue d'ensemble des périphériques accessibles (2 trouvés) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Autres alternatives :

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

et

$ wodim -prcap

Voici quelques exemples utiles avec 'scd0' :

Informations sur un CD/DVD vierge :
$ wodim dev=/dev/scd0 -atip

ou

$ cdrdao disk-info --device ATA:1,0,0
Effacer un disque réinscriptible :
$ wodim -blank=fast -v dev=/dev/scd0

ou

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Cloner un CD :
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Cloner un CD à la volée :
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Graver un CD audio à partir de fichiers .wav en 12x :
$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav
Graver un CD à partir de fichiers bin/cue :
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Graver une image ISO :
$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v quelquechose.iso

Si vous obtenez une erreur driveropts, c'est parce que l'option burnfree est désuette pour certains périphériques. Utilisez alors : 

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

ou

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso
Créer une image ISO à partir des sous-dossiers et fichiers contenus dans un répertoire :

Elle pourra ensuite être gravée au moyen de la commande précédente (graver une image ISO) :

$ genisoimage -o MonImage.iso -r -J -l répertoire

Si vous disposez d'un graveur de DVD, vous pouvez utiliser growisofs pour graver un DVD, comme graver une image ISO vers un DVD :

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Graver des fichiers multiples sur un DVD :
$ growisofs -Z /dev/dvd -R -J données1 données2 données3 ...
Si de l'espace est encore disponible sur le DVD, vous pouvez y ajouter de nouveaux fichiers :
$ growisofs -M /dev/dvd -R -J unautrefichier encoreunautrefichier...
Pour clore la session, utilisez :
$ growisofs -M /dev/dvd=/dev/zero
