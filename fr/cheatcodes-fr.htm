Codes de démarrage

Si des paramètres sont listés dans la colonne "Paramètres", alors ils doivent être adressés au code de démarrage avec un signe = . Par exemple, si 1280x1024 est la valeur de paramètre que vous souhaitez adresser pour l'affichage (screen), alors entrez screen=1280x1024 dans la ligne de commande de Grub, ou pour utiliser la langue française lang=fr.

Référence vers une list exhaustive de codes de démarrage pour le noyau

Codes de démarrage spécifiques à aptosid (seulement pour le Live.iso)

Code de démarrage Paramètre Description
blacklist modulename Désactiver temporairement les modules
desktop kde Permet de choisir un bureau graphique
fluxbox
fromiso Pour cette option reportez-vous à la section "fromiso"
hostname monnomdemachine Change le nom d'hôte du système sur le Live-CD
lang be, bg, cz, da, de, de_CH, el, en, en_AU, en_GB, en_IE, es, fr, fr_BE, ga, hr, hu, it, ja, nl, nl_BE, pl, pt (pt_BR), pt_PT, ro, ru, zh Paramètre la langue. Paramètrera également votre locale, le mapping de clavier (en console et pour X), fuseau horaire et un miroir debian.
Quand la forme étendue lang=ll_cc ou lang=ll-cc est entrée , ll sera utilisé pour la sélection de la langue et cc pour le mapping de clavier, le miroir et le fuseau horaire (exemple : lang=fr-be). Les valeurs par défaut pour l'anglais sont en_US avec UTC comme fuseau horaire et pour l'allemand, de avec Europe/Berlin comme fuseau horaire. Exemple: lang=pt_PT tz=Pacific/Auckland
md5sum Commence par une vérification de l'intégrité du support
noaptlang Désactiver l'installation des paquets de localisation basé sur la langue sélectionnée
nodhcp Désactive le recours direct au DHCP (Dynamic Host Configuration Protocol) pour vous connecter automatiquement en ethernet.
noeject Coupe l'invite d'éjection du cd à la fermeture
nofstab Ne pas écrire une nouvelle fstab
nointro Empêche l'affichage du fichier index.html à l'entrée du bureau du Live-CD
nocpufreq Désactive Speedstep/Powernow
nonetwork Désactive la configuration automatique des périphériques réseaux au démarrage
noswap Désactive la partition swap
persist Pour cette option reportez-vous à la section "fromiso et persist"
smouse Cherche une souris série avec hwinfo
toram Copie le CD dans la RAM puis démarre depuis cette copie (une quantité suffisante de RAM est bien entendu requise)
tz tz=Europe/Dublin Configure votre fuseau horaire. Si votre bios/horloge machine est fixée sur UTD, utilisez utc=yes. A list of all timezones supported can be found by copying and pasting into your browser: file:///usr/share/zoneinfo/ .
Codes de démarrage se rapportant à X

Les codes de démarrage respectifs pour xrandr et xmodule doivent être également invoqués si vous utilisez d'autres codes de démarrage se rapportant à X avec des cartes graphiques radeon, intel ou mga.

Code de démarrage Paramètre Description
dpi auto ou XX Fixe la valeur en points par pouces (dpi) pour votre écran. Vous pouvez configurer la résolution en dpi de votre écran en divisant sa longueur en pixels par sa diagonale en pouces puis multipliant la valeur obtenue par 1,25 pour un écran 4:3; 1:18 pour unéran 16:10 ou 1.147 pour unécran 16:9. Pour un 24" 1920x1080 cela ferait 1.147*1920/24 donnant en dpi=92 ou pour un 15" 1600x1200 ça donnerait 1.25*1600/15 en dpi=133.
hsync 80 (par exemple) Valeur de rafraîchissement horizontale pour votre moniteur (kHz)
noml Empêche la configuration de Xorg de stocker une liste de Modelines, et implique donc par la suite une détection automatique du mode correct
noxrandr Désactiver l'utilisation des extensions RandR 1.2 utilisé par les nouveaux pilotes de X.org et utiliser des anciens tests techniques de moniteurs
screen 800x600 Fixe la résolution de votre écran : 1024x768, 1600x1200, etc.
vsync 60 (exemple) Assigne une valeur de rafraîchissement vertical au moniteur (Hz)
xdepth valeurs: 8 15 16 24 Profondeur de couleurs par défaut à utiliser par X.org (tous les pilotes ne supportent pas 1 et 4)
keytable (ex: de, gb, fr, us) Disposition de clavier à utiliser par X.org
xkbmodel (ex) pc105 Modèle de clavier à utiliser par X.org
xkboptions (e.g.) grp:alt_shift_toggle Variante de clavier à utiliser par X.org
xkbvariant (e.g.) nodeadkeys, pour définir une variante du clavier
xmode (ex: 800x600) Fixe la résolution de votre écran : 1024x768, 1600x1200, etc.
xmodule or xdriver ati, fbdev, i810, intel, mga, nv, radeon, savage, vesa Utilise le module-X correspondant
xrandr Force X.org à se configurer en utilisant les extensions RandR 1.2 des nouveaux pilotes X.org
xrate XX Force le rafraîchissement préféré des pilotes X.org RandR 1.2 supportés ; doit s'utiliser en conjonction avec le Code dédémarrage xmode.Une documentation compréhensive en anglais est disponible ici
xhrefresh 80 (exemple) Assigne une valeur de rafraîchissement horizontal pour X (kHz)
xvrefresh 60 (exemple) Assigne une valeur de rafraîchissement vertical pour X (kHz)
