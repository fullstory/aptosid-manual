Codes de démarrage

Si des paramètres sont listés dans la colonne "Paramètres", alors ils doivent être adressés au code de démarrage avec un signe = . Par exemple, si 1280x1024 est la valeur de paramètre que vous souhaitez adresser pour l'affichage (screen), alors entrez screen=1280x1024 dans la ligne de commande de Grub, ou pour utiliser la langue française lang=fr.

Référence vers une list exhaustive de codes de démarrage pour le noyau

Codes de démarrage spécifiques à sidux (seulement pour le Live.iso)

Code de démarrage Paramètre Description
blacklist modulename Désactiver temporairement les modules
desktop kde Permet de choisir un bureau graphique
fluxbox
fromiso Pour cette option reportez-vous à la section "fromiso"

When the extended form lang=ll_cc or lang=ll-cc is given, ll will be used for language selection and cc for keyboard, mirror and timezone selection (e.g. lang=fr-be). The defaults for English are en_US with America/NewYork as the timezone and for German, de with Europe/Berlin as the timezone. Example: lang=pt_PT tz=Pacific/Auckland

X related cheatcodes

Either the xrandr or xmodule cheatcode should also be used if using other X related cheatcodes with a radeon, intel or mga graphics card.

Code de démarrage Paramètre Description
dpi auto ou XX Set desired Dots Per Inch for your display. You can work out your screen's dpi from the width in pixels divided by the diagonal size in inches and multiplying this by 1.25 for a 4:3 screen, 1.18 for a 16:10 screen or 1.147 for a 16:9 screen. For a 24" 1920x1080 screen it would be 1.147*1920/24 giving dpi=92 or a 15" 1600x1200 screen would be 1.25*1600/15 giving dpi=133.
hsync 80 (par exemple) Valeur de rafraîchissement horizontale pour votre moniteur (kHz)
noml Empêche la configuration de Xorg de stocker une liste de Modelines, et implique donc par la suite une détection automatique du mode correct
noxrandr Désactiver l'utilisation des extensions RandR 1.2 utilisé par les nouveaux pilotes de X.org et utiliser des anciens tests techniques de moniteurs
screen 800x600 Fixe la résolution de votre écran : 1024x768, 1600x1200, etc.
vsync 60 (exemple) Assigne une valeur de rafraîchissement vertical au moniteur (Hz)
xdepth valeurs: 8 15 16 24 Profondeur de couleurs par défaut à utiliser par X.org (tous les pilotes ne supportent pas 1 et 4)
keytable (ex: de, gb, fr, us) Disposition de clavier à utiliser par X.org
xkbmodel (ex) pc105 Modèle de clavier à utiliser par X.org
xkboptions (e.g.) grp:alt_shift_toggle Keyboard variant to be used by X.org
xkbvariant (e.g.) nodeadkeys, to set a keyboard variant
xmode (ex: 800x600) Fixe la résolution de votre écran : 1024x768, 1600x1200, etc.
xmodule or xdriver ati, fbdev, i810, intel, mga, nv, radeon, savage, vesa Utilise le module-X correspondant
xrandr Force X.org à se configurer en utilisant les extensions RandR 1.2 des nouveaux pilotes X.org
xrate XX Force le rafraîchissement préféré des pilotes X.org RandR 1.2 supportés ; doit s'utiliser en conjonction avec le Code dédémarrage xmode.Une documentation compréhensive en anglais est disponible ici
xhrefresh 80 (exemple) Assigne une valeur de rafraîchissement horizontal pour X (kHz)
xvrefresh 60 (exemple) Assigne une valeur de rafraîchissement vertical pour X (kHz)
