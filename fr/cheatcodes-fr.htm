Codes de démarrage

Si des paramètres sont listés dans la colonne "Paramètres", alors ils doivent être adressés au code de démarrage avec un signe = . Par exemple, si 1280x1024 est la valeur de paramètre que vous souhaitez adresser pour l'affichage (screen), alors entrez screen=1280x1024 dans ligne de commande de Grub, or for language, lang=pt, ou pour acpi, acpi=off

sidux specific cheatcodes (Live-cd only)

Code de démarrage Paramètre Description
desktop kde permet de choisir un bureau graphique
fluxbox
dpi auto ou XX règle la résolution en pixel par pouce de votre écran (par exemple mon 15 pouces de portable affiche en 1600x1200, alors 1600/(15*(4/5)) = 133 , et donc dpi=133)
fromiso Pour cette option reportez-vous à la section "fromiso"
hostname myhostname hostname, change le nom d'hôte du système sur live-cd
hsync 80 (par exemple) valeur de rafraîchissement horizontale pour votre moniteur (kHz)
lang au, be, bg, br, ch, cn, cz, da, de, es, fi, fr, gr, gb, il, hr, ie, ga, it, jp, nl, pl, pt, ru, si, sk, tr, tw, us Fixe les paramètres liguistiques. La valeur influencera également la locale par défaut, le schéma de clavier (console et X), le fuseau horaire et le choix de miroirs debian.
Pour les versions de sidux sur lesquelles les paquets i18n appropriés seraient installés, ces préférences s'appliqueraient.
La forme "lang=ll-cc" est utilisable, et qui permet d'assigner à ll et cc tout type de valeur en deux lettres admise. "ll" s'utilise pour sélectionner la langue et "cc" pour le clavier et le fuseau horaire (ex: "lang=fr-be"). Ceci permet de découpler langue et situation géographique.
md5sum Commence par une vérification de l'intégrité du support
nodhcp désactive le recours direct au DHCP (Dynamic Host Configuration Protocol) pour vous connecter en ethernet automatiquement.
noeject coupe l'invite d'éjection du cd à la fermeture
nointro empêche l'affichage du fichier index.html à l'entrée du bureau du live-cd
noml Prevents X.org configuration from containing a list of Modelines and causes audodetection of the correct Mode
nocpufreq désactive Speedstep/Powernow
noxorgconfig Disable automatic creation of X.org configuration file (/etc/X11/xorg.conf
noxrandr Disable use of RandR 1.2 extensions by new X.org drivers and use legacy monitor probing techniques
persist Please refer to "fromiso" and persist
screen 800x600 fixe la définition pour votre écran : 1024x768, 1600x1200, etc.
smouse Probe for serial mice with hwinfo
toram copie le CD dans RAM puis démarre depuis cette copie (suffisament de RAM est requis, bien entendu)
vsync 60 (exemple) assigne une valeur de rafraîchissement vertical au moniteur (Hz)
xdepth values: 1 4 8 15 16 24 Default color depth to be used by X.org (not all drivers support 1 and 4
xkblayout (eg, us, de, gb) Keyboard Layout to be used by X.org
xkbmodel (eg, 104, 105) Keyboard model to be used by X.org
xkbvariant eg, nodeadkeys Keyboard variant to be used by X.org
xmode (eg 800x600) Sets your screen resolution, 1024x768, 1600x1200 etc
xrandr Force X.org to be configured to use RandR 1.2 extensions of new X.org drivers
xrate XX Force preferred refresh rate of RandR 1.2 supported X.org drivers, must be used in conjunction with xmode cheatcode Comprehensive documentation is here
xhrefresh 80 (exemple) assigne une valeur de rafraîchissement horizontal pour X (kHz)
xvrefresh 60 (exemple) assigne une valeur de rafraîchissement vertical pour X (kHz)

Common Linux kernel parameter cheatcodes

Code de démarrage Paramètre Description
acpi off désactive complètement l'ACPI. Advanced Configuration and Power Interface (ACPI) est une norme industrielle co-développée par Intel (TM), Microsoft (TM) et Toshiba (TM), qui définit un contrôle d'alimentation direct par le système d'exploitation et permet à celui-ci des options avancées de configuration.
oldboot les parties d'ACPI nécessaires au démarrage sont gardées actives, et les autres sont désactivées.
force force l'activation de l'ACPI
apm off passe le contrôle d'alimentation en mode manuel
irqpoll utilise IRQ poll
mem 128MB déclare la quantité de ram à utiliser
noacpi désactive le contrôleur de programmation avancée des interruptions acpi (ACPI).
noagp désactive l'agp
noapic désactive apic (Advanced Programmable Interrupt Controller)
nodma désactive l'usage de la DMA (Direct Memory Access) pour vos disques
noisapnpbios désactive l'ISA Plug'n Play
nomce Disable Machine Check Exception
nosmp désactive le support multi-processeurs (Symetric Multi Processor - smp)
pci noacpi empêche l'utilisation d'ACPI pour router les interruption PCI
quiet empêche le listing exhaustif des opérations de démarrage dans la console
vga normal Option détaillée ici: Codes de démarrage VGA
