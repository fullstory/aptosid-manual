Où trouver de l'aide ?

Recevoir une aide à temps peut faire la différence entre finir dans la purée et régler ce qui importe pour vous. Ce chapitre est fractionné pour vous donner une idée d'où et comment trouver le support qu'une distribution comme sidux peut vous offrir.:

  • Forums et Wiki
  • IRC
  • Tools to know in init 3 and text mode (tty)
  • Using IRC in text mode and/or init 3
  • Web browsing in text mode and/or init 3
  • et infobash

Les forums sidux

Les forums sidux offrent un endroit où poser vos questions et y recevoir des réponses. S'il vous plait, avant d'y poser vos questions, cherchez si d'autres personnes ne les ont pas déjà posées, auquel cas vos réponses s'y trouvent déjà.

Les forums sidux sont en anglais et en allemand.

Le Wiki de sidux

Le Wiki sidux vous est ouvert à tous, utilisateurs de sidux. Nous espérons que la documentation sidux s'enrichira avec le temps pour devenir un outil de référence tout au long du développement du projet.

Nous espérons des contributions des utilisateurs de Linux de tous niveaux de compétences, étant donné que ce wiki a pour but d'aider tous les utilisateurs, quelque soit leur expérience. Les quelques minutes que vous offrirez au wiki pourront aider d'autres personnes (et même vous-mêmes) en attente et recherche de solutions, dans des moments difficiles.

Le Wiki utilise les langues allemande et anglaise.

L'aide interactive directe par IRC (CHAT)

Conventions de comportement sur IRC

Ne vous connectez jamais sur IRC en tant que 'root" ...vous y serez bienvenus comme simples utilisateurs mais jamais comme 'root' : si vous êtes en difficulté, parlez-en tout de suite.

Joindre le canal #sidux

Il existe 2 méthodes vous permettant d'obtenir une aide en temps réel

  1. Cliquer sur l'icône 'sidux-irc' sur le bureau de KDE ou utiliser un autre client IRC
  2. Cliquer sur Chat au menu sidux.com

Konversation

La façon la plus aisée est de cliquer sur l'icône 'sidux-irc' sur le bureau de KDE ou utiliser Kmenu parce que konverstion y est préconfiguré.

Si vous préferez un autre client IRC, choisisez votre serveur:

IRC.oftc.net
port 6667

Activation du chat à partir du site de sidux

Visitez sidux.com et cliquez sur Chat (cgi) ou Chat (java) dans le menu du site. Ça vous mène vers le site web IRC chat, qui vous permet de vous connecter au canal #sidux.

Entrez votre pseudo choisi dans l'espace de pseudo et écrivez #sidux dans l'espace de canal, et cliquez sur 'login' et suivez les instructions.

!paste

!paste sera votre ami sur le canal, si vous ne voulez pas vous faire expulser pour surcharge de contenus adressés. Aussi si vous êtes invité à utiliser !paste, pour fournir le contenu de votre console, tapez !paste dans votre ligne de chat et il vous sera répondu ceci :

"paste" is 
http://paste.debian.net/ :please choose no
syntax highlighting;make your paste;
then give #sidux the link to your paste..

Tout ce que vous aurez à faire consistera à cliquer sur le lien, qui vous conduira au site en question, où vous collerez ce qui vous aura été demandé, puis remonter en haut de page et cliquer sur send.

Vous constaterez ainsi que votre entrée a été ajoutée à la base de données avec diverses options pour copier et coller l'adresse du lien sur le canal IRC. (Exemple: Pour coller le lien vers votre entrée faites: http://paste.debian.net/524 ). Le 'paste' (coller) de Debian vous permet de choisir la durée de sauvegarde de votre information (jusqu'à 72 heures). Vous avez aussi la possibilité d'effacer ces données.

sidux-paste

La commande 'sidux-paste' vous permet de coller le fichier ou le résultat de commande du terminal et tty, c'est donc idéal lorsque vous avez une difficulté dans le mode console (avec 'init 3'). sidux-paste utilise le lien http://rafb.net, et la donnée sera disponible pendant 24 heures.

Vous pouvez utiliser sidux-paste audssi bien comme utilisateur que comme administrateur.

sidux-paste <fichier>
ou
sidux-paste -c "commande1 --paramètres optionnels ; commande2"
Exemple de sidux-paste <file>
sidux-paste /boot/grub/menu.lst
Your paste can be seen here: http://rafb.net/p/n8miQp85.html

Le lien http://rafb.net/p/n8miQp85.html est ce que vous devez fournir à #sidux sur IRC

Exemple de sidux-paste -c
sidux-paste -c "fdisk -l; cat /etc/fstab; blkid"
Your paste can be seen here:http://rafb.net/p/ZrhlVc59.html

Le lien http://rafb.net/p/ZrhlVc59.html est ce que vous devez fournir à #sidux sur IRC

Utilitaires qu'il vaut mieux connaître en mode texte (tty) and init 3

En règle générale vous vous trouvez en mode texte/init 3 lorsque que vous sous apprêtez à mettre-à-niveau votre distribution par un dist-upgrade (ou bien quelque chose de très problématique est survenu à votre système).

gpm

Un bon outil à avoir en mode texte est gpm. Il va vous permettre d'utiliser votre souris pour copier et coller entre différents terminaux/consoles.

Pour installer gpm :

gpm -t imps2 -m /dev/input/mice

assurez-vous que /etc/gpm.conf existe

/etc/init.d/gpm restart

Maintenant vouz pouvez avoir votre souris en mode texte (tty)

File browsing and text editing

Midnight Commander is a text-mode (tty) full-screen file manager and text editor installed by default and is not complex to use.

Apart from the normal keyboard commands it also responds to mouse clicks for navigation because of gpm.

mc brings up the file system while mcedit brings up a blank file or you can go directly to a file.

Example to go directly to a file:

mcedit /etc/apt/sources.list.d/debian.list

You can now edit the file to change parameters and save the alteration to have immediate effect.

Read:

man mc

Accéder au canal irc #sidux en mode texte et/ou init 3

Ne vous connectez jamais sur IRC en tant que 'root" ...vous y serez bienvenus comme simples utilisateurs mais jamais comme 'root' : si vous êtes en difficulté, parlez-en tout de suite.

irc en mode texte et/ou init 3

Par défaut sidux utilise irssi.

En mode init 3 vous pouvez basculer d'un terminal à un autre en utilisant:

# CTRL-ALT-F2
$ siduxbox login:  <nomdutilisateur> <motdepasse> (ne pas utiliser root)
then type
$ sidux-irc (this brings up irssi)

ou si vous avez installé un autre client irc comme weechat:

#apt-get install weechat-curses

alors connectez-vous à irc.oftc.net sur le port 6667. et une fois connecté, changez votre pseudo :

/nick choix_de_pseudo

pour rejoindre le canal sidux:

/join #sidux

ou pour rejoindre le canal sidux en français:

/join #sidux-fr

Pour vous connecter à un autre serveur:

/server adresse.du.serveur

Dans la barre inférieure, vous voyez des numéros d'activation si des canaux sont actifs ; pour changer de canal faites ALT-1 ALT-2 ALT-3 ALT-4, etc.

Pour quitter

/exit

Si vous êtes entrain de faire un dist-upgrade, et par exemple pour suivre sa progression :

pour revenir au dist-upgrade
$ CTRL-ALT-F1
pour retourner sur weechat
# CTRL-ALT-F2

irssi documentation

La documentation WeeChat (EN).

Navigation web en mode texte et/ou init 3

Les application comme wm3 et/ou elinks vous permettent la navigation web dans un terminal ou en mode texte, aussi dans le mode init 3

Véfifier que w3m ou elinks est installé, sinon:

apt-get update
apt-get install w3m elinks

Par example, pour utiliser w3m ouvrez un terminal/konsole:

$w3m URL
ou
w3m ?
ou
w3m sidux.com

Si vous avez un écran noir avec les instructions tombées en bas, ne paniquez pas, utilisez la combination de touches

SHIFT+U

Vous allez alors voir qu'il va essayer d'ouvrir le fichier comme Goto URL: file:///home/username/$ , tapez la touche 'retour' plusieux fois jusqu'à ce qu'il n'y ait plus que GOTO URL, et tapez par exemple:

http://sidux.com

Vous pouvez déjà naviguer sur sidux.com ou autres sites de votres choix.

Ou bien dans init 3:

# CTRL-ALT-F2
$ siduxbox login:  <nomdutilisateur> <motdepasse> (pas comme root)
ensuite tapez
$ w3m sidux.com
pour retourner au d-u par example,
$ CTRL-ALT-F1

Pour aller directement au manuel:

$w3m manual.sidux.com
  ou
$w3m manual.sidux.com/fr/welcome-fr.htm

La documentation de w3m (EN).

It is advisable to familiarise yourself with using elinks/w3m, irssi/weechat and midnight commander before an emergency happens, if one should occur. This would be a good page to print out as a reference on how to get online help in such emergencies.

Infobash

Infobash est un système de scripts d'information, indépendant de tout Client IRC particulier. Ce script système peut afficher toutes sortes de choses à propos de votre matériel et de vos logiciels pour les personnes connectées sur le canal, qui peuvent les aider à vous porter assistance...ou juste pour que tombiez en pamoison devant vos caractériques machine et la version de votre noyau, quand vous le voulez, depuis une console.

To use the infobash script in Konversation type into the chat box:

/cmd infobash -v3

To use the infobash script in other clients type into the chat box:

/exec -o infobash -v3
ou
/infobash -v3

Dans une console, entrez :

infobash -v3

Prenez intégralement connaissance du didactitiel d'utilisation d'infobash (en)

man infobash
