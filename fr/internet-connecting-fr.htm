Se connecter / Comment configurer le réseau - Ceni

Si votre réseau local est équipé d'un serveur DHCP et que votre ordinateur s'y connecte au démarrage du Live-CD, vos paramètres réseau devraient être configurés automatiquement - dans le cas contraire, vous devrez lancer le script Ceni, en cliquant sur menu Kmenu>sidux>Internet>Ceni. Un terminal/konsole sera ouvert, en demandant le mot de passe pour root, (sur le live-CD il n'y a pas de mot de passe configuré)

La façon vite pour accéder Ceni est ouvrir un terminal/konsole et entrer 

ceni

qu'il vous demandra le mot de passe pour root.

Un point fort de Ceni parmi les autres, c'est son habilité prodigieuse pour configurer les cartes sans fil:

Vous pouvez faire la même chose dans le centre de controle en graphique siduxcc de KDE-Start-Menu

Connexion avec un modem RTC 56k (Dial-up Modem)

KDE est équipé d'une interface pour utiliser les modems téléphoniques qui s'appelle KPPP Connexion internet par modem , dans le sous-menu du menu K, intitulé 'Internet".

L'application est équipé d'une aide interne très bien conçue, qui fournit toutes les informations utiles pour vous connecter à internet avec un modem RTC.

Pare-feu / Firewall

Les pare-feux sont généralement inutiles si vous vous connectez au travers d'un routeur/passerelle correctement configuré (recommandé), mais ils sont d'une importance capitale en terme de sécurité si par contre, vous vous connectez directement avec un modem RTC ou ADSL USB :

apt-get install guarddog
OU
apt-get install firestarter

Guarddog est un outil pare-feu convivial de KDE. Pour l'utiliser, ne manquez pas de lire ses tutoriaux intégrés en commençant par le premier d'entre eux.

Firestarter - Un pare-feu Open Source graphique pour GNU/Linux. [Lorsque vous quittez son interface, le pare-feu reste actif en tâche de fond.]
