Support PHP pour Apache2

Assurez-vous que les lignes qui suivent soient bien présentes et décommentées, dans votre fichier apache2.conf (ou dans votre fichier conf.d/php.ini) : 

LoadModule php5_module modules/libphp5.so

# Cause the PHP interpreter to handle files with a .php extension.
AddType application/x-httpd-php .php

Si vous souhaitez pouvoir utiliser divers types de fichiers d'index, il vous faut vérifier la présence de la ligne déclarative suivante dans le fichier /etc/apache2/apache2.conf

DirectoryIndex index.html index.cgi index.pl index.php index.xhtml index.shtml

Redémarrage du serveur apache :

#/etc/init.d/apache2 restart
