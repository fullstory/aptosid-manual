Recherche en ligne

To search on line, type or paste into the search box of your browser, (where xxxx is the word you wish to search for), the following:

xxxxx site:manual.sidux.com/fr

Recherche en local

The sidux Operating System Manual has created some special configs in conjunction with the Developer for Recoll as a part of the default package. Recoll, un outil de recherche personnel pour systèmes UNIX/Linux. Cet outil exploite le puissant moteur de recherche Xapian, auquel il ajoute une interface graphique utilisant Qt. A very special thank you to Recoll.

Recoll est intitulé Local Text Search au menu Utilitaires de KDE ou au menu Debian -> Applications -> Gestion de données -> Data Search Tool (Outil de recherche de données).

Étape 1

Le manuel est actuellement disponible via apt pour langues suivantes :
Danois : da , Allemand : de, Grec : el, Anglais : en, Espagnol : es, Français : fr, Hongrois : hr, italien : it, Japonais : ja, Néerlandais : nl, portugais du Brésil : pl, Polish :, pt-br, Roumain : ro, Russe : ru.

apt-get update
apt-get install sidux-manual-xx (où -xx correspond au code de la langue désirée)
Étape 2
apt-get install recoll
Étape 3
  • Recoll on the first start will ask you if you want to build the database, click cancel, where upon an indexing configuration box will appear.
  • In the Global parameters - Top directories box, click on the + (plus) sign and go to /usr/share/sidux-manual and click on your language and click ok. Next click on the - (minus) sign and delete any other folders including ~ (tilde).
  • Finally File>Update Index and search the topic you need in the sidux-manual.

Si vous avez besoin de reconfigurer Recoll suite à la mise-à-jour du manuel sidux à l'aide d'apt, (dans une console utilisateur $):

recollindex -z

Puis: Kmenu >Debian >Applications > Gestion de données > Personal Search Tool et dans Recoll Fichier> Indexer.

The 'Preview' link will highlight all instances of your search word in the file, whereas 'Open' will open Konqueror, et utilisez la commande Chercher dans cette page pour une micro-recherche du mot cherché avec recoll.
