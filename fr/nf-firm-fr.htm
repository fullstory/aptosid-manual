Sources list and firmware drivers: non-free

Listes de sources

Les live-CD aptosid embarquent exclusivement des logiciels libres respectant le contrat social debian. De ce fait, vous pourrez souhaiter ajoûter les sources contrib non-free à vos fichiers /etc/apt/sources.list.d/debian.list:

mcedit /etc/apt/sources.list.d/debian.list

Les lignes pour le faire devraient ressembler à ceci, variant en fonction des miroirs que vous préférerez utiliser, (refer to Liste de serveurs et miroirs Debian:

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Si vous modifiez vos liste de sources vous devez sitôt après lancer apt-get update afin de pouvoir chercher et installer les paquets des nouvelles sources ( apt-get update est toujours une bonne habitude à prendre dans sid avant d'installer tout paquetage).

Firmware detection - non-free

To install non-free firmware, (assumes contrib non-free has been added to sources):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

Le besoins de vos matériel en pilotes non-libres peuvent être identifiés soit en utilisant la liste qui suit, soit en recourant au nouveau script fw-detect (inclus dans aptosid-scripts).

$ fw-detect

Le retour de fw-detect vous décrit les commandes nécessaires à l'installation et à l'activation de pilotes non-libres dont vous aurez besoin:

Exemple:

#Detected driver that requires firmware to operate
#Follow these instructions to obtain the correct firmware
# and activate the zd1211rw driver:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Pour installer le pilotes depuis le dépot git, faites:

apt-get install git-core

pilotes non-libres en paquets .deb stockés sur une clé

Vous pouvez récupérer un ensemble de pilotes sous forme d'archives .deb, que vous pouvez ensuite stocker sur une clé pour installer sur ordinateur tiers. Cet ensemble se récupère soit sous forme d'archive .zip soit sous forme d'archive .tar.gz à l'url http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/. Décompactez l'archive dans un répertoire "pilotes", ou vous pourrez distinguer différents fichiers .deb.

Téléchargez ensuite firmware-linux-free*.deb et ajoutez cette archive à votre répertoire "pilotes". Copiez ce répertoire sur une clé usb et lorsque vous connecterez la clé à un ordinateur ayant besoin de pilotes non-libres, vous vous déplacerez dans la clé puis ferez:

dpkg -i firmware-linux-nonfree*.deb

Nous nous efforçons de fournir des paquetages pour tout firmware librement redistribuable depuis nos dépots d'outils non-libres, mais tous les fournisseurs ne l'autorisent pas.
