Le Centre de Contrôle de sidux (siduxcc)

Le Centre de Controle de Sidux (siduxcc) est une interface d'accès pour l'administration de sidux. Elle se présente sous la forme de module du Centre de Contrôle. Elle existe dans les langues Croatienne, Danoise, Neerlandaise, Anglaise, Française, Dllemande, Italienne, Japonaise, Polish, Portuguese, Roumaine, Russe et Espagnole. It is available in KDE and GTK versions.

siduxcc contient 7 modules avec les fonctions suivantes:

1. Module pour l'affichage
  • Changer Résolution et Fréquence de rafraichissement
  • Régler la profondeur de couleur et PPP
  • Configurer le serveur X
  • Installer le pilote de la carte graphique
Kmenu > System > sidux Control Centre > Affichage
2. Module pour l'Internet
  • Paramétrage du navigateur par défaut de sidux
Kmenu > System > sidux Control Centre > Internet
3. Module pour le noyau
  • Installer le nouveau noyau
  • Enlever les anciens noyaux
  • Installer les modules de noyau
Kmenu > System > sidux Control Centre > Noyau
4. Module pour le réseau
  • paramétrage des cartes réseau
  • Changer le nom d'hôte
  • Choisir le serveur de nom de domaine
  • Installer la carte Wifi (Ndiswrapper)
  • Installer la carte Wifi (fwdetect)
Kmenu > System > sidux Control Centre > Réseau
5. Module pour les services
  • Gestion courante des (Démarrer, Arrêter, Redémarrer, Recharger, Ajoûter/Supprimer les entrées par niveau) (Apache, Samba, SSH...):
  • Démarrer des services au boot
Kmenu > System > sidux Control Centre > Services
6. Module pour les logiciels
  • Affichage des logiciels pouvant être mis-à-jour
  • Lancement du Méta-Installeur/siduxcc-hermes
Kmenu > System > sidux Control Centre > Logiciels
7. Module pour l'information
  • Affichage des informations se rapportant à votre système
Kmenu > System > sidux Control Centre > System info

La version en mode console

Lancez: siduxcc

sidux-hermes

  • sidux-hermes (dist-upgrade sentinel) est une application du tableau de bord KDE qui vous avertit en cas d'alertes rendant un dist-upgrade problématique. Elle scanne la section de news du site sidux.com avec pour mot clé 'Warning' et montre un signal rouge dans le cas où elle en trouve.
dans un konsole comme utilisateur
siduxcc-hermes
ou
KMenu > sidux > Hermes (dist-upgrade sentinel)
