Le Centre de Contrôle de sidux (siduxcc)

Le Centre de Controle de sidux (siduxcc) est une interface d'accès pour l'administration de sidux. Elle se présente sous la forme de module du Centre de Contrôle. Elle existe dans les langues Croate, Danoise, Neerlandaise, Anglaise, Française, Allemande, Italienne, Japonaise, Polonaise, Portuguaiese, Roumaine, Russe, Espagnole et Grèque. Il est disponible en version KDE et GTK.

siduxcc contient 7 modules avec les fonctions suivantes:

1. Module pour l'affichage
  • Changer Définition et Fréquence de rafraichissement
  • Régler la profondeur de couleur et la résolution
  • Configurer le serveur X
  • Installer le pilote de la carte graphique
Menu K > Système > sidux Control Centre > Affichage
2. Module pour l'Internet
  • Paramétrage du navigateur par défaut de sidux
Menu K > Système > sidux Control Centre > Internet
3. Module pour le noyau
  • Installer le nouveau noyau
  • Enlever les anciens noyaux
  • Installer les modules de noyau
Menu K > Système > sidux Control Centre > Noyau
4. Module pour le réseau
  • Paramétrage des cartes réseau
  • Changer le nom d'hôte
  • Choisir le serveur de nom de domaine
  • Installer la carte Wifi (Ndiswrapper)
  • Installer la carte Wifi (fwdetect)
Menu K > Système > sidux Control Centre > Réseau
5. Module pour les services
  • Gestion courante des (Démarrer, Arrêter, Redémarrer, Recharger, Ajoûter/Supprimer les entrées par niveau) (Apache, Samba, SSH...):
  • Démarrer des services au boot
Menu K > Système > sidux Control Centre > Services
6. Module pour l'information
  • Affichage des informations se rapportant à votre système
Menu K > Système > sidux Control Centre > System info

sidux hermes

sidux-hermes (sentinelle de dist-upgrade) est une application qui se place dans la boîte à miniatures de KDE et qui vous avertit de la présence éventuelle de problèmes liés à une mise-à-niveau (dist-upgrade). Cette application recherche le mot-cle 'Warning' (avertissement) dans la section des news du site sidux.com et affiche une icône en forme de feu rouge s'il trouve des avertissements. Une version gtk existe aussi.

sidux-hermes-qt4

Il est recommandé d'installer la version qt4 : elle possède le plus de fonctions et fonctionne sur tous les environnements de bureau (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 possède des options comme l'affichage d'informations sur le noyau et la possibilité de télécharger des fichiers prêts à être mis à jour (vous devrez ensuite lancer le dist-upgrade en init 3).

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Utilisez la version gtk2 si vous désirez avoir une application sans les dépendances qt.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contient la sous-couche de toutes les versions hermes. Il sera automatiquement installé lorsque vous installerez les versions qt4 ou gtk2.

apt-get install sidux-hermes
Légende des icônes de Hermes

dist-upgrade - Vous pouvez mettre à jour, vérifiez les News et les Forums sur sidux

dist-upgrade - Avertissement, soyez prudent lors de la mise à jour, vérifiez les News et les Forums sur sidux

dist-upgrade - Ne mettez pas à jour, vérifiez les News et les Forums sur sidux
