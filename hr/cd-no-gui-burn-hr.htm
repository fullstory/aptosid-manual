Prženje CD/DVD bez GUI

OVO JE VRLO VAŽNO: sidux, kao Linux LIVE-CD, je bazirana na tehnologiji visoke kompresije, (i zbog toga, dobro pazite kad pečete ISO-datoteku. Samo koristite CDs visoke kvalitete [ili DVD+RW] i pecite sa DAO-modusom (disk-at-once) i ne brže nego x8.

burniso

Ako hoćete da pržite CD / DVD, nije vam potreban GUI da sredite posao.

Problemi sa prženjem se dogode većinom s frontendom (k3b), ali ne tako puno sa backendima (growisofs, wodim ili cdrdao).

sidux provides a ready made script to burn a sidux ISO called 'burniso'.

burniso burns ISO images on media, using Disk-At-Once mode and a hard coded speed of 8x, using wodim. 

apt-get install sidux-scripts

As $User:

$ cd /dir/containing/your/ISO

$ burniso

All ISO images in the current directory are presented and will start burning right after your selection of an ISO, so be sure you have your media inserted.

What devices to use, as $user:

Za ATAPI uređaje možemo ih naći sa:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Other alternatives are:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

and

$ wodim -prcap

Ovdje su upotrebljivi primjeri:

Informacije o praznoj CD/DVD:
$ wodim dev=/dev/scd0 -atip

ili

$ cdrdao disk-info --device ATA:1,0,0
Izbrisati rewritable disk:
$ wodim -blank=fast -v dev=/dev/scd0

ili

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Klonirati cd:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Klonirati cd u brzo:
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Kreirati audio cd od wav datoteka s 12x brzinom:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav
Prži cd od bin/cue datoteka:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Prži ISO datoteku:
$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

If you get a driveropts error, it is because burnfree is deprecated for some devices, therefore:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

or

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso
Kreirati ISO datoteku s svim datotekama (i podirektorijama) direktorija.

To se može pržiti sa naredbom gore (prži ISO datoteku):

$ genisoimage -o myImage.iso -r -J -l directory

Ako imate DVD pržilicu, možete koristiti growisofs za prženje na DVD,isto kao prženje ISO datoteke na DVD:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Prženje višestrukih datoteka na DVD:
$ growisofs -Z /dev/dvd -R -J datoteka1 datoteka2 datoteka3 ...
Ako imate još mjesta na DVD, možete dodati još više datoteka:
$ growisofs -M /dev/dvd -R -J još jedna datoteka i još jedna...
Da zatvorite session, koristite:
$ growisofs -M /dev/dvd=/dev/zero
