Binary, closed source drivers for: Nvidia / ATI

For proprietary drivers read the documentation from your graphic card manufacturer.

Open Source Xorg drivers for nVidia, ATI, Radeon, Intel

Open Source Xorg drivers for nVidia, ATI, Radeon, Intel are pre-installed with sidux.

Pogonitelji za x-org X-Server (pogledajte u xserver-xorg za dalje informacije) nudi podržaj za 2D za NVIDIA Riva, TNT, GeForce, i Quadro kartice i za ATI Mach, Rage, Radeon, i FireGLcards sve do atimisc', 'r128' i 'radeon' sub-pogonitelje. Isto za Intel free series za 3D

Da ga instalirate:

Odjavite se iz KDE s Ctrl+Alt+F1
prijavite se kao root
onda upišite init 3
(to osigura da X ne radi više)
apt-get update
apt-get install xserver-xorg-video-nv
ILI
apt-get install xserver-xorg-video-ati
ILI
xserver-xorg-video-intel
init 5 && exit

For some cards it may add 3D support, however it is usually 2D.

Kao iduće sad morate editirati /etc/X11/xorg.conf, KAKO GOD OSIGURAJTE JE; PRIJE PROMJENA, jer ako pogrešno editirate /etc/X11/xorg.conf završiti će te u suzama.

Da editirate /etc/X11/xorg.conf desno kliknite na nju, otvorite je sa root privilegijama. Nađite sekciju koja se zove SECTION DEVICE i promijenite grafički pogonitelj u nv OR ati

UPOZORILI SMO VAS!!! EDITIRANJE XORG.CONF DATOTEKE JE NA VAŠ RIZIK!!

Savjet za hardver s non-free potrebama

Zbog kompleksnosti zakona, sidux samo nudi dfsg-free softver.

Sources List

Zato što sidux samo sadrži dfsg slobodnu softver, možda hoćete da dodate contrib / non-free u vašu /etc/apt/sources.list.d/debian.list and /etc/apt/sources.list.d/sidux.list ili /etc/apt/sources.list i osigurajte se da ste spojeni s Internetom:

# See sources.list(5) for more information, especially
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.
# The /etc/apt/sources.list.d/ directory provides a way to add sources.list
# entries in separate files that end with sources.list file.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
Firmware

The required firmware can be determined with the following device/ firmware enumeration or by using the newly developed fw-detect script (packaged in sidux-scripts).

$ fw-detect

It is good practice to do a modprobe -r <modulename> before modprobe <modulename>, especially for intel.

Pisati na NTFS-particije s ntfs-3g

Budite upozoreni: ntfs-3g je mlad beta pogonitelj, zato ga nikada nemojte koristiti bez sigurnosne kopije, i naravno ne na produktivnim sistemima! Ako to uradite, vaša je pogreška, ako izgubite podatke, koristite ga na svoj rizik!

Otvorite konzolu i upišite sljedeće naredbe: Pogledajte na Partitioning your HD - Disk Naming

umount /dev/XdXX
mount -o silent,umask=0,no_def_opts,allow_other -t ntfs-3g /dev/XdXX /media/XdXX
exit
exit

Sad možete mountati vaš NTFS-Volume rw i možete spremiti podatke na njemu. Ali opet, POZOR! EKSPERIMENTALAN JE!
