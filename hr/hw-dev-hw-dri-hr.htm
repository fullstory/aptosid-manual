nVidia 3d pogonitelji

Postoje 3 metode instaliranja 3d nVidia pogonitelja u vaš sistem

  1. sgfxi / smxi

    Use the sidux community contributed multi-function administration tool, smxi/sgfxi, which is short for sidux-maintenance. To read more refer to smxi and sgfxi Syntax Command Sets.

  2. Koristite binarni Nvidia/ATIinstaler
    Instaliraj iz source

Open Source Xorg drivers for nVidia, ATI, Radeon, Intel

Open Source Xorg drivers for nVidia, ATI, Radeon, Intel are pre-installed with sidux. If you have overwitten them with closed source drivers you can revert or roll back to them with sgfxi

Pogonitelji za x-org X-Server (pogledajte u xserver-xorg za dalje informacije) nudi podržaj za 2D za NVIDIA Riva, TNT, GeForce, i Quadro kartice i za ATI Mach, Rage, Radeon, i FireGLcards sve do atimisc', 'r128' i 'radeon' sub-pogonitelje. Isto za Intel free series za 3D

Da ga instalirate:

Odjavite se iz KDE s Ctrl+Alt+F1
prijavite se kao root
onda upišite init 3
(to osigura da X ne radi više)
apt-get update
apt-get install xserver-xorg-video-nv
ILI
apt-get install xserver-xorg-video-ati
ILI
xserver-xorg-video-intel
init 5 && exit

For some cards it may add 3D support, however it is usually 2D.

Kao iduće sad morate editirati /etc/X11/xorg.conf, KAKO GOD OSIGURAJTE JE; PRIJE PROMJENA, jer ako pogrešno editirate /etc/X11/xorg.conf završiti će te u suzama.

Da editirate /etc/X11/xorg.conf desno kliknite na nju, otvorite je sa root privilegijama. Nađite sekciju koja se zove SECTION DEVICE i promijenite grafički pogonitelj u nv OR ati

UPOZORILI SMO VAS!!! EDITIRANJE XORG.CONF DATOTEKE JE NA VAŠ RIZIK!!

Savjet za hardver s non-free potrebama

Zbog kompleksnosti zakona, sidux samo nudi dfsg-free softver.

Sources List

Zato što sidux samo sadrži dfsg slobodnu softver, možda hoćete da dodate contrib / non-free u vašu /etc/apt/sources.list.d/debian.list ili /etc/apt/sources.list i osigurajte se da ste spojeni s Internetom:

# See sources.list(5) for more information, especially
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.
# The /etc/apt/sources.list.d/ directory provides a way to add sources.list
# entries in separate files that end with sources.list file.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
Video kartice
  • ATI Radeon: 3d ubrzavanje za starije kartice do r35x bi trebalo da radi, novije Radeon X1xxx kartice trebaju propietarne pogonitelje za ubrzan rezultat, ali do sad još nisu kompatibilne sa X.org 7.2. Ovisno o vašem chipsetu "xmodule=vesa" ili "xmodule=fbdev" kao cheatcode moglo bi biti u redu za novije Radeon X1xxx modele.
Firmware
  • Da vam pomogne da pronađete koja vam je firmware potrebna, izvodite, kao korisnik:
$ fw-detect

To će vam pokazati list sa preporukama koje naredbe trebate izvoditi. Vratiti će 'null' ako ne trebate raditi ništa, ili firmware je več instalirana.

  • Atheros wlan: 
    Use: ath5k/ ath9k 
as root REMOVE madwifi
apt-get remove --purge madwifi-tools 
otherwise madwifi will blacklist ath5k / ath9k
  • Atmel AT76c50x 11 MBit/s wlan: 
    apt-get install atmel-firmware
  • Broadcom, b43/ b43 legacy wlan: 
    apt-get install b43-fwcutter
  • Broadcom Blutonium Bluetooth chipset (BCM2033): 
    apt-get install bluez-firmware
  • Eagle USB ADSL modem: zgrabite firmware s http://eagle-usb.org/ueagle-atm/non-free/ i spremite je pod /lib/firmware/.
  • DVB firmwares za razne DVB TV kartice (vecina buđetnih kartica to ne treba): povucite potrebnu firmware (provjerite dmesg) od http://www.linuxtv.org/downloads/firmware/ i spremite je pod /lib/firmware/.
  • IVTV firmware:povucite firmware od http://dl.ivtvdriver.org/ivtv/firmware/ i memorirajte u/lib/firmware/.
  • Intel ipw2100 and ipw2200, 11/54 MBit/s wlan: 
    apt-get install firmware-ipw2x00
(obsolete ipw2100-firmware and ipw2200-firmware ought to be purged)
  • Intel ipw3945 (54 Mbit/s wlan)/ ipw 4965 (draft-n) iwlwifi, 54 Mbit/s wlan Izbrišite iwlwifi-ucode, iwlwifi-3945, firmware-iwl3945 i firmware-iwl4965 ako su več instalirane.: 
    apt-get install firmware-iwlwifi
  • Intel ipw4965/ iwlwifi, draft-n wlan: 
    apt-get install firmware-iwlwifi
  • Intersil prism54, 54 MBit/s wlan: povucite potrebnu firmware od http://prism54.org/firmware/ i spremite je pod /lib/firmware/.
  • nVidia graphics: 3d ubrzanje do sad nije moguće s slobodnim pogoniteljima, koristite get-sidux-binary-gfx ili sgfxi da dobijete install skripte za te kartice.
  • RaLink rt61 i rt73 54 MBit/s wlan,: 
    apt-get install firmware-ralink

    Izbrišite rt61-fwcutter, rt73-fwcutter, firmware-rt61 i firmware-rt73, ako su več instalirani

  • Ricoh R5U870 OEM webcams:povucite firmware od http://mediati.org/r5u870/r5u870-0.11.0.tar.gz i kopirajte r5u870_*.fw datoteke u /lib/firmware/.
  • Texas Instruments ACX100 (22 Mbit/s)/ ACX111 (54 MBit/s) wlan: 
    m-a a-i acx100
    povucite potrebnu firmware od http://www.kazer.org/acx-firmware-20060207.tar.bz2 i spremiti pod /lib/firmware/.
  • ZyDAS zd1201 11 MBit/s wlan: povucite firmware od http://surfnet.dl.sourceforge.net/sourceforge/linux-lc100020/zd1201-0.14-fw.tar.gz i memorirajte u /lib/firmware/.
  • ZyDAS zd1211 54 MBit/s wlan: 
    apt-get install zd1211-firmware

Provjeriti će mo dali vam možemo ponuditi pakete za poneke od tih uređaja, ali mora biti legalno.

Pisati na NTFS-particije s ntfs-3g

Budite upozoreni: ntfs-3g je mlad beta pogonitelj, zato ga nikada nemojte koristiti bez sigurnosne kopije, i naravno ne na produktivnim sistemima! Ako to uradite, vaša je pogreška, ako izgubite podatke, koristite ga na svoj rizik!

Otvorite konzolu i upišite sljedeće naredbe: Pogledajte na Partitioning your HD - Disk Naming

umount /dev/XdXX
mount -o silent,umask=0,no_def_opts,allow_other -t ntfs-3g /dev/XdXX /media/XdXX
exit
exit

Sad možete mountati vaš NTFS-Volume rw i možete spremiti podatke na njemu. Ali opet, POZOR! EKSPERIMENTALAN JE!
