nVidia 3d pogonitelji

Postoje 3 metode instaliranja 3d nVidia pogonitelja u vaš sistem

  1. install-binary-gfx

    Novi script za instalaciju grafičkih pogonitelja je zvan: install-binary-gfx. Da dobijete taj script, morate izvoditi jedan put get-sidux-binary-gfx, u konzoli kao root

    get-sidux-binary-gfx
install-binary-gfx -a (da gradite sid source)

    Izabrati će potrebnog pogonitelja za vašu karticu, graditi deb datoteku, instalirati je i automatski će modificirati vašu xorg.conf datoteku. Da možete koristiti novi pogonitelj morate restartovati X (Ctrl+Alt+Backspace).

    Opcije

    Postoje opcije koje bi možda htjeli aktivirati manualno:

     Usage: ./install-binary-gfx [options]
-a | --auto            automatski mode
-b | --build           samo MOD gradnja (nVidia|nVidia-legacy|fglrx)
-d | --detect          samo prepoznavanje kartice
-u | --update          ponoviti script
-v | --verbose         detalji
--aiglx                aktivirati AIGLX

    Primjeri

    install-binary-gfx -a (To će prepoznati vašu karticu i automatski instalirati sid pogonitelje.)
install-binary-gfx -a --aiglx (To će prepoznati vašu karticu i automatski instalirati sid pogonitelje i aktivirati AIGLX.)
install-binary-gfx -b nVidia (To će samo graditi nVidia-module....deb u /usr/src. Isto je za fglrx i nVidia-legacy.)
install-binary-gfx -d (To će prepoznati vašu karticu i pokazati informacije.)
install-binary-gfx --aiglx (To će aktivirati AIGLX.)
install-binary-gfx -u (To će obnoviti script. Zapamtite, to neće biti obavljeno od dist-upgrade dok god ne koristite sidux contrib repozitorij u sources.list.)

    Otkrivanje smetnji

    Ako script ne izvrši kako treba za vašu karticu, molimo vas, startate ga u debug mode, i upišite output isto kao i log datoteku /var/log/sidux-CARD-installer.log na Sidux Forum ili koristite http://paste.debian.net i spojite se sa #sidux i dajte im link da dobijete IRC pomoć

  2. sgfxi / du-fixes-h2.sh

    cd /usr/local/bin ; wget -Nc http://techpatterns.com/sgfxi
chmod +x sgfxi

    sgfxi currently supports ATI fglrx and Nvidia drivers. It also supports converting from or to xorg free drivers like ati, intel, or nv. du-fixes-h2.sh uses sgfxi as its graphics driver installer engine, but simply adds some option support for users.

    See sgfxi -h for complete option list. Options will be updated as time goes on. When you use the -h option you will also see a list of all currently supported drivers, as well as a brief printout of your current system specifications, kernel, xorg version. The default driver is the driver that is installed if no arguments are used to override it. Unless your card is too old, and requires a legacy driver, in which case the script will it determine for you.

    When you run the script, it will stop, and tell you what driver it is going to install. You can accept that, or exit and redo it with an override option if you want something different.

    ILI::Povucite datoteku !h2 d-u-fixes, onda: korištaj samo za instaliranje 3d pogonitelja

     du-fixes-h2.sh -kiwdt

nVidia X.Org X server-Native nv grafički pogonitelj

Taj pogonitelj za X.Org X server (posjetite xserver-xorg za vise informacija) nudi podržaj za NVIDIA Riva, TNT, GeForce, i Quadro kartice.

Da ga instalirate:

Odjavite se iz KDE s Ctrl+Alt+F1
prijavite se kao root
onda upišite init 3
(to osigura da X ne radi vise)
apt-get update
apt-get install xserver-xorg-video-nv
init 5 && exit

Zapamtite da to nije isto kao binarni 'nVidia' pogonitelj, koji nudi 3D podržaj, ali je samo binarni i ne podržavan.

Kao iduće sad morate editirati /etc/X11/xorg.conf, KAKOGOD OSIGURAJTE JE; PRIJE PROMJENA, jer ako pogrešno editirate /etc/X11/xorg.conf završiti ce te u suzama.

Da editirate /etc/X11/xorg.conf desno kliknite na nju, otvorite je sa root privilegijama. Nađite sekciju koja se zove SECTION DEVICE i promijenite grafički pogonitelj u nv

UPOZORILI SMO VAS!!! EDITIRANJE XORG.CONF DATOTEKE JE NA VAS RIZIK!!

Pisati na NTFS-particije s ntfs-3g

Budite upozoreni: ntfs-3g ja mlad beta pogonitelj, zato ga nikada nemojte koristiti bez sigurnosne kopije, i naravno ne na produktivnim sistemima! Ako to uradite, vaša je pogreška, ako izgubite podatke, koristite ga na svoj rizik!

Otvorite konzolu i upišite sljedeće naredbe: Pogledajte na Partitioning your HD - Disk Naming

umount /dev/XdXX
mount -o silent,umask=0,no_def_opts,allow_other -t ntfs-3g /dev/XdXX /media/XdXX
exit
exit

Sad možete mountati vaš NTFS-Volume rw i možete memorirati podatke na njemu. Ali opet, POZOR! EKSPERIMENTALAN JE!

Savjet za hardver s non-free potrebama

Zbog kompleksnosti zakona, sidux samo nudi dfsg-free softver.

Zato sto sidux samo sadrži dfsg free softver, možda hoćete da dodate contrib / non-free u vašu /etc/apt/sources.list i da osigurate Internet spoj:

# See sources.list(5) for more information, especialy
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
  • ATi Radeon graphics: 3d ubrzavanje za starije kartice do r35x bi trebalo da radi, novije Radeon X1xxx kartice trebaju propietarne pogonitelje za ubrzan rezultat, molimo vas da koristite get-sidux-binary-gfx da dobijete install scripts za te kartice.
  • Atheros/ "madwifi" wlan: 
    m-a a-i madwifi
  • Atmel AT76c50x 11 MBit/s wlan: 
    apt-get install atmel-firmware
  • AVM ISDN/ ADSL PCI/ USB kartice: AVM's closed source pogonitelj nije do sad kompatibilan sa kernel 2.6.19.x, molimo, povucite kernel 2.6.18.x (http://sidux.net/files/kernel/) za sad, i koristite: 
    m-a a-i avm
  • Broadcom/ bcm43xx wlan: 
    apt-get install bcm43xx-fwcutter
  • Eagle USB ADSL modem: zgrabite firmware s http://eagle-usb.org/ueagle-atm/non-free/ i memorirajte je pod /lib/firmware/.
  • DVB firmwares za razne DVB TV kartice (vecina budget kartica to ne treba): povucite potrebnu firmware (provjerite dmesg) od http://www.linuxtv.org/downloads/firmware/ i memorirajte je pod /lib/firmware/.
  • hostap baziran 11 MBit/s wlan s loadable firmware (e.g. D-Link DWL-520 rev. E1 i druge): http://www.oakcourt.dyndns.org/~andrew/dwl520e1.html
  • Intel ipw2100, 11 MBit/s wlan: povucite potrebnu firmware od http://ipw2100.sf.net/ i memorirajte je pod /lib/firmware/.
  • Intel ipw2200, 54 MBit/s wlan: povucite potrebnu firmware od http://ipw2200.sf.net/ i memorirajte je pod /lib/firmware/.
  • Intel ipw3945, 54 Mbit/s wlan: 
    apt-get install ipw3945d firmware-ipw3945
  • Intersil prism54, 54 MBit/s wlan: povucite potrebnu firmware od http://prism54.org/firmware/ i memorirajte je pod /lib/firmware/.
  • nVidia graphics: 3d ubrzanje do sad nije moguce s slobodnim pogoniteljima, koristite get-sidux-binary-gfx da dobijete install scripts za te kartice.
  • RaLink rt61 54 MBit/s wlan, povucite potrebnu firmware od http://www.ralinktech.com/ralink/Home/Support/Linux.html
  • exas Instruments ACX100 (22 Mbit/s)/ ACX111 (54 MBit/s) wlan, povucite potrebnu firmware od http://www.kazer.org/acx-firmware-20060207.tar.bz2 and place it under /lib/firmware/.
  • ZyDAS zd1201 11 MBit/s wlan: 
    apt-get install zd1201-firmware
  • ZyDAS zd1211 54 MBit/s wlan: povucite potrebnu firmware od http://sourceforge.net/project/showfiles.php?group_id=129083 i memorirajte je pod /lib/firmware/.

Provjeriti će mo dali vam možemo ponuditi pakete za poneke od tih uređaja, ali mora biti legalno.
