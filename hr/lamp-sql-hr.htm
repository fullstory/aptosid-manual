Mysql postavke

Osigurajte se da su sljedeće linije aktivirane negdje u vašoj apache2.conf datoteci (ili u vašoj conf.d/php.ini):

# extension=php_mysql.so

Tako da postane

extension=php_mysql.so

Konfiguracijska datoteka mysqla je pod: /etc/mysql/my.conf

Standardno mysql kreira korisnika, koji radi bez lozinke. Da promijenite root lozinku:

mysql> USE mysql;
mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE user='root';
mysql> FLUSH PRIVILEGES;
Kreiranje korisnika

Ne morate nikada koristiti root lozinku, tako da morate kreirati korisnika koji se spoji s mysql bazom podataka za PHP skript. Alternativno možete dodati korisnike u mysql bazu podataka s kontrolnim okvirom kao phpMyAdmin da lagano kreirate ili odredite priviligije za bazu podataka za korisnike.

Da uđete u web interface upišite sljedeće u vaš web browser:

http://ipaddress/phpmyadmin/

Kad ste pitani za korisničko ime i lozinku upišite root kao korisničko ime za bazu podataka i upišite lozinku (standardno ne postoji lozinka za root korisnika za bazu podataka)
