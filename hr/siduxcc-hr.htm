sidux Control Centre

The sidux control center is a frontend for managing sidux, and sits as a module in the Control Centre and is available in Croatian, Danish, Dutch, English, French, German, Greek, Italian, Japanese, Polish, Portuguese, Romanian, Russian and Spanish. It is available in KDE and GTK versions.

siduxcc contains 7 modules with the following functions:

1. Display module
  • Change Resolution and Scanning Frequency
  • Set Colordepth and DPI
  • Configure X-Server
  • Install graphic card drivers
Kmenu>sidux>sidux Control Centre>Display
2. Internet module
  • Set sidux default browser
Kmenu>sidux>sidux Control Centre>Internet
3. Kernel module
  • Install new kernel
  • Remove old kernel
  • Install kernel modules
Kmenu>sidux>sidux Control Centre>Kernel
4. Network module
  • Configure networkcards
  • Set Hostname
  • Set Nameservers
  • Install Wlan-Cards (Ndiswrapper)
  • Install Wlan-Cards (fwdetect)
Kmenu>sidux>sidux Control Centre>Network
5. Services module
  • Manage common services (Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Samba, SSH...)
  • Start services at boot
Kmenu>sidux>sidux Control Centre>Services
6. Infomation module
  • Shows you information about your system
Kmenu>sidux>sidux Control Centre>System info

Console version

Start: siduxcc

sidux hermes

sidux-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match. There is also a gtk version.

sidux-hermes-qt4

It is recommended to install the qt4 version. It has the most features and it works on all desktops (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 has options like kernel information and the ability to download files ready to update (dist-upgrade to be carried out in init 3 later). 

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Use the gtk2 version if you want to have an application without qt dependencies.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contains the backend of all hermes versions. It will be automatically installed when you install the qt4 or gtk2 versions.

apt-get install sidux-hermes
