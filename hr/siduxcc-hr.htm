sidux Control Centre

The sidux Control Centre is a frontend for managing sidux. and sits as a module in the Control Centre and is available in Croatian, Danish, Dutch, English, French, German, Italian, Romanian, Russian and Spanish..

siduxcc contains 7 modules with the following functions:

1. Display module
  • Change Resolution and Scanning Frequency
  • Set Colordepth and DPI
  • Configure X-Server
  • Install graphic card drivers
  • Font-settings
Kmenu>sidux>sidux Control Centre>Display
2. Internet module
  • Set sidux default browser
Kmenu>sidux>sidux Control Centre>Internet
3. Kernel module
  • Install new kernel
  • Remove old kernel
  • Install kernel modules
Kmenu>sidux>sidux Control Centre>Kernel
4. Network module
  • Configure networkcards
  • Set Hostname
  • Set Nameservers
  • Install Wlan-Cards (Ndiswrapper)
  • Install Wlan-Cards (fwdetect)
Kmenu>sidux>sidux Control Centre>Network
5. Services module
  • Manage common services (Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Samba, SSH...)
  • Start services at boot
Kmenu>sidux>sidux Control Centre>Services
6. Software module
  • Show upgradable packages
  • Start siduxcc-hermes/meta-installer:
Kmenu>sidux>sidux Control Centre>Software
7. Infomation module
  • Shows you information about your system
Kmenu>sidux>sidux Control Centre>System info

Console version

Start: siduxcc

siduxcc-hermes

  • siduxcc-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match.
in a konsole as user
siduxcc-hermes
or
KMenu>sidux>Hermes (dist-upgrade sentinel)
