Clamav

apt-get install clamav-docs
apt-get install clamav
apt-get install clamav-freshclam

apt-get install clamav-freshclam
da dobijete aktualne signature
Da skenirate
clamscan
Da vidite help meni
man clamscan
Ako želite koristiti GUI frontend za clamav dostupan je za KDE
apt-get install klamav

Site clamav-a

AntiVir PersonalEdition Classic za Linux

Sljedeće informacije su iz jednog forum posta

Da instalirate antvir PE Linux obavite sljedeće naredbe:

1. kao root
# cd /mp

2. Molimo povucite odavde:
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. ekstrahirajte povučenu tar datoteku
# tar xvpzf antivir-workstation-pers.tar.gz

4. Promijenite direktorij
# cd antivir-workstation-pers-2.1.9-18
(ili bilo koji verzijski broj je primjenjljiv u to vrijeme)

5. startujte install skript. i pratite instrukcije za instalaciju.
./install

6. izbrišite staru antivir.conf
# mv /etc/antivir.conf /etc/antivir.conf.old

Nemojte zaboraviti da obavite update
# antivir --update

Site od AntiVir PersonalEdition Classic

rkhunter

rkhunter rootkit skener je skening-alat da osigura vaš sistem. Taj alat traži rootkits, backdoors i lokalne exploits s testiranjem:
- MD5 hash uspoređaj
- Traženje standardnih rootkit datoteka
- Pogrešna prava za binarne datoteke
- Traženje sumnjivih nizova u KM i KLD modulima
- Traženje za sakrivene datoteke
- Opcionalni sken u čistom tekstu i binarnim datotekama
- Rootkit Hunter je izdan kao GPL licenziran projekt i slobodan za svakoga.

apt-get update
apt-get install rkhunter
rkhunter --update

rkhunter će vas isto pitati jeli hoćete da postavite cron da skenira na određeno vrijeme

Skeniranje koristeći rkhunter
rkhunter -c

Site od rkhunter

chkrootkit

chkrootkit je alat da lokalno pronađete znakove rootkita.

apt-get install chkrootkit
Kako skenirati sa chkrootkit:
chkrootkit

chkrootkit traži ove tipove definicija:

ifpromisc.c
provjerava da li je interface u "promiscuous modus".

chklastlog.c
provjerava zadnje brisanje log-datoteka

chkwtmp.c
provjerava wtmp brisanje

chkproc.c
provjerava znakove LKM trojanca

chkdirs.c
provjerava znakove LKM trojanca

strings.c
brzo i prljavo izmjenivanje nizova

chkutmp.c
provjerava utmp brisanje
