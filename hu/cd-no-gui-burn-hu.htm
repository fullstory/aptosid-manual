CD/DVD írása grafikus felület nélkül

FONTOS MEGJEGYZÉS: A sidux, mint Linux LIVE-CD, erős tömörítési technológián alapul, ezért az ISO lemezképfájl lemezre írása odafigyelést igényel. Kizárólag jó minőségű CD (illetve DVD+RW) lemez használatát, DAO (disk-at-once) mód kiválasztását és legfeljebb 8x-os írási sebességet ajánlunk a megfelelő minőség eléréséhez.

burniso

CD/DVD lemezek írásához nincs feltétlenül szükség grafikus felületre.

Megfigyelésünk szerint a problémák többsége inkább a grafikus előtét alkalmazásokban (pl. K3b) jelentkezik, kevésbé a mögöttük dolgozó cd-író programokban (growisofs, wodim vagy cdrdao).

A sidux tartalmaz egy 'burniso' nevű kész szkriptet a sidux ISO kiírásához.

A burniso szkript ISO képfájlok kiírására alkalmas a wodim program meghívásával, automatikusan DAO (disk-at-once) módot és 8x sebességet használva.

apt-get install sidux-scripts

Felhasználóként:

$ cd az/ISO/könyvtára/

$ burniso

A könyvtárban lévő összes ISO képfájl közül választhatunk. Az írás azonnal megkezdődik, ezért győződj meg arról, hogy a meghajtóban van-e üres lemez.

Az író eszköz meghatározása, felhasználóként:

ATAPI eszközök esetén:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Egyéb lehetőségek:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

és

$ wodim -prcap

Néhány hasznos példa:

Információ egy üres CD/DVD lemezről:
$ wodim dev=/dev/scd0 -atip

vagy

$ cdrdao disk-info --device ATA:1,0,0
Újraírható lemez törlése:
$ wodim -blank=fast -v dev=/dev/scd0

vagy

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
CD másolása:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
CD másolás röptében, két eszköz használatával:
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Audio-CD létrehozása wav fájlokból, 12x írási sebességgel:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav
CD írása bin/cue fájlokból:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
ISO lemezképfájl kiírása:
$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

Úgynevezett Driveropts hiba akkor jelentkezhet, ha a burnfree-vel túlságosan új eszközre írunk. Ekkor használjuk a következőt:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

vagy

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso
ISO képfájl létrehozása egy könyvtárban található fájlokból (és alkönyvtárakból).

Ezután az előző parancs segítségével ki is írható (ISO lemezképfájl kiírása):

$ genisoimage -o myImage.iso -r -J -l directory

Ha DVD íróval rendelkezel, a growisofs-t használhatod DVD íráshoz, csakúgy, mint ISO fájlok DVD-re írásához:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Több fájl DVD-re írása:
$ growisofs -Z /dev/dvd -R -J adat1 adat2 adat3 ...
Ha maradt még hely a DVD-n, további fájlokat adhatsz hozzá:
$ growisofs -M /dev/dvd -R -J másikfájl ésmégegyfájl...
A szakasz lezárása így történik:
$ growisofs -M /dev/dvd=/dev/zero
