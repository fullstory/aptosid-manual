sidux-*.iso - Root-jelszó a sidux LiveCD-n

Figyelmeztetés: Mielőtt a root felhasználó nevében teszel bármit, gondold végig, mit teszel! A web böngészéshez nincs szükség root felhasználóra.

In case of a lockout on a sidux-*.iso you need to do the following and follow the prompts to set a password:

alt+ctrl+F1
sudo passwd

Once the password is active, action the following:

alt+ctrl+F7

sidux-*.iso default password

A Sidux Live CD-n nincs root jelszó beállítva. Egy program root-jogokkal való futtatására többféle lehetőséged van:

Egy (időszakos) root jelszó beállítása

Nyiss meg egy parancssort:

sidux@0[sidux]$ sudo passwd
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
sidux@0[sidux]$

A beállított jelszót a gép újraindításáig lehet használni.

sudo is not supported on hard disk installations, see sudo and sux.

Program futtatása root shell-ből

Gépeld a sux parancsot a terminálba, hogy root jogosultságot szerezz.

Parancssorban:

sidux@0[sidux]$ sux
root@0[sidux]# gparted (vagy bármilyen egyéb parancs...)

Néhány KDE alkalmazást a dbus-launch alkalmazással együtt kell elindítani:

dbus-launch <Alkalmazás>

Más lehetőség, amelyet a népszerűbb desktop környezetekben használhatsz:

Alt+F2 su-to-root -X -c <Alkalmazás>

Példa:

Alt+F2 su-to-root -X -c gparted

A root parancssor bezárásához gépeld be:

exit

vagy kattints a parancssor ablak jobb felső sarkába a bezáráshoz.

Programok telepítése a Live CD használata közben

apt-get update
apt-get install a-telepítendő-alkalmazás

Megjegyzés: A Live CD leállítása után a változások elvesznek, kivéve ha a fromiso és persist boot paraméterekkel használod.

NTFS partíciók írása az ntfs-3g segítségével

Figyelmeztetés: Bár az ntfs-3g-t stabilnak nyilvánították, külső egységre való mentés nélkül ne használd, főleg ne éles szerveren! Ha mégis használod és ennek következtében elvesznek az adataid, az egyedül a Te hibád; csak saját felelősségedre használd!

Egy parancssorba gépeld a következőket (és lásd a Merevlemez partícionálása - Diszk nevek fejezetet).

sux
apt-get update && apt-get install ntfs-3g
umount /media/xdxx
mount -t ntfs-3g /dev/disk/by-uuid/xxyyzz[etc] /media/xdxx
A parancssorból való kilépéshez gépeld be: exit

Az NTFS partíció fel van csatolva írhatóan, tehát képes az adataid eltárolására. Ismét figyelmeztetünk! Csak vészhelyzetben használd, a napi használat nem ajánlott.
