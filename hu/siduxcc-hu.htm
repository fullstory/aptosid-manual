sidux Vezérlő Központ

A sidux vezérlő központ egy frontend a sidux kezelésére, és egy modulként foglal helyet a Vezérlő Központban. Többek közt elérhető Horvát, Dán, Holland, Angol, Francia, Német, Olasz, Japanese, Polish, Portuguese, Román, Orosz, Spanyol és Greek nyelven. It is available in KDE and GTK versions.

A siduxcc 7 modult tartalmaz, a következő funkciókkal:

1. Kijelző modulok
  • Felbontás és Scannelési Frekvencia megváltoztatása
  • Színmélység és DPI beállítása
  • X-Szerver konfigurálása
  • Grafikus kártyák illesztőprogramjának telepítése
Kmenü>Rendszer>sidux Vezérlő Központ>Display(Kijelző)
2. Internet modul
  • sidux alapértelmezett böngésző beállítása
Kmenu>Rendszer>sidux Vezérlő Központ>Internet
3. Kernel modul
  • Új kernel telepítése
  • Régi kernel eltávolítása
  • Kernel modulok telepítése
Kmenu>System>sidux Vezérlő Központ>Kernel
4. Hálózati modulok
  • Hálózati vezérlők beállítása
  • Számítógépnév Megadása
  • Névszerverek Megadása
  • Wlan-Kártyák telepítése (Ndiswrapper)
  • Wlan-Kártyák telepítése (fwdetect)
Kmenu>System>sidux Vezérlő Központ>Hálózat
5. Szolgáltatások Modul
  • Általános szolgáltatások kezelése (Indítás, Leállítás, Újraindítás, Újratöltés, Futási Szint bejegyzések hozzáadása/eltávolítása) (Apache, Samba, SSH...)
  • Szolgáltatások elindítása rendszerbetöltésnél
Kmenu>System>sidux Vezérlő Központ>Szolgáltatások
6. Információs modul
  • Információ mutatása a rendszeredről
Kmenu>System>sidux Vezérlő Központ>Rendszer Információ

sidux hermes

sidux-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match. There is also a gtk version.

sidux-hermes-qt4

It is recommended to install the qt4 version. It has the most features and it works on all desktops (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 has options like kernel information and the ability to download files ready to update (dist-upgrade to be carried out in init 3 later). 

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Use the gtk2 version if you want to have an application without qt dependencies.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contains the backend of all hermes versions. It will be automatically installed when you install the qt4 or gtk2 versions.

apt-get install sidux-hermes
