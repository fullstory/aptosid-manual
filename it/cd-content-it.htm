Contenuti del Live ISO

sidux fornisce solo software conforme al Debian dfsg-free nella ISO live, vedere questo collegamento se si desiderano ulteriori informazioni sulle fonti non-libere di firmware.

L'immagine ISO è completamente basata su Debian sid (aggiornata al 2009-07-xx) arricchita e stabilizzata con i pacchetti e gli script propri di sidux. Viene fornita con gli ultimi kernel sidux ottimizzati, i quali sono basati sul kernel standard 2.6.3x.xx. ACPI e DMA sono abilitati per default.

Varianti di rilascio ISO

sidux propone 5 live-ISO aggiornati come punto d'accesso a Debian Sid e l'installazione di solito non impiega più di 10-20 minuti.

Le varianti sono:
  1. KDE lite 32 bit , live-ISO di circa 600MB: include un minimo di applicazioni per il desktop KDE4. Aggiungere altre applicazioni per soddisfare le necessità di lungo termine può essere fatto semplicemente attraverso apt.
  2. KDE lite 64 bit , live-ISO di circa 600MB: include un minimo di applicazioni per il desktop KDE4. Aggiungere altre applicazioni per soddisfare le necessità di lungo termine può essere fatto semplicemente attraverso apt.
  3. XFCE 32 bit, live-ISO di circa 500MB: include un ambiente desktop completo (non minimale) ed ha tutte le applicazioni per essere produttivi da subito. Richiede minori risorse rispetto a KDE4.
  4. XFCE 64 bit , live-ISO di circa 500MB: include un ambiente desktop completo (non minimale) ed ha tutte le applicazioni per essere produttivi da subito. Richiede minori risorse rispetto a KDE4.
  5. KDE-full, a combined 32 bit + amd64 bit, live-ISO di circa 2.5GB: include un insieme completo di applicazioni per KDE4 con il supporto a 16 lingue e, fra le altre cose, suite per ufficio come OpenOffice.org e browser web come Iceweasel (Firefox).

Applicazioni e strumenti

Requisiti di sistema minimi for KDE4 and XFCE

AMD64

  • CPU:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • qualsiasi CPU con x86-64/EM64T oppure più recente
    • Nuovi processori a 64 bit AMD Sempron e Intel Pentium 4 (controllare il flag "lm" in /proc/cpuinfo oppure utilizzare infobash -v3).
  • RAM:
    • KDE: ≥512 MB RAM (≥768 MB RAM recommended), ≥1 GB RAM for liveapt.
    • XFCE: ≥256 MB RAM, ≥512 MB RAM for liveapt.
  • scheda grafica VGA con almeno una risoluzione di 640x480 pixel.
  • lettore CD/DVD oppure un supporto USB.
  • ≥3 GB HDD di spazio, ≥10 GB consigliati.

i686

  • CPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (no K5/ K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, no C3 Samuel or Ezra)/ C7
    • qualsiasi CPU con x86-64/EM64T oppure più recente
    • è richiesto il set completo dei comandi i686.
  • RAM:
    • KDE: ≥384 MB RAM (≥768 MB RAM recommended), ≥512 MB RAM for liveapt.
    • XFCE: ≥192 MB RAM, ≥512 MB RAM for liveapt.
  • scheda grafica VGA con almeno una risoluzione di 640x480 pixel.
  • lettore CD/DVD oppure un supporto USB.
  • ≥3 GB HDD di spazio, ≥10 GB consigliati.

Applicazioni e strumenti

La ISO di sidux contiene una varietà di programmi e strumenti che coprono quasi ogni necessità di un utente nell'uso giornaliero.

Scelta di 2 dei maggiori Desktop Manager per accedere ai programmi/applicazioni:
KDE 4.2.x (en + de), uno dei migliori desktop manager
Xfce4 4.6.1 (en + de), un desktop manager di medio spessore.

Fluxbox 1.1.1 (en + de) is included on both variations as a lightweight desktop manager option.

Per navigare in internet, a seconda della versione, ci sono Konqueror e Iceweasel.

Applicazioni per la produttività d'ufficio (di nuovo a seconda della versione e solo per nominarne alcune), Koffice, Openoffice.org, Abiword, Gnumeric fino alla gestione dei file con Dolphin e Thunar.

Per la configurazione delle connessioni di rete e di internet c'è Ceni e per il WIFI vedere la nostra documentazione WiFi - roaming WPA-GUI .

Le informazioni per i driver non liberi si trovano qui.

Per il partizionamento dell'hard disk forniamo cfdisk e GParted, quest'ultimo ha una opzione per il ridimensionamento delle partizioni ntfs.

Sono inclusi anche strumenti diagnostici come Memtest86+, uno strumento avanzato di diagnosi della memoria.

All ISO variations contain a full set of command line interface tools. Un file manifest di ognuna delle varianti rilasciate è incluso in ogni mirror di download: Mirror sidux, download e masterizzazione.
