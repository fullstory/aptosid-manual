Contenuti del Live CD

Per via della complessità della legge, sidux fornirà sul liveCD soltanto software libero secondo le linee dfsg, vedere questo collegamento se si desiderano ulteriori informazioni sulle fonti non-libere di firmware.

L'immagine ISO è completamente basata su Debian sid (aggiornata al 2008-12-xx) arricchita e stabilizzata con i pacchetti e gli script propri di sidux. È distribuita con il kernel 2.6.27.slh.xx (smp, hard preemption), il quale è basato sul più recente kernel ufficiale, con l'aggiunta delle patch avanzate di sidux.

ACPI e DMA sono abilitati per default (possono essere disabilitati con "acpi=off" e "nodma" rispettivamente nella riga di boot di grub).

Requisiti di sistema minimi:

  • amd64:
    • AMD64
      Intel Core2
      nuovi processori a 64 bit AMD Sempron e Intel Pentium 4 (controllare il flag "lm" in /proc/cpuinfo oppure utilizzare infobash -v3).
    • ≥256 MB RAM, ≥ 512 MB RAM per liveapt.
    • scheda grafica VGA con almeno una risoluzione di 640x480 pixel.
    • lettore CD/DVD oppure un supporto USB.
    • ≥3 GB HDD di spazio, ≥10 GB consigliati.
  • i686:
    • Intel Pentium pro/Pentium II
      AMD K7 Athlon (no K5/K6)
      VIA C3-2 (Nehemiah, no C3 Samuel o Ezra)/C7
      qualsiasi CPU con x86-64/EM64T oppure più recente; è richiesto il set completo dei comandi i686.
    • ≥192 MB RAM, ≥ 512 MB RAM per liveapt.
    • scheda grafica VGA con almeno una risoluzione di 640x480 pixel.
    • lettore CD/DVD oppure un supporto USB.
    • ≥3 GB HDD di spazio, ≥10 GB consigliati.
Applicazioni e strumenti

Il CD di sidux contiene una varietà di programmi e strumenti che coprono quasi ogni necessità di un utente nell'uso di ogni giorno.

KDE 3.5.10 (en + de). L'ambiente desktop grafico che permette all'utente di accedere a programmi e applicazioni con un click.

Per la navigazione di internet c'è Konqueror uno dei più brillanti browser web con qualcosa in più: integra un gestore di file, un client FTP e un trasferimento sicuro di file ssh-gui (sftp): molto raro nel mondo dei browser web, ed iceweasel come alternativa.

Per la rete e la connessione internet c'è Andare online / Come configurare la rete con lo script "Ceni" insieme a wpa/wlan e wpa roaming GUI e i driver non-free necessari qui.

Per partizionare un disco fisso troviamo sia GParted che cfdisk. GParted ora include anche la funzione per ridimensionare una partizione NTFS!

Sono presenti anche strumenti per la diagnostica come Memtest86+ - uno strumento per la diagnostica avanzata della memoria.

Su ogni sito mirror di scaricamento del liveCD si trova un completo elenco di ogni variante di tutte le versioni.
