Download del CD di sidux

Bittorrent è un protocollo per la condivisione di file, il client è disponibilie sul CD di sidux. Questo permette di eseguire contemporaneamente download e upload di file. Consulta sidux.com per il link al torrent giusto. Poi basta eseguire il comando:

# btdownloadcurses --max_upload_rate 12 ftp://dummy/sidux/sidux-200xxxxx/sidux-200xxxxx.iso.torrent

in una console; subito dopo il tuo download dovrebbe cominciare. Assicurati che ci sia abbastanza spazio libero (circa 700 MB) per il file ISO nella cartella da cui hai eseguito il comando. Una volta completato il tuo download, per favore tieni aperto il programma così da condividere il file con altri utenti. Questo ridurrà il carico totale sui server, e i file si diffonderanno più rapidamente. Un buon tutorial è Brians BitTorent FAQ and Guide

Non dimenticare di verificare il checksum md5 del download.

Se qualcuno ha a disposizione un server FTP veloce con sufficiente banda, per favore informi il team di sidux. In alternativa vada sui Forum di sidux o nel Canale IRC su oftc.net

Checksum MD5 e Verifica

MOLTO IMPORTANTE: sidux, come LIVE-CD Linux, si basa su una tecnologia ad alta compressione, (adottata inizialemente da Knoppix, ma non identica a quella di Knoppix) e, per questo motivo, è richiesta particolare attenzione per masterizzare l'immagine ISO. Utilizza solo supporti CD (o DVD+RW) di alta qualità e masterizza in modalità DAO (disk-at-once) e non più velocemente di 8x.

Si tratta di un checksum utilizzato per verificare l'integrità dei file. Il checksum MD5 corrente del file viene confrontato con un checksum noto. In questo modo è possibile verificare se un file è stato danneggiato o modificato; ne è consigliato l'uso in caso di file scaricati da internet.

Se il checksum md5 del file scaricato corrisponde a quello del file md5, puoi essere sicuro che il file è stato scaricato correttamente. Su Linux puoi ottenere il checksum con: (il calcolo richiede un po' di tempo)

$ md5sum file_da_controllare

Il checksum verrà scritto sulla console.
Con md5summer (486 kb) il checksum md5 può essere verificato anche in Windows.

Le immagini ISO di sidux sono sempre fornite con un file checksum md5 e dovrebbero essere sempre controllate prima della masterizzazione. Questo garantisce che, in caso di problemi, essi non sono dovuti al file scaricato, e libera i forum da problemi che non possono essere risolti perché l'immagine ISO era danneggiata. Si dovrebbe effettuare il controllo in una console. Basta spostarsi nella cartella che contiene l'immagine ISO e il file con il checksum md5 e poi eseguire:

$ md5sum -c sidux.iso.md5

Se non corrispondono, otterrai un errore:

"sidux.iso: Error
md5sum: Warning: calculated checksum does not match!"

Se il file scaricato è corretto, il programma termina senza messaggi.

How Do I Burn The CD With Windows?

MOLTO IMPORTANTE: sidux, come LIVE-CD Linux, si basa su una tecnologia ad alta compressione, (adottata inizialemente da Knoppix, ma non identica a quella di Knoppix) e, per questo motivo, è richiesta particolare attenzione per masterizzare l'immagine ISO. Utilizza solo supporti CD (o DVD+RW) di alta qualità e masterizza in modalità DAO (disk-at-once) e non più velocemente di 8x.

Naturalmente, puoi masterizzare il CD anche con Windows. Il file scaricato va masterizzato come file ISO. Se Winrar (o qualche altro strumento di archiviazione) è associato ai file .ISO, potrebbe essere seccante perché si potrebbe scambiarlo per un archivio. Dal file ISO devi masterizzare un CD. Non masterizzare il file ISO come disco dati!! È necessario masterizzare il file ISO come Disk-At-Once

Hai a disposizioni varie e valide opzioni per masterizzare in Windows.

Masterizzare il CD con Linux

MOLTO IMPORTANTE: sidux, come LIVE-CD Linux, si basa su una tecnologia ad alta compressione, (adottata inizialemente da Knoppix, ma non identica a quella di Knoppix) e, per questo motivo, è richiesta particolare attenzione per masterizzare l'immagine ISO. Utilizza solo supporti CD (o DVD+RW) di alta qualità e masterizza in modalità DAO (disk-at-once) e non più velocemente di 8x.

Se utilizzi già Linux sulla tua macchina, dovresti masterizzare il cd o da shell o con uno dei programmi di masterizzazione che hai installato.

Con sidux e altre distribuzioni Linux viene fornito K3b come strumento di masterizzazione di default. Avvialo, segli "Extras" -> "burn CD-Image..." e scegli il file ISO da masterizzare (per esempio sidux.iso).

Prima di tutto K3B calcola il checksum MD5 per il file ISO (questo richiede un po' di tempo). Se il checksum mostrato è simile al checksum del file MD5 (per esempio sidux.iso.md5), il tuo download è stato eseguito con successo e puoi masterizzare il file semplicemente cliccando su "Start".

Creare un CD di sidux usando Patch XDELTA

MOLTO IMPORTANTE: sidux, come LIVE-CD Linux, si basa su una tecnologia ad alta compressione, (adottata inizialemente da Knoppix, ma non identica a quella di Knoppix) e, per questo motivo, è richiesta particolare attenzione per masterizzare l'immagine ISO. Utilizza solo supporti CD (o DVD+RW) di alta qualità e masterizza in modalità DAO (disk-at-once) e non più velocemente di 8x.

xdelta è un programma che, tra le altre cose, crea una nuova immagine ISO da una immagine ISO iniziale e alcuni file modificati. Permette anche di eseguire correzioni di piccoli bug senza incorrere in elevati costi di download.

I file modificati da PATCH-RC1-RC2.XDELTA hanno, per esempio, dimensioni di soli 5.4 MB, mentre l'immagine ISO stessa è di 553 MB.

Supponiamo che tu abbia:

  • Un immagine ISO su hard disk, per esempio sidux.iso
  • Un file modificato, per esempio PATCH-RC1-RC2.XDELTA

Basta scrivere in un terminale/console come utente:

$ cd /path/to/iso/and/patch
$ xdelta patch PATCH-RC1-RC2.XDELTA

Il programma estrarrà i file necessari dalla patch. Eseguirà anche la verifica del checksum md5. Ricorda: non controllerà la disponibilità la presenza di spazio libero a sufficienza sul disco.
