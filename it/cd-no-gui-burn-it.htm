Masterizzare CD/DVD senza Interfaccia Grafica

MOLTO IMPORTANTE:sidux, come liveCD Linux, si basa su una tecnologia ad alta compressione, e, per questo motivo, è richiesta particolare attenzione per masterizzare l'immagine ISO. Utilizzare solo supporti CD (o DVD+RW) di alta qualità e masterizzare in modalità DAO (disk-at-once) non più velocemente di 8x.

burniso

Non c'è bisogno di un'interfaccia grafica per masterizzare un CD/DVD.

Di solito i problemi di masterizzazione si hanno con il front-end (k3b), non con i back-end (growisofs, wodim o cdrdao).

sidux fornisce uno script già pronto per masterizzare una ISO di sidux stessa che si chiama 'burniso'.

burniso masterizza immagini ISO, utilizzando la modalità Disk-At-Once ed una velocità di 8x, tramite wodim.

apt-get install sidux-scripts

come utente ($):

$ cd /dir/contenente/la/ISO

$ burniso

Tutte le immagini ISO nella directory corrente vengono listate e la masterizzazione inizierà subito dopo la selezione di una di queste, quindi assicurarsi di avere un supporto ottico inserito.

Quali dispositivi utilizzare, come utente ($):

Per i dispositivi ATAPI usare:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/hdc'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/hdd'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Le altre alternative sono:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

e

$ wodim -prcap

Ecco un po' di esempi utili:

Informazioni per CD/DVD vuoto:
$ wodim dev=/dev/hdc -atip

oppure

$ cdrdao disk-info --device ATA:1,0,0
Formattare un disco riscrivibile:
$ wodim -blank=fast -v dev=/dev/hdc

oppure

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal
Clonare un cd:
$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2
Clonare un cd "al volo":
$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2
Creare un cd audio a velocità 12x partendo da file wave:
$ wodim -v -eject -pad -dao speed=12 dev=/dev/hdc defpregap=0 -audio *.wav
Masterizzare un cd da file bin/cue:
$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue
Masterizzare un'immagine ISO:
$ wodim dev=/dev/hdc driveropts=burnfree,noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

Se si ottiene un errore driveropts, è dovuto a burnfree che è deprecato per alcune periferiche, quindi:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v qualcosa.iso

oppure

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v qualcosa.iso
Creare un'immagine ISO con tutti i file (e sottodirectory) di una directory.

In seguito questa può essere masterizzata con il comando precedente (masterizzare un'immagine ISO):

$ genisoimage -o miaImmagine.iso -r -J -l directory

Se si possiede un masterizzatore DVD, si può utilizzare anche growisofs per masterizzare DVD, ecco come fare per masterizzare un'immagine ISO su DVD:

$ growisofs -dvd-compat -Z /dev/dvd=immagine.iso
Masterizzare file multipli su DVD:
$ growisofs -Z /dev/dvd -R -J dato1 dato2 dato3 ...
Se è rimasto spazio libero sul DVD, si possono aggiungere altri file:
$ growisofs -M /dev/dvd -R -J unaltrofile eunaltrofile...
Per finalizzare la sessione, usare:
$ growisofs -M /dev/dvd=/dev/zero
