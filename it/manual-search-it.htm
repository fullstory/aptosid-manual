Ricerca On-Line

Opzione 1

Scrivere una o più parole nel campo di ricerca e premere Cerca (oppure premere Invio).



Matching any words Matching all words

Language:

Hits per page:

Opzioni di ricerca

Se è selezionato Cerca TUTTE le parole, verranno visualizzati i documenti che contengono tutte le parole della ricerca. Con Cerca QUALSIASI parola verranno visualizzati i documenti che contengono almeno una parola della ricerca.

In alternativa si può mettere un segno più (+) direttamente davanti ad una o più parole per ottenere solo quei file che includono tutte quelle parole. Scrivendo invece le parole con un segno meno (-) direttamente davanti ad esse modifica il risultato in modo che siano visualizzati solo i documenti che non contengono nessuna di queste parole.

I risultati sono ordinati per rilevanza con i documenti maggiormente attinenti alla ricerca per primi. La rilevanza dipende dal numero e dalla posizione delle parole trovate nei documenti.

Opzione 2

Per fare ricerche on line, scrivere o incollare dentro la casella di ricerca del browser, (dove xxxx è la parola che si vuole cercare), la seguente riga:

xxxxx site:manual.aptosid.com/it

Ricerca Off-line

Il manuale del sistema operativo aptosid ha creato alcune configurazioni speciali insieme allo sviluppatore di Recoll come parte del pacchetto default. Recoll è uno strumento di ricerca personale di testo per Unix/Linux. Si basa sul solido backend di Xapian per il quale fornisce un'interfaccia grafica sviluppata con QT semplice ma piena di funzioni. Un ringraziamento speciale a Recoll.

Recoll si chiama Recoll - Local Text Search nel menu Accessori di KDE.

Passo 1

Il manuale è attualmente disponibile attraverso apt per le seguenti lingue:
Default - Deutsch (Tedesco, -de)
Default - English (Inglese, -en)
*Čeština (Ceco, -cz)
*Dansk (Danese, -da)
*Español (Spagnolo, -es)
*Français (Francese, -fr)

*Nihongo (Giapponese, -ja)
*Nederlands (Olandese, -nl)
*Polski (Polacco, -pl)
*Português (Portoghese, -pt-br)
*Română (Rumeno,- ro)
*Русский (Russo, -ru)
*Українска (Ucraino, -uk)

apt-get update
apt-get install aptosid-manual-xx (dove -xx è la tua lingua. Per esempio: aptosid-manual-en, aptosid-manual-de, aptosid-manual-es, ...)
Passo 2
apt-get install recoll
Passo 3
  • Recoll al primo avvio chiederà se si vuole costruire il database, premere cancel, comparirà una finestra per la configurazione dell'indice.
  • In Global parameters - Top directories box, premere su + (più) poi andare su /usr/share/aptosid-manual, scegliere la lingua e premere ok. Quindi premere su - (meno) e cancellare tutte le directory compreso ~ (tilde).
  • Infine File > Update Index e cercare l'argomento di cui si ha bisogno nel manuale aptosid.

Se si deve inizializzare recoll a causa dell'aggiornamento di aptosid-manual attraverso apt, (come utente $ in konsole):

recollindex -z

Quindi in Recoll: File > Aggiorna Indice.

Il collegamento 'Preview' evidenzierà tutti i risultati della parola cercata nel file, mentre 'Open' aprirà Konqueror, e con l'utilizzo di Cerca in questa pagina si può fare una micro ricerca della parola che è stata cercata con recoll.
