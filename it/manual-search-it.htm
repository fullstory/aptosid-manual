Ricerca On-Line

Scrivere una o più parole nel campo di ricerca e premere Cerca (oppure premere Invio).

Opzioni di ricerca

Se è selezionato Match ALL words, verranno visualizzati i documenti che contengono tutte le parole della ricerca. Con Match ANY word verranno visualizzati i documenti che contengono almeno una parola della ricerca.

In alternativa si può mettere un segno più (+) direttamente davanti ad una o più parole per ottenere solo quei file che includono tutte quelle parole. Scrivendo invece le parole con un segno meno (-) direttamente davanti ad esse modifica il risultato in modo che siano visualizzati solo i documenti che non contengono nessuna di queste parole.

Notare che la ricerca di frasi non è supportata, ad es. non ha senso mettere gli apici (') o i doppi apici (") per deliitare una frase "come questo".

I risultati sono ordinati per rilevanza con i documenti maggiormente attinenti per primi. La rilevanza dipende dal numero e dalla posizione delle parole trovate nei documenti.

Ricerca Off-line

Il manuale del s.o. sidux ha creato alcune configurazioni speciali insieme allo sviluppatore di Recoll, uno strumento di ricerca testo per Unix/Linux. Si basa sul solido backend di Xapian per il quale fornisce un'interfaccia grafica sviluppata con QT semplice ma piena di funzioni.

Recoll si chiama Recoll - Local Text Search nel menu Accessori di KDE.

Per configurare la ricerca locale è necessario fare alcune operazioni!

Passo 1

Il manuale è attualmente disponibile attraverso apt per le seguenti lingue:
da, de, el, en, es, fr, hr, it, ja, nl, pt-br, ro, ru.

apt-get update
apt-get install sidux-manual-xx (dove -xx è la tua linuga)
Passo 2
apt-get install recoll
Passo 3

Scaricare il file di recoll specifico per sidux come utente ($) in una directory temporanea ed estrarlo:

wget http://bluewater.sidux.com/outgoing/sidux-recoll-configs.tar.gz

Creare una directory .recoll nella $HOME, se non c'è già, (diventa una directory nascosta).

Copiare i seguenti file sidux-recoll-config in .recoll
mimeconf
mimemap
mimeview
recoll.conf

Per i passi 4 e 5 si deve essere root per modificare le directory /usr/share/recoll :
Passo 4

Copiare il file chiamato rclsiduxman che è nella directory filters di sidux-recoll-config, in /usr/share/recoll/filters .

mv sidux-recoll-configs/filters/* /usr/share/recoll/filters
Passo 5

Copiare il file chiamato sidux-book.png che è nella directory images di sidux-recoll-config, in /usr/share/recoll/images .

mv sidux-recoll-configs/images/* /usr/share/recoll/images
Passo 6

Come root in konsole:

update-menus

Come utente nel Kmenu, andare a Kmenu > Accessori > Recoll - Local Text Search.

Recoll al primo avvio chiederà se si vuole costruire il database, cliccare Cancel, quindi File > Aggiorna Indice e poi cercare l'argomento di cui si ha bisogno nel manuale sidux.

Se si deve inizializzare recoll a causa dell'aggiornamento di sidux-manual attraverso apt, (come utente $ in konsole):

recollindex -z

Quindi: Kmenu > Accessori > Recoll - Local Text Search e in Recoll: File > Aggiorna Indice.

Nel caso si avesse bisogno, in Konqueror, usare il comando Cerca in questa pagina per una micro ricerca della parola che è stata cercata con recoll.
