Centro di controllo sidux

Il Centro di controllo sidux è un'interfaccia grafica per un insieme di moduli di gestione del sistema, ed è disponibile in Croato, Danese, Tedesco, Inglese, Francese, Italiano, Giapponese, Polacco, Portoghese, Rumeno, Russo, Spagnolo e Greco. È disponibile nelle versioni per KDE e GTK.

siduxcc contiene 7 moduli con le seguenti funzioni:

1. Modulo Display
  • Cambia la risoluzione e la frequenza di scansione dello schermo
  • Imposta la profondità di colore ed i DPI (dot per inch: punti per pollice)
  • Configura il server X
  • Installa i driver delle schede grafiche
Kmenu > sidux > sidux Control Centre > Display
2. Modulo Internet
  • Imposta il navigatore internet predefinito
Kmenu > sidux > sidux Control Centre > Internet
3. Modulo Kernel
  • Installa un nuovo kernel
  • Rimuove un vecchio kernel
  • Installa i moduli del kernel
Kmenu > sidux > sidux Control Centre > Kernel
4. Modulo Network
  • Configura le schede di rete
  • Imposta l'Hostname
  • Imposta i Nameserver
  • Installa le schede di rete wireless (via Ndiswrapper)
  • Installa le schede di rete wireless (via fwdetect)
Kmenu > sidux > sidux Control Centre > Network
5. Modulo Services
  • Gestisce i servizi comuni (con comandi di Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Samba, SSH...)
  • Fa partire i servizi all'avvio del sistema
Kmenu > sidux > sidux Control Centre > Services
6. Modulo Information
  • Mostra informazioni sul sistema
Kmenu > sidux > sidux Control Centre > System info

sidux hermes

sidux-hermes (dist-upgrade sentinel) è un'applicazione kde che avverte quando ci sono problemi con dist-upgrade. Cerca nella sezione News di sidux.com per la parola "Warning" e mostra una luce rossa in caso positivo. C'è anche una versione gtk.

sidux-hermes-qt4

Si raccomanda di installare la versione qt4. Ha il maggior numero di opzioni e funziona su tutti i desktop (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 contiene opzioni come informazioni kernel e la capacità di scaricare file per poi aggiornare il sistema (dist-upgrade da eseguire successivamente in init 3).

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Utilizzare la versione gtk2 se si vuole un'applicazione senza le dipendenze qt.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contiene il backend di tutte le versioni. Viene automaticamente installato insieme alla versione qt4 oppure a quella gtk2.

apt-get install sidux-hermes
