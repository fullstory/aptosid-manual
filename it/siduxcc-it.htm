sidux Control Centre

Il centro di controllo di sidux e' un'interfaccia grafica per un insieme di moduli di gestione di sidux, riuniti nel Centro di Controllo ed e' disponibile in Croato, Danese, Tedesco, Inglese, Francese, Italiano, Rumeno, Russo e Spagnolo..

siduxcc contiene 7 moduli con le seguenti funzioni:

1. Modulo Display
  • Cambia la risoluzione e la frequenza di scansione dello schermo
  • Imposta la profondita' di colore e i DPI
  • Configura il server X
  • Installa i driver delle schede grafiche
  • Impostazioni dei caratteri
Kmenu>sidux>sidux Control Centre>Display
2. Modulo Internet
  • Imposta il navigatore internet predefinito
Kmenu>sidux>sidux Control Centre>Internet
3. Modulo Kernel
  • Installa un nuovo kernel
  • Rimuove un vechchio kernel
  • Installa i module del kernel
Kmenu>sidux>sidux Control Centre>Kernel
4. Modulo Network
  • Configura le schede di rete
  • Imposta lo Hostname
  • Imposta i Nameservers
  • Installa le schede di rete wireless (via Ndiswrapper)
  • Installa le schede di rete wireless (via fwdetect)
Kmenu>sidux>sidux Control Centre>Network
5. Modulo Services
  • Gestisce i servizi comuni (con comandi di Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Samba, SSH...)
  • Fa partire i servizi all'avvio del sistema
Kmenu>sidux>sidux Control Centre>Services
6. Modulo Software
  • Mostra i pacchetti aggiornabili
  • Avvia il Sidux-hermes/meta-installer:
Kmenu>sidux>sidux Control Centre>Software
7. Modulo Infomation
  • mostra informazioni sul vostro sitema
Kmenu>sidux>sidux Control Centre>System info

siduxcc-hermes

  • siduxcc-hermes (la sentinella di dist-upgrade) e' una applicazione per il vassoio (tray)del desktop KDE che vi avvisa se ci sono dei problemi con dist-upgrade problems. Esso scansione la sezione news di sidux.com cercando la parola chiave 'Warning' e mostra una luce rossa di stop nel caso ne dovesse trovare una.
in a konsole as user
siduxcc-hermes
or
KMenu>sidux>Hermes (dist-upgrade sentinel)
