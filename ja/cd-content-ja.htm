ライブISOの内容

sidux only provides Debian dfsg-free software on the live-ISO。 firmwareのnon-freeソースについての追加情報はこちらへ

このISOは(2009-07-xx)Debian Sidに完全にもとづいており、sidux独自のパッケージとスクリプトを加えて豊かに、より安定したものになっています。It comes with the latest enhanced sidux kernel, which is based on the most recent vanilla kernel 2.6.3X.xx.。 ACPIとDMAはデフォルトでオンになっています。

いろいろな ISO リリース

sidux はDebian Sid システムへの導入点として5種類の最新ISOリリースを提供しています。インストールは通常10分から20分程度しかかかりません。

リリースの種類とは:
  1. KDE lite 32 bit , live-ISO 約600MB: KDE4 デスクトップの最小限のアプリケーションのみ搭載しています。その他のアプリケーションが長期的に必要であれば、aptシステムを使って簡単にインストールできます。
  2. KDE lite 64 bit , live-ISO 約600MB: KDE4 デスクトップの最小限のアプリケーションのみ搭載しています。その他のアプリケーションが長期的に必要であれば、aptシステムを使って簡単にインストールできます。
  3. XFCE 32 bit, live-ISO 約500MB: (最小限ではない）デスクトップシステム環境を搭載して、インストール後すぐに生産的な作業にとりかかれます。KDE4 より小さくシステムに負担をかけません。
  4. XFCE 64 bit, live-ISO 約500MB: (最小限ではない）デスクトップシステム環境を搭載して、インストール後すぐに生産的な作業にとりかかれます。KDE4 より小さくシステムに負担をかけません。
  5. KDE-full, 32 bit と+ 64 bitの複合ISO, live-ISO 約2.5GB: KDE4のアプリケーションスーツ全部を含み、16言語をサポートして、他にもOpenOffice.orgのオフィススーツやIceWeasel(FireFox)などのアプリケーションも搭載しています。

アプリケーションとツール

KDE4 とXFCE システムの最低必要条件

AMD64

  • 必要なCPU:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • x86-64/ EM64TのできるCPU
    • それ以降の64ビット可能な AMD Sempron または Intel Pentium 4 CPUs (infobash -v3 のコマンドまたは /proc/cpuinfo で ”lm" フラッグが表示されるCPU).
  • 必要なRAMサイズ:
    • KDE: ≥512 MB RAM (なるべく≥768 MB RAM ), ライブAPTオプションによる言語サポートのためには≥1 GB RAM .
    • XFCE: ≥256 MB RAM, ライブAPTオプションによる言語サポートのためには≥512 MB RAM
  • 最低 640x480 ピクセルの解像度のあるVGAグラフィックカード
  • CD/DVDリーダーまたはUSBドライブ.
  • ≥3 GB HDD space, なるべく≥10+ GB

i686

  • 必要なCPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (K5/ K6 はダメ)
    • Intel Atom N-270/ 230
    • VIA C3-2 (NehemiahはOK, C3 Samuel または Ezra はダメ)/ C7
    • x86-64/ EM64TのできるCPU
    • i686 のフルコマンドセットが必要です
  • 必要なRAM:
    • KDE: ≥384 MB RAM (なるべく≥768 MB RAM), ライブAPTオプションによる言語サポートのためには≥512 MB RAM.
    • XFCE: ≥192 MB RAM, ライブAPTオプションによる言語サポートのためには≥512 MB RAM
  • 最低 640x480 ピクセルの解像度のあるVGAグラフィックカード
  • CD/DVDリーダーまたはUSBドライブ.
  • ≥3 GB HDD space, なるべく≥10+ GB

アプリケーションとツール

sidux-ISOには様々なプログラムとツールがあり、デスクトップ・ユーザが日々必要とするものは大部分カバーされています。

二つのデスクトップマネージャーから選ぶことができます:
KDE 4.2.x (en + de), 最先端のデスクトップマネージャー
Xfce4 4.6.1 (en + de), 中規模のデスクトップマネージャー

Fluxbox 1.1.1 (en + de) は軽量デスクトップのオプションとしてどちらの種類のリリースにもふくまれています。

インターネットのサーフィングのためには、リリースの種類によってKonqueror または iceweaselが入っています。

生産的オフィスプログラムとして、リリースごとに様々ですが、例としてKOffice、Openoffice.org、Abiword、Gnumeric など、ファイルマネージャーとして Dolphin または Thunarなどが入っています。

ネットワークやインターネット接続の設定のためにはCeni、無線LANの設定には WIFI Roaming を読んでください。

Non-free ドライバについては こちら.

ハードディスクのパーティションのためには cfdisk と GParted が搭載されています。GParted を使えば NTFS のサイズ変更もできます。

Memtest86+などの、メモリー診断ツールも含まれています。

すべての種類の ISO には通常必要なコマンドラインツールがすべて揃っています。それぞれの ISO に搭載されているソフトウェアの完全なリストはダウンロードミラーサイトのマニフェストファイルを見てください: sidux Mirrors, Downloading and Burning.
