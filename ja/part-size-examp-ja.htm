パーティションサイズの例

普通の使用にあたってはよくメンテナンスされていてsiduxのデフォルトであるext3を推奨します。

GPartedというパーティションマネージャでハードドライブのパーティションやフォーマットができます。 このプログラムにはGUIがあって使い方もすぐわかります。

Gparted can also shrink or move partitions and also manipulate NTFS partitions [with a special caveat that once you alter an NTFS partition you must reboot immediately before doing any other operations ]. Gpartedの文書全体を読むにはこちらへ。 Partition Magic(TM)やAcronis(TM)といった商業ソフトウェアを使ってもNTFSのパーティションの変更ができます。

常にデータのバックアップを!

Should a partition show up as mounted, unmount the device, including swap with a right click on the partition icons in gparted or via a terminal, for example:

umount /dev/sda1

The swap partiton can be unmounted in a terminal with: 

swapoff -a

In principle, 5 GB is more than sufficient for a hd-install, but you won't have much fun with this. A reasonable minimum install should have 12 GB. For those new to linux, we suggest only 2 partitions for a start (root/home and swap), because this simplifies your first install quite a bit, then establish extra partitions for a separate /home and additional data partitions.

You really should have a swap partition (equivalent to the windows swapfile, but is much more effective). For normal usage, the swap partition should be up to twice your ram.

For data-exchange with a Windows installation you should use vfat (fat32) or ext2 as an MS Windows™ driver is available for data-swapping. [XFS is not supported].

Whomever creates more than 1 partition should write their names down for reference!

さまざまなHDサイズやコンピュータの使い方に応じた例をいくつかあげます。 サイズはキリのいい数字に丸めてあります。

試運転の設定 - 60GBのハードディスク, windowsがかなりの部分を占めているケース
Disk Size Filesystem Mountpoint/System
sda1 20GB NTFS WindowsXP
sda5 30GB FAT32 WindouwとLinuxのデータ
sda6 8GB ext3 /
sda7 500MB Swap Swap
TVとビデオ用のPC - 200GB, Windowsなし,(ビデオの)巨大ファイル
Disk Size Filesystem Mountpoint/System
sda1 8GB ext3 /
sda5 6GB ext3 /home
sda6 5GB ext3 /var
sda7 178.5GB ext3 /video
sda8 1GB Swap Swap
標準のPC 120GB Windowsあり、両方を最大限に使う
Disk Size Filesystem Mountpoint/System
sda1 20GB NTFS WindowsXP
sda5 8GB ext3 /
sda6 20GB ext3 /home
sda7 70GB FAT32 WindowsXPとLinuxでデータ交換
sda8 500MB Swap Swap
Linuxでゲームとマルチメディア 160GB
Disk Size Filesystem Mountpoint/System
sda1 18GB ext3 /
sda5 20GB ext3 /home
sda6 30GB ext3 /opt
sda7 90GB ext3 /data
sda8 2GB ext3 /var
sda9 500MB Swap Swap
LinuxとWindowsが20GBのディスクに共存
Disk Size Filesystem Mountpoint/System
sda1 5GB NTFS Win-System
sda2 10GB FAT32 Windowsのプログラムと、WindowsとLinux両方のデータ
sda5 5GB ext3 /
sda6 500MB Swap Swap

この最後の設定はラップトップのようにハードディスクの容量が少ない場合お勧めできる設定です。 Windowsは3~5GBの独自のパーティションに他のプログラムなしでインストールされ、プログラムとデータはすべて別の大きなパーティションに置かれます。 ディスクの容量にさらに余裕があればWindowsのプログラムとデータを２つのパーティションに分けます。

総論

ハードディスクをパーティションに分ける方法はたくさんありますが、スタートとしては上の例があれば十分でしょう。 パフォーマンス、使いやすさとデータの安全を考えるともう一つのハードディスクを加えるのはとても意味のあることです。
その場合MS Windows(TM)は最初のハードディスクの最初のパーティションになります!
