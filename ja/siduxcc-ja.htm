sidux Control Centre

siduxコントロールセンターはsiduxの管理のフロントエンドで、KDEコントロールセンターのモジュールとしても使え、英語、イタリア語、オランダ語、クロアチア語、スペイン語、デンマーク語、ドイツ語、フランス語、ルーマニア語、ロシア語などに翻訳されています。

siduxccには次ような機能の7つのモジュールがあります:

1. ディスプレイモジュール
  • 画素数と走査周波数の変更
  • 色深度とDPI
  • X-サーバーの設定
  • グラフィックカードドライバのインストール
  • フォント設定
Kmenu>sidux>sidux Control Centre>Display
2. インターネットモジュール
  • siduxのデフォルトブラウザを設定
Kmenu>sidux>sidux Control Centre>Internet
3. カーネルモジュール
  • 新しいカーネルをインストール
  • 古いカーネルを取り除く
  • カーネルモジュールをインストール
Kmenu>sidux>sidux Control Centre>Kernel
4. ネットワークモジュール
  • ネットワークカードの設定
  • ホストネーム
  • ネームサーバー
  • Wlanカード(Ndiswrapper)のインストール
  • Wlanカード(fwdetect)のインストール
Kmenu>sidux>sidux Control Centre>Network
5. サービスモジュール
  • よく使われるサービスの管理 (Start, Stop, Restart, Reload, Runlevelへの追加や削除) (Apache, Samba, SSH...)
  • 起動時の自動スタート設定
Kmenu>sidux>sidux Control Centre>Services
6. ソフトウェアモジュール
  • アップグレードできるパッケージの表示
  • siduxcc-hermesとメタインストーラのスタート:
Kmenu>sidux>sidux Control Centre>Software
7. インフォメーションモジュール
  • システムについての情報の表示
Kmenu>sidux>sidux Control Centre>System info

siduxcc-hermes

  • siduxcc-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match.
in a konsole as user
siduxcc-hermes
or
KMenu>sidux>Hermes (dist-upgrade sentinel)
