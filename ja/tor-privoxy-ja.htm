Privoxy

Privoxyとはプライバシー保護やウェッブページのデータ変更、クッキーの管理、アクセスコントロール、広告、バナー、ポップアップなど煩わしいのジャンクの除去といった進んだフィルタリングのできるウェッブプロクシーです。 Privoxyはとても柔軟に設定ができるため個々のユーザーの好みや必要に合わせてカスタマイズできます。 スタンドアローンモードとマルチユーザーモードの使い方が選べます。 PrivoxyはInternet Junkbuster(tm)をベースにしています。

Privoxyのインストール:

apt-get update
apt-get install privoxy

デフォルトのインストールはたいていのユーザーの出発点としてそのまますぐに役に立つと思います。 設定を調整する必要もあるでしょうが、必要の起こった時に徐々に変えていけばいいでしょう。 大抵の場合デフォルトの設定のままかちょっと手直しする程度で十分でしょう。

privoxyの設定ファイルから必要な行を選びコメント記号をとって有効にします。どの行が必要かはユーザーごとに違うでしょう。

Privoxyのユーザーマニュアルと進んだ設定についてはこちらへ

Tor

Torはバーチャルトンネルのネットワークで、これを使ってユーザーやグループのネット上のプライバシーやセキュリティーを向上することができます。 ソフトウェア開発者はこれを使ってプライバシー保護がビルトインされたコミュニケーションツールを作ることもできます。 個人や組織がプライバシーを犠牲にせず公共のネットワークを使って情報を共有するアプリケーションをTorを基盤にして開発することができます。

代償としてインターネットの処理スピードが若干遅くなることも理解してください。

Torのインストール:

apt-get update
apt-get install tor

The default torrc file should work out of the box for most Tor users. You will need to configure Privoxy to use Tor on the internet, see: Tor and internet browsing.

You should also be aware that internet throughput speeds will be negatively affected.

For the Iceweasel brower there is a Torbutton add-on available.

The Tor documentation offers comprehensive on all aspects of Tor.

Vidalia is a controller GUI for the Tor software and allows you to start and stop Tor, view the status of Tor at a glance,and monitor Tor's bandwidth usage.

apt-get update
apt-get install vidalia
