Linux用のAntiVir PersonalEdition Classic

次の情報はこのフォーラムから得たものです

antvir PE Linuxをインストールするには:


1. rootとして
# cd /tmp

2. ここからダウンロードします:
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. ダウンロードしたtarファイルを展開します
# tar xvpzf antivir-workstation-pers.tar.gz

4.ディレクトリをかえます 
# cd antivir-workstation-pers-2.1.9-18

(バージョン番号はファイルによって異なります)

5. インストールスクリプトを実行しあとは指示に従ってください
./install

6. 古いantivir.confは削除します
# mv /etc/antivir.conf /etc/antivir.conf.old

アップデートを忘れないように
# antivir --update

Avira AntiVir PersonalEdition Classicのホームページ

rkhunter

rkhunter rootkit scannerはシステムに厄介なツールがインストールされていないことを確認するツールです. ルートキット、バックドアなどを
- MD5 hashの比較
- rootkitのデフォルトファイルの検索
- バイナリファイルの権利設定のチェック
- LKMとKLDモジュールの中に怪しい文字列を探す
- 隠しファイルの検索
- オプションとしてテキストファイルとバイナリファイルの内容のスキャン
といったテストを行って発見します

apt-get update
apt-get install rkhunter
rkhunter --update

rkhunterのインストールの際、定期的にスキャンを実行するcronも設定できます.

rkhunterでスキャンするには
rkhunter -c

Rootkit HunterはGPLライセンスでリリースされているので、誰でも自由に使えます.

rkhunterのホームページ

chkrootkit

chkrootkitは単独のコンピュータでルートキットがないのを確認するツールです.

apt-get install chkrootkit
chkrootkitでスキャンするには
chkrootkit

chkrootkitは次のようなタイプのチェックを行います:

ifpromisc.c
インターフェイスがプロミスキャスモードでないかかチェック.

chklastlog.c
lastlogが消されていないかチェック

chkwtmp.c
wtmpが消されてないかチェック

chkproc.c
LKMトロヤンの痕跡をチェック

chkdirs.c
LKMトロヤンの痕跡をチェック

strings.c
大雑把なな文字列交換

chkutmp.c
utmpが消されていないかチェック
