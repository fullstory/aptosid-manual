Xfceのシステムにあると便利な追加ソフト

sidux Control Centre

siduxコントロールセンターはGTK2をベースにしたsidux-xfceの管理に役立つGUIのフロントエンドプログラムです。 クロアチア語、デンマーク語、オランダ語、英語、フランス語、ドイツ語、ギリシャ語、日本語、ポーランド語、ポルトガル語、ルーマニア語、ロシア語、スペイン語に翻訳されています。KDEのバージョンもあります。

apt-get update
apt-get install siduxcc-gtk2

Xfce 用の sidux アートとテーマ

以前からインストールされたシステムで最新のsiduxアートに更新するには:

apt-get install sidux-art-xfce-xxxx sidux-art-wallpaper-xxxx
(Where xxxx is the name of the release for example sidux-art-xfce-uranos)

このコマンドでsiduxのワールペーパーとテーマがインストールされます。その後 Xfce の設定メニューでデフォルト設定してください。
