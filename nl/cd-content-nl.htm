Inhoud van de Live ISO

aptosid levert alleen Debian dfsg-free software op de live-ISO, bekijk aub deze link voor meer informatie over niet-gratis bronnen voor firmware.

De ISO is compleet gebaseerd op Debian Sid (recentelijk van de datum 2010-06-13) verrijkt en gestabiliseerd met aptosid eigen pakketten en scripts. It comes with the latest enhanced aptosid kernel, which is based on the most recent vanilla kernel 2.6.3X.xx. ACPI en DMA zijn standaard ingeschakeld.

Een compleet manifest van elke release variant is te vinden op elke download mirror: aptosid Mirrors, Downloaden en branden.

Release ISO variatie's

aptosid biedt 5 up-to-date live-ISO release variatie's van Debian Sid aan en het installeren duurt meestal niet langer dan 10-20 minuten.

De variatie's zijn:
  1. KDE lite 32 bit , live-ISO ongeveer 600MB:
    bevat het minimale aan applications voor de KDE4 desktop. Meer applicaties toevoegen voor je langdurig gebruik is makkelijk te doen via apt.
  2. KDE lite 64 bit , live-ISO ongeveer 600MB:
    bevat het minimale aan applications voor de KDE4 desktop. Meer applicaties toevoegen voor je langdurig gebruik is makkelijk te doen via apt.
  3. XFCE 32 bit, live-ISO ongeveer 500MB:
    bevat een volledig Desktop Omgeving (niet minimaal), en heeft alle applications om direct productief te zijn. Het is kleiner dan KDE4.
  4. XFCE 64 bit , live-ISO ongeveer 500MB:
    bevat een volledig Desktop Omgeving (niet minimaal), en heeft alle applications om direct productief te zijn. Het is kleiner dan KDE4.
  5. KDE-full, een combinatie van 32 bit + amd64 bit, live-ISO ongeveer 2.5GB:
    bevat een volledige suite van KDE4 applicatie's met ondersteuning voor 16 talen en inclusief de volgende dingen, office productiviteit suites zoals OpenOffice.org en web browsers zoals iceweasel (Firefox)

Applicatie's en tools

Systeemeisen voor KDE4 en XFCE

AMD64

  • Processor eisen:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • elke x86-64/ EM64T capable CPU of nieuwer
    • nieuwere 64 bit AMD Sempron en Intel Pentium 4 CPUs (let op de "lm" flag in /proc/cpuinfo of gebruik infobash -v3)
  • Geheugen eisen:
    • KDE: ≥512 MB RAM (≥768 MB RAM aanbevolen), ≥1 GB RAM voor liveapt.
    • XFCE: ≥256 MB RAM, ≥512 MB RAM voor liveapt.
  • VGA grafische kaart welke tenminste een resolutie van 640x480 pixels kan weergeven.
  • Optische diskdrive of USB media.
  • ≥3 GB HDD space, ≥10 GB aanbevolen.

i686

  • Processor eisen:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (niet K5 / K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, niet C3 Samuel of Ezra) / C7
    • elke x86-64/ EM64T capabele CPU of nieuwer
    • de volledige i686 command set is vereist.
  • Geheugen eisen:
    • KDE: ≥384 MB RAM (≥768 MB RAM aanbevolen), ≥512 MB RAM voor liveapt.
    • XFCE: ≥192 MB RAM, ≥512 MB RAM voor liveapt.
  • VGA grafische kaart welke tenminste een resolutie van 640x480 pixels kan weergeven.
  • Optische diskdrive of USB media.
  • ≥3 GB HDD space, ≥10 GB aanbevolen.

Applicatie's en Tools

Op de aptosid-ISO zit een variteit aan verschillende programma's en tools waarmee je bijna alles voor het dagelijks gebruik nodig hebt.

Keuze uit 2 grote Desktop Managers om de programmas/applicaties te benaderen:
KDE 4.3.x (en + de), een top desktop manager
Xfce4 4.6.1.x (en + de), een middel gewicht desktop manager.

Fluxbox 1.1.1-x (en + de) is included on both variations as a lightweight desktop manager option.

Voor surfen op Internet, afhankelijk van de release variatie, is er Konqueror en iceweasel.

Office applicatie's (ook afhaneklijk van de variatie en om een paar te noemen), varieren van Koffice, Openoffice.org, Abiword, Gnumeric tot file managers zoals Dolphin en Thunar.

Om netwerk en internet connecties te configureren is er Ceni en voor WIFI zie onze WIFI Roaming documentatie.

Non-free driver informatie is hier.

Voor het partitioneren van de harddisk leveren we cfdisk en GParted gparted heeft een ntfs resize optie.

Diagnosche tools zoals Memtest86+, een geavanceerde geheugen diagnose tool.

All ISO variations contain a full set of command line interface tools. Een complete manifest van elke release variant is te vinden op elke download mirror: aptosid Mirrors, Downloaden en branden.
