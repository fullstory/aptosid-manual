Cheat codes

De mogelijke opties in de "Waarde" kolom van de tabel, moeten met de "cheatcode" door een = teken gekoppeld worden. Voorbeeld: als 1280x1024 de gekozen waarde is voor de beeldscherm cheatcode, dan moet ingevuld worden screen=1280x1024 bij de grub boot prompt, voor xmodules xmodule=vesa, of voor acpi, acpi=off.

Cheatcode Waarde Omschrijving
acpi off ACPI volledig uit
oldboot benodigde ACPI onderdelen om het boot actief te houden, de anderen zijn uit
force forceer ACPI op aan
apm off power-off
desktop kde Kies je desktop
fluxbox
gnome
icewm
xfce
dpi auto or XX zet het aantal benodigde Dots Per Inch voor je beeldscherm (bv. mijn 15" 1600x1200 laptop is 1600/(15*(4/5)) = 133 so dpi=133
failsafe minimale auto detectie
fromiso some/path/sidux.iso zoals fromhd, maar gebruikt dan de sidux iso image. Deze optie hoeft ook geen cd om ervanaf te booten, heeft alleen een correcte kernel en initrd/minirt nodig
hostname whatever-your-host-name-is hostnaam, verander de hostnaam van de live-cd systeem
hsync 80 (zB) horizontale monitor refresh rate (kHz)
irqpoll gebruik IRQ poll
md5sum test integriteit van de media
mem 128MB (zB) hoeveel geheugen er gebruikt moet worden
noacpi acpi uitschakelen
noagp agp uitschakelen
noapic apic uitschakelen
noddc uitschakelen van het automatisch Direct Digital Calibration (probeert je monitor automatisch te configureren)
nodhcp uitschakelen van het automatisch gebruik van Dynamic Host Configuration Protocol (probeert je ethernet verbinding automatisch te configureren).
nodma schakelt Direct Memory Access voor je disken uit
noeject Geef geen melding dat de CD ge-eject moet worden bij een shutdown
nofloppy floppy uitschakelen
nointro de index.html komt niet tevoorschijn na het starten van de live desktop
noisapnpbios uitschakelen van ISA Plug'n Play
nomce geen bericht van onbelangrijke foutmeldingen
noml verwerp alle geleverde modelines. vraag beeldscherm om de instellingen
nocpufreq geen activatie van Speedstep/Powernow
nosmp schakel Symetric Multi Processor support uit
notouchpad schakel touchpad support uit
nowheel forceer het PS/2 protocol voor PS/2 muis
pci noacpi gebruik geen ACPI voor PCI interrupts
quiet geef niet alles op het console weer
screen 800x600 zet je schermresolutie, 1024x768, 1600x1200 enzovoort
toram copieer de CD naar Geheugen en start vanaf hier op
vga normal kijk hier voor vga codes: VGA Cheatcodes
vsync 60 verticale monitor refresh rate (Hz)
xmodule ati gebruik aangegeven X-module
fbdev gebruik aangegeven X-module
i810 gebruik aangegeven X-module
mga gebruik aangegeven X-module
nv gebruik aangegevenX-module
radeon gebruik aangegeven X-module
savage gebruik aangegeven X-module
vesa gebruik aangegeven X-module
xhrefresh 80 horizontale monitor refresh rate (kHz)
xvrefresh 60 verticale monitor refresh rate
