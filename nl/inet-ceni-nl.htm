Online komen / hoe configureer ik het netwerk met Ceni

Ceni is geen service. Het is een configuratie toepassing dat stopt wanneer het geconfigureerd is, en gebruikt daardoor geen systeem resources.

Many wifi cards are deemed as non-free in Debian and cannot be shipped on an aptosid-iso*. You will most likely need non-free firmware to be available on a USB-stick to install on the operating system. Please refer to non-free firmware debs on a stick.

Indien je een DHCP server in je LAN hebt en je computer hiermee verbonden is, dan zal de netwerk instellingen tijdens het opstarten van de computer automatisch geconfigureerd moeten worden is dit niet het geval, start dan: Ceni op door op Kmenu>aptosid>Internet>Ceni te klikken. Dit zal een terminal/konsole openen , waar naar het je om de root wachtwoord vraagt, ( op de live-Cd is voor root geen wachtwoord ingesteld , dus gewoon enter drukken).

De snellste manier om Ceni te starten is via het openen van een terminal/konsole en dan het intikken van 

ceni

hierna wordt je om het root wachtwoord gevraagd.

Verbinden met een 56k Dial-up Modem

KDE heeft een frontend voor dial up modems genaamd KPPP Internet Dial-up Tool, terug te vinden in het hoofd menu onder internet.

De toepassing heeft een interne help manual en geeft een volledige beschrijving hoe je de modem moet instellen om online te gaan.

Firewalls

Een firewall is in principe niet nodig als je verbinding via een router wordt gemaakt, deze router dient dan wel goed geconfigureerd te worden. Maar een firewall heeft een belangrijke rol voor de veiligheid als de internetverbinding per ADSL-usb-modem of via een dial-up modem tot stand komt.

ferm

ferm is een terminal toepassing om ingewikkelde firewalls te beheren, zonder het veelvuldig herschrijven van ingewikkelde regels. ferm laat toe om de volledige firewall regelset op te slaan in een apart bestand, en deze te laden met Ã©Ã©n commando. De firewall configuratie lijkt op een gestructureerde programmeertaal, dewelke niveau's en lijsten kan bevatten.

apt-get install ferm

ferm manual :: Opmerking: ferm zal je niet leren hoe je een firewall moet opzetten, noch hoe je goede regels moet schrijven.

gufw

Er is een GUI toepassing gebasseerd op de backend toepassing ufw.

apt-get install gufw

De Debian beschrijving is :: gufw is een makelijke en intuitieve methode om je Linux firewall te beheren. Het ondersteunt taken zoals het blokkeren of toelaten van voorgeconfigureerde poorten, de gewone p2p of individuel poorten. Het ondersteunt ook IPv6 en vele andere opties.
