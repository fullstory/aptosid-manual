Mysql Setup

Voorafgaand moet gecontroleerd worden of de volgende regels niet √∫itgeschakeld'zijn in het bestand apache2.conf (of in .d/php.ini):

# extension=php_mysql.so

wordt dus

extension=php_mysql.so

Het configuratie-bestand van mysql is te vinden in: /etc/mysql/my.cnf

Standaard wordt voor mysql de user root gebruikt en is geen password nodig. Het root password kan veranderd worden door:

mysql> USE mysql;
mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE user='root';
mysql> FLUSH PRIVILEGES;
Hoe kan een user worden aangemaakt

Het is niet verstandig om een root password voor mysql te gebruiken. Het is dus zinvol om een user aan te maken om de mysql database te bnaderen via een PHP script. Users kunnen voor de mysql database ook aangemaakt worden door programma als of phpMyAdmin te gebruiken om rechten voor users vast te leggen.

Om de web interface te benaderen met de browser wordt in de adresregel getypt:

http://ipaddress/phpmyadmin/

Zodra gevraagd wordt na de username en het password benader je de database als root en gebruik het root-password (standaard is geen password voor root vastgelegd)
