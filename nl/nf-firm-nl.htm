Sources list en firmware drivers: "non-free"

Sources list

Hoewel aptosid alleen dfsg free software bevat, wil je misschien desondanks non-free software toevoegen aan je /etc/apt/sources.list.d/debian.list:

mcedit /etc/apt/sources.list.d/debian.list

De lijnen zouden op iets onderstands moeten lijken, afhankelijk van de gekozen mirror-site (refereer naar Lijst van Debian Servers en mirrors met de huidige status):

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Na het wijzigen van je bronnen moet je een apt-get update uitvoeren voordat je gaat zoeken of nieuwe pakketten wilt installeren, ( apt-get update is een goede gewoonte voordat je een nieuwe pakket gaat installeren).

Firmware detectie - non-free

Om non-free firmware te installeren, (veronderstelt dat contrib non-free werd toegevoegd aan de sources):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

De benodigde firmware kan met de onderstaande device / firmware lijst verkregen worden of met het fw-detect script (onderdeel van aptosid-scripts).

$ fw-detect

De uitvoer van fw-detect beschrijft de commando's die nodig zijn om ze te installeren en te activeren v/d firmware:

Voorbeeld:

#Detected driver that requires firmware to operate
#Follow these instructions to obtain the correct firmware
# and activate the zd1211rw driver:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Om firmware te installeren vanuit de git repos, moet je het volgende doen:

apt-get install git-core

niet-gratis firmware debs op een stick

Indien je firmware .debs gedownload hebt, sla deze dan op op een usb-stick om zo de bestanden over te halen naar een andere computer, je kunt ze download als zip of tar.gz bestand van http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/ en pak ze uit in een folder genaamd firmware, je zou nu een aantal .deb bestanden moeten zien.

Download nu firmware-linux-free*.deb en voeg deze toe aan de gedownloade en uitgepakte firmware bestanden. Kopieer deze nu middels de usb stick naar de computer waar je de firmware nodig hebt en:

dpkg -i firmware-linux-nonfree*.deb

Wij onderzoeken nog of wij sommige drivers kunnen intergeren in aptosid. Helaas is de wettelijke status van deze drivers nogal ingewikkeld.
