sidux Control Center

Het sidux control center is een frontend om sidux aan te passen. Het is als een module in het Control Center geplaatst en is beschikbaar in Kroatisch, Deens, Nederlands, Engels, Frans, Duits, Italiaans, Japans, Pools, Portuguees, Roemeens, Russisch, Spaans en Grieks. Het is beschikbaar in KDE en GTK versies.

siduxcc bevat 7 modules met de onderstaande functies:

1. Beeldscherm module
  • aanpassen van resolutie en scannen van de frequentie
  • vastleggen van kleurdiepte en DPI
  • configureren van de X-Server
  • installeren van drivers voor de grafische kaart
2. Internet module
  • legt de standaard sidux browser vast
3. Kernel module
  • installatie van een nieuwe kernel
  • verwijderen van oude kernels
  • installatie van kernel modules
4. Netwerk module
  • configureren van netwerkkaarten
  • invoeren v/d hostname
  • invoeren v/d nameservers
  • installatie Wlan-Cards met Ndiswrapper
  • installatie Wlan-Cards met fwdetect
5. Diensten/services module
  • instellingen voor algemene services (Start, Stop, Restart, Reload, add/remove Runlevel entries) (Apache, Cupsys, MySQL, Samba, SSH)
  • starten van diensten bij het booten van sidux
6. Infomatie module
  • toont informatie over je systeem

konsole versie

Start siduxcc als root in de konsole.

sidux hermes

sidux-hermes (dist-upgrade bewaker) is een kde-tray applicatie waarmee waarschuwingen getoond worden als er problemen te verwachten zijn bij een dist-upgrade. Hiermee wordt de nieuwssectie van sidux.com gecontroleerd op het trefwoord 'Warning' en als er een waarschuwing is, verschijnt een rood stoplicht naast je klok. Er is ook een gtk versie van hermes.

sidux-hermes-qt4

Het wordt aanbevolen de qt4 versie te installeren. Deze versie heeft de meeste functies en werkt ook met alle desktops (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 heeft opties zoals de kernel informatie en de mogelijkheid om bestanden te downloaden die later geïnstalleerd kunnen worden bij een dist-upgrade in runlevel 3. 

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Gebruik de gtk2 versie als je een programma wilt zonder qt afhankelijkheden.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes bevat de backend voor alle hermes versies. Het wordt automatisch geinstalleerd zowel voor de qt4- als voor de gtk2-versie.

apt-get install sidux-hermes
