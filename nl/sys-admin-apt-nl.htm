Een klein APT-kookboek

Wat betekent APT?

APT is een afkorting voor Advanced Packaging Tool en is dus een verzameling van programma's en scripts, die (de systeembeheerder) root bij de installatie en het beheer van Deb-paketten ondersteunen.

Het "APT"-systeem heeft als basis hiervoor een bestand nodig /etc/apt/sources.list. Hierin staat informatie over de bronnen waaruit de paketten gehaald moeten worden. Hieronder wordt voor 1 bestand de sidux installatie uitgelegd.

NB: In het voorbeeld wordt de Duitse-mirrorserver beginnend met ftp.de gebruikt. Deze instelling kan als root veranderd worden, door de landen-code aan te passen zoals ftp.us, ftp.uk. In de meeste landen zijn locale Debian-mirrors beschikbaar. Hierdoor kan de gebruiker sneller downloaden en wordt tevens bandbreedte bespaard.

# See sources.list(5) for more information, especialy
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.

deb http://ftp.debian.org/debian/ sid main
#deb-src http://ftp.debian.org/debian/ sid main

#deb http://ftp.debian.org/debian/ testing main
#deb-src http://ftp.debian.org/debian/ testing main

#deb http://ftp.debian.org/debian/ experimental main
#deb-src http://ftp.debian.org/debian/ experimental main

deb http://sidux.com/debian/ sid main fix.main
#deb-src http://sidux.com/debian/ sid main fix.main

Om te kunnen beoordelen of recentere bestanden/paketten beschikbaar zijn, wordt de eigen installatie vergeleken met de recente lijst van de bestanden op de Debian-server. Het programma apt-get gebruikt deze informatie, om onderlinge verbanden/afhankelijkheden op te lossen en op deze manier te garanderen, dat de gekozen paketten werken. De actualisatie van deze datenbase wordt gedaan via het commando apt-get update.

# apt-get update
Get:1 http://sidux.com sid Release.gpg [189B]
Get:2 http://sidux.com sid Release.gpg [189B]
Get:3 http://sidux.com sid Release.gpg [189B]
Get:4 http://ftp.de.debian.org unstable Release.gpg [189B]
Get:5 http://sidux.com sid Release [34.1kB]
Get:6 http://ftp.de.debian.org unstable Release [79.6kB]

Hoe kan ik een nieuw paket installeren?

Als de APT-databank geactualiseerd is en de naam van het paket bekend is, dan is onderstaand commando voldoende (later wordt getoond, hoe een paket te vinden is):

# apt-get install qemu
Reading package lists... Done
Building dependency tree... Done
The following extra packages will be installed:
  bochsbios openhackware proll vgabios
Recommended packages:
  debootstrap vde
The following NEW packages will be installed:
  bochsbios openhackware proll qemu vgabios
0 upgraded, 5 newly installed, 0 to remove and 20 not upgraded.
Need to get 4152kB of archives.
After unpacking 11.9MB of additional disk space will be used.
Do you want to continue [Y/n]? n

Verwijderen van een paket:

De manier van werken komt grotendeels overeen met de installatie van een programmapaket. Verder is van belang dat bepaalde programmapaketten, die een onderling verband (afhankelijkheden) hebben, niet gewist worden. Dat wordt later bij deb-orphan uitgelegd.

# apt-get remove gaim
Reading package lists... Done
Building dependency tree... Done
The following packages will be REMOVED:
  gaim gaim-irchelper
0 upgraded, 0 newly installed, 2 to remove and 310 not upgraded.
Need to get 0B of archives.
After unpacking 4743kB disk space will be freed.
Do you want to continue [Y/n]?
(Reading database ... 174409 files and directories currently installed.)
Removing gaim-irchelper ...
Removing gaim ...

In deze procedure blijven de configuratiebestanden staan. Deze bestanden kunnen zo later bij een herinstallatie van het programmapaket (in dit voorbeeld gaim) hergebruikt worden. Het kan zinvol zijn om de configuratiebestanden ook te verwijderen (bijv. als niet duidelijk is of het 'niet werken' aan de programma-versie of aan de configuratie ligt). Er wordt dan het volgende commando gebruikt:

# apt-get remove --purge gaim
Reading package lists... Done
Building dependency tree... Done
The following packages will be REMOVED:
  gaim*
0 upgraded, 0 newly installed, 1 to remove and 309 not upgraded.
Need to get 0B of archives.
After unpacking 4678kB disk space will be freed.
Do you want to continue [Y/n]? Y
(Reading database ... 174405 files and directories currently installed.)
Removing gaim ...
Purging configuration files for gaim ...

In het voorbeeld werden de configuratiebestanden met het programmapaket gaim gewist, maar niet de programmapaketten, van welke gaim afhankelijk was (er zijn paketten die door meerdere applicaties gebruikt worden - die moeten dus ook blijven staan).

De update van een enkel paket (hier bijv. gcc-4.0) kan correct worden doorgevoerd incl. de controle van de onderlinge afhankelijkheden:

# apt-get install gcc-4.0
Reading package lists... Done
Building dependency tree... Done
gcc-4.0 is already the newest version.
0 upgraded, 0 newly installed, 0 to remove and xxx not upgraded

Upgrade/bijwerken van het geïnstalleerde systeem:

Als het complete systeem bijgewerkt moet worden, dan gebeurt dat via "dist-upgrade" . Hierbij moet worden opgelet, met welke programmapaketten een upgrade uitgevoerd wordt.

Voor upgrades moet standaard de grafische omgeving worden verlaten. De upgrade wordt in een tekstkonsole uitgevoerd. Het is echter mogelijk, de paketten eerst te downloaden en later pas in de tekstkonsole te wisselen). Door deze kleine maatregelen kan men zich soms uren aan reparaties besparen.

(Alleen) downloaden:

# apt-get dist-upgrade -d

Deze optie is weinig bekend, maar het grote voordel hiervan is, dat de procedure gestopt en later voortgezet kan worden:

apt-get dist-upgrade -d
xxx upgraded, 30 newly installed, 25 to remove and 2 not upgraded.
Need to get 342MB of archives.
After unpacking 3916kB of additional disk space will be used.
Do you want to continue [Y/n]?
Y
(Y will download the upgraded packages without touching the installed system)

Na "dist-upgrade -d" worden de onderstaande stappen uitgevoerd.

Downloaden en installeren:

A.u.b. NOOIT "apt-get dist-upgrade" of "apt-get upgrade" met een actieve grafische omgeving X uitvoeren.

Voordat een upgrade wordt uitgevoerd, wordt aanbevolen de sidux-Homepage te bezoeken. Hier staan misschien de recente upgrade-waarschuwingen. Deze waarschuwingen zijn door het systeem Debian sid/unstable noodzakelijk. In de bestandenlijsten worden namelijk dagelijks nieuwe programmapaketten opgenomen.

Volgorde voor de upgrades: afmelden uit de grafische omgeving zoals de KDE desktop, met Ctrl+Alt+F1 in de tekstmodus wisselen; aanmelden als root, vervolgens "init 3" uitvoeren (zonder aanhalingstekens). Hierdoor komen wij in een omgeving zonder toeters en bellen. De volgende opdrachten zijn: apt-get update; apt-get dist-upgrade; fix-fonts. Verder kunnen de scripts voor de installatie van 3D-videokaartdrivers uitgevoerd worden (na een update van xorg en indien deze drivers gebruikt worden, is een herinstallatie van de 3D-videokaartdrivers verstandig). Na de upgrades wordt de opdracht "init 5 && exit" gegeven.

Voor de zekerheid nogmaals onze waarschuwing: a.u.b. NOOIT een dist-upgrade of upgrade met een grafisch programma zoals adept of kpackage uitvoeren!

Met apt-cache naar programmapaketten zoeken

Een zeer zinvol commando in het APT-systeem is apt-cache. Hiermee worden de APT-databank doorzocht en informatie over de paketten opgevraagd. De lijst van alle paketten, die een "screensaver" bevatten of gebruiken, kan door het volgende commando worden verkregen:

# apt-cache search screensaver
brightside - Add reactivity to the corners and edges of your GNOME desktop

electricsheep - screensaver showing collective dream of sleeping computers
epplets - The Epplets for the Enlightenment Window Manager
gkrellshoot - Plugin for gkrellm to lock the screen and make screenshots
kscreensaver - additional screen savers released with KDE
kscreensaver-xsavers - KDE hooks for standard xscreensavers
ktux - Tux screensaver for KDE
lg-issue24 - Issue 24 of the Linux Gazette.
lockvc - screensaver to lock your Linux console(s)
rss-glx - Really Slick Screensavers GLX Port
shermans-aquarium - Sherman's aquarium applet for GNOME 2
wmmatrix - View The Matrix in a Window Maker dock application
xscreensaver - Automatic screensaver for X
xscreensaver-gl - GL(Mesa) screen hacks for xscreensaver
ksetisaver - SETI screensaver for KDE
