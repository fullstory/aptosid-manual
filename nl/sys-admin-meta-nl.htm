Meta-pakketten

Tijdens de installatie-procedure, kun je ook kiezen voor de zogenaamde meta-pakketten van verscheidene toepassingen en games om die bij je andere programma's te voegen.

Deze zogenaamde 'meta-packages installer' is ook beschikbaar op je gewone geinstalleerde systeem door naar: Kmenu>sidux>sidux-meta-installer te gaan.

De lijst bevat tot nog toe:

Scan partitie en herstel tools
Games
Multimedia -VDR
OpenOffice.org- (Lite ISO)
Koffice - Russisch
Russische 'lang' pakketten voor OpenOffice.org en Iceweasel
Algemene server- (de LAMP stack)
Xfce4 - (lichtgewicht desktop)
Fluxbox - (lichtgewicht desktop)
fvwm-crystal - (lichtgewicht desktop)

Opmerking: Als de installer eenmaal loopt, zul je de volgende regel zien:

paste your code here: (niet per se voor een nomale installatie, alleen preconf)

Het enige dat je hoeft te doen, is enter in te drukken.

De lijst wordt regelmatig bijgewerkt (controleer de meta-installer in het sidux menu, de lijst is correct vanaf 07-03-07

Meta-Installer and Proxies

install meta must run as root and if the settings are not in the apt.conf file they can not be read. There are 2 methods to enable the use of a proxy, and of the two, Method 1 is better, as it is permanent:

Method 1

Define the proxy in the apt.conf file, edit as root with kwrite /etc/apt/apt.conf and add in the following:

Acquire{
HTTP::proxy "http://192.168.1.1:3128";
FTP::proxy "http://192.168.1.1:3128";
}

Method 2

Open a terminal use sux to become root to set the http_proxy and ftp_proxy environment variables and then call the install meta. You will need to do this each time . you want to access the meta-installer:

export http:proxy=http://192.168.1.1:3128"
export ftp_proxy="ftp://192.168.1.1:3128/"
