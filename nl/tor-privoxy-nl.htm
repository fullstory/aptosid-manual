Privoxy

Privoxy is een web proxy met geavanceerde filter technieken voor het beschremen van je privacy, wijzigen van web pagina data, beheren van cookies, toegangscontrole, en het verwijderen van advertenties, banners, pop-ups en andere onaangename Internet junk. Privoxy heeft een erg flexibele configuratie en kan naar je eigen wensen aangepast worden. Privoxy heeft een applicatie voor zowel stand-alone systemen als multi-user networken. Privoxy is gebaseerd op Internet Junkbuster (tm).

Installatie van privoxy:

apt-get update
apt-get install privoxy

Een standaard installatie zou een redelijk uitgangspunt voor de meesten moeten verstrekken. Er zal ongetwijfeld gelegenheden zijn waar u de configuratie zult willen aanpassen, maar dat kan als de behoefte zich voordoet worden behandeld. In de meeste gevallen behoeft de standaard configuratie niet of nauwelijks te worden aangepast.

Gebruik maken van privoxy en de sidux meta-installer leest u dit dan even door.

In de privoxy configuratie moet je misschien regels activeren (uncomment, oftewel # hekje weg). Dat is afhankelijk van je voorkeuren en verschilt per user.

Tor

Tor is een netwerk van virtuele tunnels welke het personen of groepen van personen toestaat om hun privacy en veiligheid op het internet te verbeteren. Het staat software ontwikkelaars ook toe om nieuwe communicatie tools te creeeren met ingebouwde privacy mogelijkheden. Tor levert de basis voor een hele range applicaties welke het toestaat om informatie van organistaties of individuelen te delen over publieke netwerken zonder aan veiligheid/privacy in te boeten.

Je moet je wel beseffen je internet snelheid door TOR negatief beinvloed wordt. Maar wel je privacy gewaarborgd wordt.

Installeren van Tor:

apt-get update
apt-get install tor

The default torrc file should work out of the box for most Tor users. You will need to configure Privoxy to use Tor on the internet, see: Tor and internet browsing.

You should also be aware that internet throughput speeds will be negatively affected.

For the Iceweasel brower there is a Torbutton add-on available.

The Tor documentation offers comprehensive on all aspects of Tor.

Vidalia is a controller GUI for the Tor software and allows you to start and stop Tor, view the status of Tor at a glance,and monitor Tor's bandwidth usage.

apt-get update
apt-get install vidalia
