Conteúdo do LiveISO

O aptosid fornece somente software Debian dfsg-free na ISO Live. Favor clicar neste link para maiores informações sobre fontes não-livres de firmware.

A imagem ISO é totalmente baseada no Debian Sid (atualizado até 2012-12-01), enriquecida e estabilizada por pacotes e scripts do próprio aptosid. Ela vem com a imagem mais recente e ajustada do kernel, baseada no mais recente vanilla kernel 3.6.9-rc1. ACPI e DMA são ativados por padrão.

Uma listagem completa dos pacotes ('manifest') de cada variante pode ser encontrada nos espelhos: Espelhos do aptosid, Baixar e Queimar.

Variedades das ISOs do aptosid

O aptosid oferece 5 variantes atualizadas de Live-ISOs contendo o Debian sid; a instalação, normalmente, não leva mais que 10-20 minutos.

Essas variedades são:
  1. KDE lite 32 bit , live-ISO com aproximadamente 600MB:
    contém uma seleção mínima de aplicações do KDE4. Adicionar mais programas para atender suas necessidades é facilmente feito com o apt.
  2. KDE lite 64 bit , live-ISO com aproximadamente 600MB:
    contém uma seleção mínima de aplicações do KDE4. Adicionar mais programas para atender suas necessidades é facilmente feito com o apt.
  3. XFCE 32 bit, live-ISO com aproximadamente 500MB:
    contém um ambiente desktop completo, com todas as aplicações que você precisa para começar a produzir imediatamente. É menos pesado que o KDE4.
  4. XFCE 64 bit live-ISO com aproximadamente 500MB:
    contém um ambiente desktop completo, com todas as aplicações que você precisa para começar a produzir imediatamente. É menos pesado que o KDE4.
  5. KDE-full, uma mistura de 32 bit com amd64 bit, live-ISO com aproximadamente 2.5GB:
    contém uma suite completa de aplicações do KDE4, com suporte a 16 línguas. Inclui, entre outras coisas, suites de escritório como o LibreOffice e navegadores de Internet, como o Iceweasel (Firefox)

Aplicações e ferramentas

Requisitos mínimos para rodar o KDE4 e o XFCE

AMD64

  • Requisitos de CPU:
    • AMD64
    • Intel Core2
    • Intel Atom Series (2xx/ 3xx, N4xx/ D4xx/ D5xx; chips da série N2xx não proporcionam EM64T)
    • VIA Nano
    • qualquer CPU x86-64/ EM64T ou mais recente
    • CPUs mais recentes, com capacidade de processamento de 64bit, AMD Sempron e Intel Pentium 4 (observe a 'flag' "lm" em /proc/cpuinfo ou use infobash -v3)
  • Requisitos de memória RAM:
    • KDE: ≥512MB RAM (≥1 GBRAM recomendados), ≥1GB RAM para liveapt.
    • XFCE: ≥512MB RAM.
  • placa de vídeo VGA com resolução mínima de 640x480 pixels.
  • drive ótico (leitor/gravador de CD ou DVD) ou mídia USB.
  • ≥espaço livre de 3GB no HD, ≥10GB recomendados.
  • Métodos de boot suportados:
    • Tabela de partições PC-BIOS+MSDOS (discos duros com um tamanho máximo de 2 TB)
    • Tabela de partições PC-BIOS+GPT (discos duros com um tamanho máximo de 8 ZB, /boot/ deve estar completamente contida abaixo de 2 TB )
    • UEFI+GPT (discos duros com um tamanho máximo de 8 ZB)

i686

  • Requisitos de CPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (K5/ K6 não)
    • Intel Atom Series (Z5xx/ Z6xx, N2xx)
    • VIA C3-2 (Nehemiah, não C3 Samuel ou Ezra)/ C7
    • qualquer CPU x86-64/ EM64T ou mais recente
    • é necessário todo o conjunto de comandos i686.
  • Requisitos de memória RAM:
    • KDE: ≥512 MB RAM (≥1 GB RAM recomendados), ≥1 GB RAM para liveapt.
    • XFCE: ≥512 MB RAM.
    • Com i686, sómente 3.1 a 3.4 GB RAM são acessíveis, amd64 (x86-64/ EM64T) é recomendado em hardware adequado para 64 bit.
  • placa de vídeo VGA com resolução mínima de 640x480 pixels.
  • drive ótico (leitor/gravador de CD ou DVD) ou mídia USB.
  • ≥espaço livre de 3GB no HD, ≥10GB recomendados.
  • Supported boot methods:
    • Tabela de partições PC-BIOS+MSDOS (discos duros com um tamanho máximo de 2 TB)
    • Tabela de partições PC-BIOS+GPT (discos duros com um tamanho máximo de 8 ZB, /boot/ deve estar completamente contida abaixo de 2 TB )

Aplicações e ferramentas

A ISO do aptosid traz uma grande variedade de programas e ferramentas que cobrem virtualmente todas as necessidades do usuário desktop no seu dia-a-dia.

Choice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE4 (inglês + alemão), gerenciador topo de linha
Xfce4 (inglês + alemão), gerenciador peso médio.

O Fluxbox (inglês + alemão) está incluído em ambas as variantes, como uma opção de gerenciador de janelas peso leve.

Para navegar na Internet, dependendo de você ter baixado a ISO com KDE ou com XFCE, há: Konqueror e Iceweasel.

Aplicações para escritório (também dependendo da variedade da ISO baixada e citando apenas alguns poucos), abrangem desde LibreOffice, Abiword, Gnumeric, até gerenciadores de arquivos como Dolphin and Thunar, (para referenciar apenas um parte).

Para configurar a Rede e conexões Internet há o Ceni; já para conexões WIFI veja nossa documentação para WIFI-Roaming .

Informações para drivers não livres aqui.

Para particionar seu HD, oferecemos cfdisk e GParted. O Gparted, inclusive, tem uma opção para redimensionar partições NTFS.

Ferramentas de diagnóstico como Memtest86+, uma Ferramenta Avançada de Diagnóstico de Memória, também fazem parte da instalação.

Todas as ISOs contêm uma coleção de ferramentas para uso na linha de comando. Uma listagem completa dos pacotes ('manifest') de cada variante pode ser encontrada nos espelhos: Espelhos do aptosid, Baixar e Queimar.
