Conteúdo do LiveISO

O sidux fornece somente software Debian dfsg-free na ISO Live. Favor clicar neste link para maiores informações sobre fontes não-livres de firmware.

A imagem ISO é totalmente baseada no Debian Sid (atualizado até 2009-11-11), enriquecida e estabilizada por pacotes e scripts do próprio sidux. Ela vem com a imagem mais recente e ajustada do kernel, baseada no mais recente vanilla kernel 2.6.3X.xx. ACPI e DMA são ativados por padrão.

Uma listagem completa dos pacotes ('manifest') de cada variante pode ser encontrada nos espelhos: Espelhos do sidux, Baixar e Queimar.

Variedades das ISOs do sidux

O sidux oferece 5 variantes atualizadas de Live-ISOs contendo o Debian sid; a instalação, normalmente, não leva mais que 10-20 minutos.

Essas variedades são:
  1. KDE lite 32 bit , live-ISO com aproximadamente 600MB:
    contém uma seleção mínima de aplicações do KDE4. Adicionar mais programas para atender suas necessidades é facilmente feito com o apt.
  2. KDE lite 64 bit , live-ISO com aproximadamente 600MB:
    contém uma seleção mínima de aplicações do KDE4. Adicionar mais programas para atender suas necessidades é facilmente feito com o apt.
  3. XFCE 32 bit, live-ISO com aproximadamente 500MB:
    contém um ambiente desktop completo, com todas as aplicações que você precisa para começar a produzir imediatamente. É menos pesado que o KDE4.
  4. XFCE 64 bit live-ISO com aproximadamente 500MB:
    contém um ambiente desktop completo, com todas as aplicações que você precisa para começar a produzir imediatamente. É menos pesado que o KDE4.
  5. KDE-full, uma mistura de 32 bit com amd64 bit, live-ISO com aproximadamente 2.5GB:
    contém uma suite completa de aplicações do KDE4, com suporte a 16 línguas. Inclui, entre outras coisas, suites de escritório como o OpenOffice.org e navegadores de Internet, como o Iceweasel (Firefox)

Aplicações e ferramentas

Requisitos mínimos para rodar o KDE4 e o XFCE

AMD64

  • Requisitos de CPU:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • qualquer CPU x86-64/ EM64T ou mais recente
    • CPUs mais recentes, com capacidade de processamento de 64bit, AMD Sempron e Intel Pentium 4 (observe a 'flag' "lm" em /proc/cpuinfo ou use infobash -v3)
  • Requisitos de memória RAM:
    • KDE: ≥512MB RAM (≥768MB RAM recomendados), ≥1GB RAM para liveapt.
    • XFCE: ≥256MB RAM, ≥512MB RAM para liveapt.
  • placa de vídeo VGA com resolução mínima de 640x480 pixels.
  • drive ótico (leitor/gravador de CD ou DVD) ou mídia USB.
  • ≥espaço livre de 3GB no HD, ≥10GB recomendados.

i686

  • Requisitos de CPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (K5/ K6 não)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, não C3 Samuel ou Ezra)/ C7
    • qualquer CPU x86-64/ EM64T ou mais recente
    • é necessário todo o conjunto de comandos i686.
  • Requisitos de memória RAM:
    • KDE: ≥384MB RAM (≥768 MB RAM recomendados), ≥512MB RAM para liveapt.
    • XFCE: ≥192MB RAM, ≥512MB RAM para liveapt.
  • placa de vídeo VGA com resolução mínima de 640x480 pixels.
  • drive ótico (leitor/gravador de CD ou DVD) ou mídia USB.
  • ≥espaço livre de 3GB no HD, ≥10GB recomendados.

Aplicações e ferramentas

A ISO do sidux traz uma grande variedade de programas e ferramentas que cobrem virtualmente todas as necessidades do usuário desktop no seu dia-a-dia.

Você tem a possibilidade de escolher entre 2 grandes Gerenciadores de Desktop para acessar programas/aplicações:
KDE 4.3.x (inglês + alemão), gerenciador topo de linha
Xfce4 4.6.1.x (inglês + alemão), gerenciador peso médio.

O Fluxbox 1.1.1-x (inglês + alemão) está incluído em ambas as variantes, como uma opção de gerenciador de janelas peso leve.

Para navegar na Internet, dependendo de você ter baixado a ISO com KDE ou com XFCE, há: Konqueror e Iceweasel.

Aplicações para escritório (também dependendo da variedade da ISO baixada e citando apenas alguns poucos), abrangem desde Koffice, Openoffice.org, Abiword e Gnumeric até gerenciadores de arquivos como Dolphin e Thunar.

Para configurar a Rede e conexões Internet há o Ceni; já para conexões WIFI veja nossa documentação para WIFI-Roaming .

Informações para drivers não livres aqui.

Para particionar seu HD, oferecemos cfdisk e GParted. O Gparted, inclusive, tem uma opção para redimensionar partições NTFS.

Ferramentas de diagnóstico como Memtest86+, uma Ferramenta Avançada de Diagnóstico de Memória, também fazem parte da instalação.

Todas as ISOs contêm uma coleção de ferramentas para uso na linha de comando. Uma listagem completa dos pacotes ('manifest') de cada variante pode ser encontrada nos espelhos: Espelhos do sidux, Baixar e Queimar.
