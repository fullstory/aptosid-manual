Busca On-Line

Digite uma ou mais palavras no campo abaixo e clique Search (ou pressione Return).

Opções de busca

Se Match ALL words for selecionado, apenas aqueles documentos que contiverem todas as palavras digitadas no campo de busca serão retornados. Com Match ANY word serão mostrados todos os documentos que contiverem pelo menos uma das palavras.

Uma alternativa é colocar o sinal de mais (+) à esquerda de uma ou mais palavras para que sejam retornados apenas os arquivos que contenham todas elas. Palavras precedidas de um sinal de menos (-) fazem com que sejam mostrados apenas os aquivos que NÃO contenham nenhuma delas.

Observe que não é possível pesquisar por frases, isto é, não faz nenhum sentido colocar aspas no início e no fim das palavras que você procura "como acontece aqui".

Os resultados são ordenados por relevância, com os documentos mais relevantes listados primeiro. A relevância depende do número e da posição das palavras nos arquivos.

Busca Off-Line

O Manual do Sistema Operacional sidux criou várias configurações especiais juntamente com o Desenvolvedor do Recoll, uma ferramenta de busca pessoal para Unix/Linux. Essa ferramenta se baseia no poderoso back-end Xapian, para quem produziu uma interface gráfica em QT, cheia de opções ainda que fácil de usar.

O Recoll é chamado de Personal Search Tool ('Ferramenta de Busca Pessoal) no menu Debian do KDE.

Para configurar a busca local, é necessário seguir alguns passos:

Passo 1

O manual está disponível via apt nas seguintes línguas atualmente:
da, de, el, en, es, fr, hr, it, ja, nl, pt-br, ro, ru.

apt-get update
apt-get install sidux-manual-xx (onde xx é a língua desejada; exemplo: pt-br, para Português do Brasil))
Passo 2
apt-get install recoll
Passo 3

Baixe os arquivos específicos do recoll para o sidux como (usuário $) em uma pasta $HOME e extraia-os:

wget http://bluewater.sidux.com/outgoing/sidux-recoll-configs.tar.gz

Crie uma pasta .recoll em sua pasta Home, se ela ainda não estiver lá (devido ao ".", ela se tornará uma pasta oculta).

Coloque os seguintes arquivos do sidux-recoll-config em .recoll
mimeconf
mimemap
mimeview
recoll.conf

Para os passos 4 & 5, você precisa se tornar root para conseguir alterar as pastas /usr/share/recoll ; não se esqueça:
Passo 4

Coloque o arquivo rclsiduxman que está na pasta filters do sidux-recoll-config, em /usr/share/recoll/filters.

mv sidux-recoll-configs/filters/* /usr/share/recoll/filters
Passo 5

Coloque o arquivo sidux-book.png que está na pasta images do sidux-recoll-config, em /usr/share/recoll/images.

mv sidux-recoll-configs/images/* /usr/share/recoll/images
Passo 6

No terminal, como root:

update-menus

Como usuário no KMenu, vá em Kmenu >Debian >Aplicações > Gestão de Dados > Personal Search Tool .

Ao ser rodado pela primeira vez, o Recoll vai perguntar se você deseja montar a base de dados, clique em 'cancelar', depois File>Update Index e procure pelo tópico que você deseja abrir no manual do sidux.

Se for necessário reconfigurar o Recoll devido a atualizações do manual via apt ou smxi (no terminal, como usuário):

recollindex -z

Daí: Kmenu >Debian >Aplicações > Gestão de Dados > Personal Search Tool e em Recoll File>Update Index.

Se necessário, use o comando do Konqueror Encontrar nesta página para uma mini-procura da palavra que você pesquisou no Recoll.
