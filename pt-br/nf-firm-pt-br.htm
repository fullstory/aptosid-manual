Sources list and firmware drivers: non-free

Sources list

Como a ISO do aptosid contém apenas software livre, você provavelmente sentirá necessidade de adicionar repositórios com aplicações não livres (contrib non-free) às suas /etc/apt/sources.list.d/debian.list :

mcedit /etc/apt/sources.list.d/debian.list

As linhas que você inserir devem se parecer com estas, dependendo dos espelhos ('mirrors') que você escolheu (refer to http://www.debian.org/mirrors/">Listagem dos servidores Debian e status atual dos espelhos ('mirrors')):

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Após alterar sua sources. list, é necessário que você atualize a base com um apt-get update antes de procurar ou instalar novos pacotes. (o 'apt-get update' deve sempre ser usado antes de instalar qualquer coisa).

Firmware detection - non-free

Para instalar firmware non-free firmware, (é supost que contrib non-free foi adicionado às suas sources):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

O firmware necessário pode ser determinado pelo script recém-disponibilizado fw-detect (empacotado em aptosid-scripts).

$ fw-detect

A saída de fw-detect detalha os comandos necessários para instalar e ativar o firmware:

Exemplo:

#Detected driver that requires firmware to operate
#Follow these instructions to obtain the correct firmware
# and activate the zd1211rw driver:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Para instalar firmare pelos repositórios git, você precisa de:

apt-get install git-core

pacotes .deb com firmware não-livre para pendrives

Se você tiver de preparar pacotes .deb de firmware e transferi-los para outro computador usando uma pendrive ou algum outro dispositivo USB, você pode baixá-los tanto como arquivos .zip quanto como arquivos tar.gz em http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/. Depois, extraia-os para uma pasta com o nome de firmware (devem aparecer vários arquivos .deb).

A seguir, baixe firmware-linux-free*.deb e adicione-o a todos os outros debs na pasta. Daí, transfira tudo para a pendrive e desta para o computador que precisa do firmware não livre e:

dpkg -i firmware-linux-nonfree*.deb

Tentamos oferecer pacotes para firmware legalmente redistribuível em nossos repositórios 'non-free', mas nem todo fabricante permite isso.
