Particionando seu HD com o GParted

AVISO: a respeito de nomes de discos veja UUID, Nomeando Partições e fstab; por padrão, o sidux agora usa UUID

Ferramentas de partição pedem senha de administrador ('root'). Como nenhuma foi criada, basta pressionar [ENTER]. Veja: Modo Live

O Básico

Uma partição precisa ter um sistema de arquivos. O Linux reconhece vários tipos, como ReiserFS, Ext3 e, para usuários experientes, XFS e JFS. O Ext2 é ótimo para armazenamento, pois existe disponível um driver Windows para troca de dados.

Para uso normal, recomendamos o sistema de arquivos ext3 que, além de ser o padrão do sidux, também conta com uma boa manutenção.

Esta versão do GParted é a 0.3.1.1. As informações a seguir foram retiradas da documentação oficial do GParted e adaptadas para ser um guia de utilização no sidux. Portanto, não devem ser vistas nem usadas como um guia definitivo. E&OE

A documentação completa pode ser encontrada aqui: GParted

Usando o GParted

Criar e gerenciar partições não são tarefas que executamos todos os dias. Portanto, o ideal é ler este guia uma vez, para se familiarizar com os diversos conceitos e com algumas das questões que surgirão.

  • Informações Gerais

    Quando o GParted é executado, ele abre uma janela e verifica seus drives.

    Ao clicar no menu GParted, no canto superior esquerdo, você é apresentado a várias opções; 'Refresh devices' permite a atualização dos drives de sua máquina; 'Devices' permite que você escolha o HD cujas partições você deseja modificar. Útil, caso você possua mais de um drive; 'Show features' é usada para se obter mais informações.

  • Edit

    Abre uma nova janela, onde podem ser vistos os sistemas de arquivos suportados e algumas opções para editar partições.

    Esse menu é o segundo da esquerda para a direita. Ele mostra duas funções desabilitadas, muito úteis, por sinal: 'Undo' (Desfazer) e 'Apply' (Aplicar), além de permitir que você acesse/veja outras áreas.

  • View

    • Device Information (Informação sobre o Dispositivo)

      O menu Harddisk Information mostra detalhes sobre o HD, como Modelo, Tamanho etc. Esse menu é mais útil quando existe mais de um drive no sistema, pois pode-se fazer uso das informações mostradas para certificar-se de que o drive examinado é realmente aquele desejado.

    • Pending Operations (Operações Pendentes)

      Na janela inferior fica a lista das operações pendentes. As informações são úteis, já que indicam as modificações que pretendemos fazer no(s) drive(s).

  • Device (Dispositivo)

    'Device' permite que você rotule um disco (Set a Disk Label), se o rótulo atual não for apropriado.

  • Partition (Partição)

    Esse é o menu mais importante. Ele permite que você faça várias operações, algumas delas perigosas.

    Delete (Apagar) é usada para apagar uma partição previamente selecionada.

    Resize/Move (Redimensionar/Mover) é uma função útil, auto-explicativa.

  • Criando uma nova partição

    Na barra de ferramentas, o botão New permite-lhe criar uma nova partição, desde que você já tenha selecionado uma área não-alocada. Na nova janela que aparece, você pode escolher o tamanho, o sistema de arquivos e se ela será do tipo Primária, Estendida ou Lógica.

  • Se você cometer um engano

    Se você errar alguma coisa, use o botão Delete para apagar a partição e recomeçar; caso a operação ainda esteja pendente, use o botão Undo.

  • Redimensionar/Mover

    Se você quiser redimensionar uma partição pré-selecionada, clique no botão Resize/Move: surge uma nova janela. Use o mouse ou as setas direcionais para reduzir ou aumentar a partição.

    Depois do comando Resize, você precisa clicar em Apply para que a alteração seja escrita no HD.

    Quanto tempo vai demorar depende do novo tamanho da partição.

    Depois de realizadas todas as alterações, reinicie o sidux para que ele possa ler a nova tabela de partições.

Redimensionando Partições NTFS

Redimensionar uma partição NTFS exige que o sistema seja reiniciado! NÃO FAÇA nenhuma outra operação na partição antes disso, senão você ficará com vários erros.

  • Quando você der o boot pelo Windows, o sistema irá mostrar uma tela especial e uma mensagem sobre consistência do HD: Checando sistema de arquivos em C:
  • Deixe o AutoCheck (AUTOCHK) rodar: NT precisa verificar seu sistema de arquivos após a operação de redimensionamento.
  • No final do processo, o computador será reiniciado automaticamente pela segunda vez. Isso assegura que tudo funcionará perfeitamente.
  • Depois de recarregado, seu Windows estará ok, mas você tem de aguardar até a tela de login!
