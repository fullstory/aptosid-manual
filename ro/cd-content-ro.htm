Conţinutul Live ISO-ului

sidux furnizează doar Debian dfsg-free software pe live-ISO, vă rugăm să verificaţi acest link pentru informaţii suplimentare despre surse non-free pentru firmware .

Imaginea ISO este complet bazată pe Debian Sid , îmbunătăţită şi stabilizată cu scripturi şi pachete concepute exclusiv de echipa sidux . Vine cu ultimul și cel mai bun kernel sidux, bazat pe cel mai recent vanilla kernel 2.6.3X.xx. ACPI şi DMA sunt activate implicit.

O descriere completă a fiecărei versiunî (manifest) este inclusă în fiecare locație de descărcare: Locații sidux, Descărcare și ardere.

Variante ale versiunii ISO

sidux oferă 5 up-to-date live-ISO variante bazate pe Debian Sid a căror instalare nu durează mai mult de 10-20 minute.

Variantele sunt:
  1. KDE lite 32 bit , live-ISO de aproximativ 600MB:
    conține un set minimal de aplicații pentru KDE4 desktop. Puteți adăuga ușor mai multe aplicații conform nevoilor voastre utilizând 'apt'.
  2. KDE lite 64 bit , live-ISO de aproximativ 600MB:
    conține un set minimal de aplicații pentru KDE4 desktop. Puteți adăuga ușor mai multe aplicații conform nevoilor voastre utilizând 'apt'.
  3. XFCE 32 bit, live-ISO de aproximativ 500MB:
    conține un mediu Desktop complet (nu minimal), avănd toate aplicațiile necesare pentru a fi productiv imediat. Ocupă mai puțin loc decăt KDE4.
  4. XFCE 64 bit , live-ISO de aproximativ 500MB:
    conține un mediu Desktop complet (nu minimal), avănd toate aplicațiile necesare pentru a fi productiv imediat. Ocupă mai puțin loc decăt KDE4.
  5. KDE-full, o combinație de 32 bit + amd64 bit, live-ISO de aproximativ 2.5GB:
    conține o suită completă de aplicații KDE4 cu supot pentru 16 limbi incluzând suita office (OpenOffice.org) și web browser (Iceweasel -o versiune a Firefox)

Aplicații și unelte

Cerințe de sistem minimale pentru KDE4 și XFCE

AMD64

  • Procesor CPU:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • orice procesor x86-64/ EM64T sau mai nou
    • noile procesoare pe 64 bit (AMD Sempron și Intel Pentium 4) (căutați specificația "lm" în /proc/cpuinfo sau utilizați 'infobash -v3').
  • Memorie RAM :
    • KDE: ≥512 MB RAM (≥768 MB RAM recommended), ≥1 GB RAM for liveapt.
    • XFCE: ≥256 MB RAM, ≥512 MB RAM for liveapt.
  • Plăci grafice VGA cu rezoluție de cel puțin 640x480 pixeli.
  • Unitate optică (CD/DVD) sau un mediu USB.
  • ≥minim 3 GB spațiu pe disc, ≥10+ GB recomandat.

i686

  • Procesor (CPU):
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (nu K5/ K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, nu C3 Samuel sau Ezra)/ C7
    • orice procesor x86-64/ EM64T sau mai nou
    • setul complet de comenzi i686 .
  • Memorie RAM :
    • KDE: ≥384 MB RAM (≥768 MB RAM recomandat), ≥512 MB RAM for liveapt.
    • XFCE: ≥192 MB RAM, ≥512 MB RAM for liveapt.
  • Placă grafică VGA cu rezoluție de cel puțin 640x480 pixeli.
  • Unitate optică (CD/DVD) sau un mediu USB.
  • ≥minim 3 GB spațiu pe disc, ≥10+ GB recomandat.

Aplicaţii şi Unelte

ISO-ul sidux conţine o diversitate de programe şi unelte care acoperă aproape orice nevoie a unui utilizator de computer în viaţa de zi cu zi.

Alegeți între 2 Desktop Managere majore pentru a accesa programe/aplicații:
KDE 4.3.x (en + de), un desktop manager de top
Xfce4 4.6.1.x (en + de), un desktop manager care cere resurse mai puține decât KDE.

Fluxbox 1.1.1-x (en + de) este inclus în ambele variante ca alternativă fiind un desktop manager ușor.

Pentru a naviga pe Internet, funcție de variantă (KDE sau XFCE, puteți folosi Konqueror și iceweasel.

Aplicațiile de Birou (Office) (din nou funcție de variantă și vom numi doar câteva), începând cu Koffice, Openoffice.org, Abiword, Gnumeric până la managere de fișiere ca Dolphin și Thunar.

Pentru a configura conexiunea la rețea (Network) și Internet sunt Ceni iar pentru WIFI vedeți documentația WIFI Roaming .

Informații despre Non-free driver este aici.

Pentru partiționarea hard disk-ului, punem la distoziție cfdisk și GParted iar gparted are și capacitatea de a redimensiona partiții ntfs.

Unelte de Diagnosticare ca Memtest86+, un puternic instrument de diagnosticare a memoriei.

Toate variantele ISO conțin un set complet de instrumente utilizabile în linie de comandă(CLI) . Un inventar complet al pachetelor (manifest) al fiecărei variante a versiunii este inclus în fiecare locație (download mirror): Conținutul ISO , Versiuni, Locații de Download și Inscripționare.
