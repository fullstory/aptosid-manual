Cheat codes

If the possible values are listed in the "Values" field of the table, they must be appended to the "cheatcode" with an = sign. For example, if 1280x1024 was the desired value for the screen cheatcode, then you would type screen=1280x1024 into the grub boot prompt, or for xmodules xmodule=vesa, or for acpi, acpi=off

Cheatcode Valoare Descriere
acpi off dezactiveaza complet ACPI
oldboot componentele ACPI necesare la boot raman active, celelalte sunt oprite
force force ACPI on
apm off power-off
desktop kde alege managerul de desktop
fluxbox
gnome
icewm
xfce
dpi auto sau XX seteaza numarul de Dots Per Inch pentru display (ex. laptopul meu de 15" la 1600x1200 este 1600/(15*(4/5)) = 133 deci dpi=133
failsafe auto detectare minimala
fromhd /dev/hda1 (hda2,...) booteaza de pe imagine a CD-ului copiata in prealabil pe partitia ... (partitia trebuie sa contina un folder sidux/sidux)
fromiso some/path/sidix.iso ca si fromhd , dar folosete o imagine sidux iso . Deasemeni nu este nevoie de un CD de pe care sa booteze , ci este necesar doar kernelul si initrd/minirt
hsync 80 (eg) rata de reimprospatare orizontala a monitorului (kHz)
irqpoll foloseste IRQ poll
mem 128MB (zB) cantitatea de ram care sa fie folosita
hostname nume computer, schimba nume computer de pe CD-ul live
noacpi dezactiveaza acpi
noagp dezactiveaza agp
noapic dezactiveaza apic
noaudio dezactiveaza audio
noddc dezactiveaza Direct Digital Calibration (incercarea de a activa automat caracteristicile monitorului)
nodhcp dezactiveaza folosirea automata a Dynamic Host Configuration Protocol (incercarea de a realiza automat conexiunea de internet).
nodma dezactiveaza Direct Memory Access pentru unitatile de disc
noeject nu cere sa faca eject de CD la oprirea sistemului
nofloppy dezactiveaza floppy
nointro sari peste afisarea paginii index.html la pornirea Live CD -ului
noisapnpbios dezactiveaza ISA Plug'n Play
nomce nu afisa mesaje pentru erori irelevante
noml renunta la toate modurile specificate prealabil. intreaba despre setarile care sa fie folosite
nocpufreq nu activa Speedstep/Powernow
nosmp dezactiveaza Symetric Multi Processor support
nosound dezactiveaza sound support
notouchpad dezactiveaza touchpad support
nowheel forteaza protocolul PS/2 pentru PS/2 mouse
pci noacpi nu folosi ACPI pentru gestionarea intreruperilor PCI
quiet nu lista totul pe consola
resume /dev/hdaX permite suspend mode
screen 800x600 seteaza rezolutia ecranului , 1024x768, 1600x1200 etc
testcd testeaza integritatea CD-ului
toram copie CD-ul in RAM si porneste de acolo
vga normal aici gasesti coduri VGA: VGA Cheatcodes
vsync 60 rata de refresh verticala (Hz)
xmodule ati foloseste modulul X specificat
fbdev foloseste modulul X specificat
i810 foloseste modulul X specificat
mga foloseste modulul X specificat
nv foloseste modulul X specificat
radeon foloseste modulul X specificat
savage foloseste modulul X specificat
vesa foloseste modulul X specificat
xhrefresh 80 rata de refresh orizontala (kHz)
xvrefresh 60 rata de refresh verticala
