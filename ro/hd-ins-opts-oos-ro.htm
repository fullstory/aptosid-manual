Instalare sidux ISO pe un dispozitiv USB stick, SSD card, shdc cu orice Sistem de Operare Linux, MS Windows™ sau Mac OS X™

N-are importanță ce SO folosești, cu următoarele metode veți putea instala un sidux ISO pe un dispozitiv USB stick, SSD card, shdc (Secure Digital High Capacity card).

Imaginea sidux ISO este scrisă pe dispozitiv dar, pentru că opțiunea 'persist' nu este posibilă, veți avea doar un 'sidux-on-a-stick'.

Dacă 'persist' este necesară, atunci vă recomandăm să utilizați 'install-usb-gui' (nu este subiectul acestui capitol) când este disponibil un sitem sidux ; pentru asta citiți Instalarea pe USB/SSD 'fromiso' - sidux-on-a-stick.

Condiții necesare:

  • Asigurați-vă că BIOS-ul PC-ului pe care vreți să-l boot-ați cu sidux-on-a-stick/card poate boot-a de pe USB stick-uri sau SSD card-uri. De obicei orice PC care are un USB 2.0 protocol poate boot-a de pe USB/SSD.
  • Când boot-ați de pe stick/card trebuie să actionați din ecranul grub F4 >Hard Drive sau să adăugați fromhd la linia de boot-are grub.
  • Este important de ținut minte că, următoarele metode vor suprascrie și vor distruge orice tabelă de partiții. Datel pierdute vor depinde de mărimea sidux-*.iso. Nu sunt limitări în privința spațiului de stocare disponibil câtă vreme Linux-ul este implicat și vă veți putea recupera datele ramase pe care imaginea ISO nu le-a șters în timp ce se pare că MS Windows alocă și recunoaște doar o singură partiție. Așa că vă rugăm să nu faceți asta pe hard-ul vostru de 100+- gig!

Linux    MS Windows    Mac OS X

Linux

Cuplați USB stick-ul sau card-reader-ul cu card-ul pe care vreți să scrieți și executați:

cat /path/to/sidux-*.iso > /dev/USB_raw_device_node

sau

dd if=/path/to/sidux-*.iso of=/dev/sdf
Examplu:

Cuplați dispozitivul, rulați dmesg și uitați-vă la output:

sd 13:0:0:0: [sdf] Assuming drive cache: write through
sd 13:0:0:0: [sdf] Assuming drive cache: write through
sdf: sdf1 sdf2
sd 13:0:0:0: [sdf] Assuming drive cache: write through
sd 13:0:0:0: [sdf] Attached SCSI removable disk

Presupunând o imagine ISO numită sidux-2009-04-moros-kde-lite-i386-200912310312.iso a fost download-ată, (puteți s-o redenumiți ca sidux-moros-kde-lite.iso), apoi comada de rulare va fi:

cat /home/username/sidux-moros-kde-lite.iso > /dev/sdf

sau

dd if=/home/username/sidux-moros-kde-lite.iso of=/dev/sdf

Când boot-ați de pe stick/card trebuie să acționați din ecranul grub F4 >Hard Drive sau să adăugați fromhd la linia de comandă grub.

MS Windows™

E rapid și simplu. Instalați un program numit flashnul, ( de aici).

Rețineți numele 'flashnul' când îl extrageți și-l instalați deoarece poate arăta ca un dosar numit "flashnul-x.x", (de examplu, 'flashnul-1rc1.zip' download-at, extras și instalat în Program Files are un dosar numit "flashnul-1rc1", care are înăuntru un alt dosar numit "flashnul-1rc1", care la rândul său conține programul 'flashnul' și fișierele/dosarele asociate ce vor fi folosite în exemplul nostru.)

Cuplați stick/card-ul.

Dacă ați extras și instalat 'flashnul' în Program Files dar nu sunteți sigur unde este mout-at USB stick-ul, analizați output-ul comenzii 'flashnul -p' facând click pe run din menu, apoi tastând cmd pentru a porni un terminal, apoi mutați-vă în directorul care conține toate fișierele individuale ale programului "flashnul", în cazul nostru:

cd C:\Program Files\flashnul-1rc1\flashnul-1rc1\
flashnul -p

Output-ul ar trebui să arate cam așa:

Available physical drives:
0       size = 60022480896 (55 Gb))
1       size = 2003828736 (1911 Mb)

Presupunând o imagine ISO numită sidux-2009-04-moros-kde-lite-i386-200912310312.iso a fost download-ată, (you could rename it to sidux-moros-kde-lite.iso), folosiți 'cut' și 'paste', (sau copiați-o), în dosarul "flashnul-1rc1" care conține toate fișierele/dosarele individuale ale programului.

Conform output-ului 'flashnul -p' din examplul nostru, vedem că dispozitivul USB a fost mount-at în 1, tipăriți urmatoarele apoi apăsați tasta 'enter', (s-ar putea să fie nevoie de drepturi de administrator):

flashnul 1 -L sidux-moros-kde-lite.iso

Când boot-ați de pe stick/card trebuie să acționați din ecranul grub F4 >Hard Drive sau să adăugați fromhd la linia de comandă grub.

Mac OS X™

Cuplați dispozitivul usb iar Mac OS X trebuie să-l mout-eze automat. În Terminal (îl găsiți în dosarul Applications > Utilities), rulați:

diskutil list

Descoperiți cum se numeste dispozitivul usb apoi 'unmount' partițiile de pe dispozitiv (să presupunem c-ar fi /dev/disk1):

diskutil unmountDisk /dev/disk1

Presupunând o imagine ISO numită sidux-2009-04-moros-kde-lite-i386-200912310312.iso a fost download-ată în /Users/username/Downloads/, (you could rename it to sidux-moros-kde-lite.iso). Dacă dispozitivul USB este desemnat ca disk1, rulați:

dd if=/Users/username/Downloads/sidux-moros-kde-lite.iso of=/dev/disk1

Când boot-ați de pe stick/card trebuie să acționați din ecranul grub F4 >Hard Drive sau să adăugați fromhd la linia de comandă grub.
