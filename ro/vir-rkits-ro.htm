Informatiile urmatoare vin de pe un forum

Pentru a instala antivirusul PE Linux executati urmatorii pasi :

1. ca root
# cd /tmp

2. Descarcati de aici :
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. extrageti fisierul  .tar descarcat 
# tar xvpzf antivir-workstation-pers.tar.gz

4. schimbati in dir
# cd antivir-workstation-pers-2.1.9-18
(sau numarul aplicatiei disponibil la acel moment

5. start install script. si urmati instructiunile de instalare
./install

6.inlaturati vechiul fisier antivir.conf
# mv /etc/antivir.conf /etc/antivir.conf.old

Nu uitati sa faceti un update
# antivir --update

rkhunter

rkhunter rootkit scanner este o unealta de scanare care va ajuta sa va asigurati ca sistemul nu este infectat cu softuri daunatoare . Programul scaneaza impotriva rootkits , backdoors si local exploits ruland teste ca :
- Compararea sumelor MD5
- Cauta fisiere care sunt de obicei folosite de rootkits
- Permisiuni incorecte pentru fisierele binare
- Cauta siruri suspecte in modulele LKM si KLD
- Cauta fisiere ascunse
- Optional scaneaza in interiorul fisierelor plaintext si binare
- Rootkit Hunter este liber spre utilizare fiind realizat sub licenta GPL .

apt-get update
apt-get install rkhunter
rkhunter --update

rkhunter va va intreba de asemeni daca doriti sa setati o scanare regulata dupa o programare .

Pentru a scana folosind rkhunter :
rkhunter -c
