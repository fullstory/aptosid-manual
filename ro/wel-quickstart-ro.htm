sidux Quick Start Guide

FOARTE IMPORTANT: sidux , ca LIVE-CD, este bazat pe tehnologie de înaltă compresie , şi din această cauză este nevoie de grijă deosebită când se scrie o imagine ISO . Folosiţi doar CD-uri sau DVD-uri de bună calitate pe care să le scrieţi doar în DAO-mode (disk-at-once) şi nu cu viteză mai mare de 8X.

sidux aims to be 100% compatible with Debian sid, however sidux may provide temporary hot-fixes from time to time. The sidux apt-repository holds sidux specific packages which includes the sidux kernel, scripts, tools and documentation.

Debian sid/stabilitate instabila

sid este un ecuson pentru depozitele instabile ale lui Debian. Debian sid este un depozit de pachete adus la zi în mod frecvent, care se sincronizează rapid cu versiunile de software cele mai recente şi cele mai bune. Datorita frecvenţei de actualizare, testările generale ale pachetelor sunt posibile mai rar, pentru o perioadă scurtă de timp, pâna la apariţia ultimelor noutaţi şi sunt distribuite ca pachete Debian.

kernel-ul sidux

Kernel-ul sidux este optimizat ca să ajute la înlăturarea bug-urilor, adaugă functionalităţi noi, sau configurează pentru o performanţă superioară şi o stabilitate sporită şi îmbunatăţită ultimul kernel din http://www.kernel.org/. .

Administrarea pachetelor

sidux is compliant with Debian packaging and uses apt and dpkg for software package management with debian and other repositories identified by the files in /etc/sources.list.d/*

Debian sid are peste 20.000 de pachete, deci puteţi să-l gasiţi pe acela pe care doriţi să-l folosiţi cuCautarea de pachete cu apt-cacheor with Debian Package Search GUI application.

Să instalaţi pachetul apt-get install <numele pachetului > Instalarea unui pachet nou.

Depozitele lui Debian sid sunt aduse la zi de patru ori pe zi ceea ce inseamnă că lista de pachete la dispoziţie şi versiunile lor se demodează foarte rapid. Executarea apt-get update este esenţial înainte de instalarea unui pachet nou sau înainte de a rula apt-get dist-upgrade ca să vă menţineţi la zi cu lista de pachete ale server-ului.

Using other Debian based distributions' repositories, Source and RPMs

Nu oferim asistenţă la instalarea pachetelor din codul sursă. Dacă chiar aveţi nevoie să compilaţi aplicaţia, efectuati-o ca user, şi puneţi-o întrun dosar din /home fără s-o instalaţi în sistem. Nici nu oferim asistenţa la folosirea de checkinstall sau la convertirea pachetelor RPM cu alien.

Debian based distributions' often folosesc adesea versiuni diferite ale locaţiilor dosarelor pentru aplicaţii, lucru ce ar putea cauza instabilitatea sistemului şi unele pachete nu se vor putea instala, datorita unor dependenţe de nerezolvat, sau a metoadelor de denumire diferite ale pachetelor sau a versiunilor numerice diferite. De exemplu chiar doar o versiune diferita de glibc poate impiedica rularea aplicatiei.

Actualizarea sistemului - dist-upgrade

apt-get dist-upgrade este metoda recomandată de actualizare în sidux. Nu este recomandată folosirea niciunui alt program grafic de actualizare în sidux. Actualizarea unui sistem instalat - dist-upgrade

Recomandăm folosirea dist-upgrade numai înafară mediului X. Lansarea "init 3" de la window manager-ul dvs. (KDE, XFCE, etc) sau într-un terminal virtual (ctrl+alt+f1, ctrl+alt+f2, etc) va impiedica rularea mediului X şi nu va permite o actualizare în condiţii de siguranţă.

Configuratia de Network

Ceni este o unealta de configuratie de network ce configureaza rapid sau cu cartela wireless fara mult efort. Functiunea wireless poate sa scaneze alte retele, sa foloseasca wep si wpa pentru incriptii, si sa foloseasca uneltele wireless sau wpa_supplicant pentru configuratia wireless. Ethernet este clar daca folositi dhcp (automatic ip address assigningsau puteti seta manual de la netmasks la nameservers.

Ceni ieste rulat cu comanda Ceni sau ceni .Daca nu este instalat, îl puteti instala cu comanda apt-get install ceni.

Alte Window Managers

KDE, XFCE şi fluxbox sunt sidux suportate. Gnome nu este sustinut de sidux. Gnome ca si un window manager în depozitele sid este ştiut ca instabil şi poate cauza probleme in acest moment. Unii utilizatori din forumul sidux /wiki şi IRC chat s-ar putea să fi avut ceva experienţe şi ar putea să vă ajute. Altfel, vă descurcaţi singuri.
