Запись CD/DVD без GUI

ВАЖНАЯ ИНФОРМАЦИЯ: sidux, как другой Live-CD, основан на технологии архивации, первоначально разработанной для Knoppix, но теперь используемой в усовершенствованной форме. По этой причине нужно уделять особое внимание методу записи ISO-образа. Мы рекомендуем использовать высококачественные носители CD (или DVD+R) для записи в режиме DAO (диск за приём) не быстрее чем восьмикратно (8x).

Для записи CD/DVD не обязательно использовать графический интерфейс (GUI).

Проблемы, которые встречаются при записи, обычно относятся к k3b, реже к таким фронтендам, как growisofs, wodim или cdrdao.

sidux предоставляет в распоряжение скрипт burniso, чтобы сжигать sidux ISO-образ.

burniso burns ISO images on media, using Disk-At-Once mode and a hard coded speed of 8x, using wodim. 

apt-get install sidux-scripts

As $User:

$ cd /dir/containing/your/ISO

$ burniso

All ISO images in the current directory are presented and will start burning right after your selection of an ISO, so be sure you have your media inserted.

Определение привода для записи:

Для устройств:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/scd0'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/scd1'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

Другие альтернативы:

$ wodim dev=/dev/scd0 driveropts=help -checkdrive

или

$ wodim -prcap

cdrdao:

$ cdrdao scanbus
Cdrdao version 1.2.1 - (C)  Andreas Mueller
ATA:1,0,0 AOPEN , CD-RW CRW2440 , 2.02
ATA:1,1,0 _NEC , DVD_RW ND-3540A , 1.01

Примеры:

Сведения о пустых CD/DVD:

$ wodim dev=/dev/scd0 -atip

или

$ cdrdao disk-info --device ATA:1,0,0

Повторно записываемые накопители можно очистить командой:

$ wodim -blank=fast -v dev=/dev/scd0

или

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal

Копирование CD:

$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2

Копирование CD "на лету" :

$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2

Запись звукового CD с файлами wav на 12ти кратной скорости:

$ wodim -v -eject -pad -dao speed=12 dev=/dev/scd0 defpregap=0 -audio *.wav

Запись CD из файла bin/cue:

$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue

Запись CD из ISO-образа:

$ wodim dev=/dev/scd0 driveropts=burnfree,noforcespeed fs=14M speed=10 -dao -eject -overburn -v something.iso

If you get a driveropts error, it is because burnfree is deprecated for some devices, therefore:

$ wodim dev=/dev/scd0 driveropts=noforcespeed fs=14M speed=8 -dao -eject -overburn -v something.iso

или

$ wodim dev=/dev/scd0 fs=14M speed=8 -dao -eject -overburn -v something.iso

Создание ISO-образа из папки и всех ее подкаталогов:

$ genisoimage -o myImage.iso -r -J -l directory

Можно использовать growisofs, чтобы записать DVD, например, ISO-файл:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso

Запись нескольких файлов на DVD:

$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...

Если на DVD еще имеется место, то добавить файлы можно так:

$ growisofs -M /dev/dvd -R -J anotherfile andanotherfile...

Чтобы закрыть сеанс:

$ growisofs -M /dev/dvd=/dev/zero $
