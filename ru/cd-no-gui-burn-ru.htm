Записка CD/DVD записать без GUI

Для записки CD/DVD можно не пользоваться графическйй интерфейс (GUI).

Проблемы, которые встречаются при появлении записи, имеют свои причины обычно в GUI для k3b, не так часто в backends, такие как growisofs, wodim или cdrdao.

Как я нахожу мой привод для записи:

Для устройств:

wodim:

$ wodim --devices
wodim: Overview of accessible drives (2 found) :
---------------------------------------------------------
 0  dev='/dev/hdc'      rwrw-- : 'AOPEN' 'CD-RW CRW2440'
 1  dev='/dev/hdd'      rwrw-- : '_NEC' 'DVD_RW ND-3540A'
---------------------------------------------------------

cdrdao чек уст­ройств:

$ cdrdao scanbus
Cdrdao version 1.2.1 - (C)  Andreas Mueller
ATA:1,0,0 AOPEN , CD-RW CRW2440 , 2.02
ATA:1,1,0 _NEC , DVD_RW ND-3540A , 1.01

Полезные примеры:

Сведения о пустых CD/DVD:

$ wodim dev=/dev/hdc -atip

или

$ cdrdao disk-info --device ATA:1,0,0

Удаляют повторно записываемые накопители так:

$ wodim -blank=fast -v dev=/dev/hdc

или

$ cdrdao blank --device ATA:1,0,0 --blank-mode minimal

CD копируют так:

$ cdrdao copy --fast-toc --device ATA:1,0,0 --buffers 256 -v2

Копируют CD "on the fly" так:

$ cdrdao copy --fast-toc --source-device ATA:1,1,0 --device ATA:1,0,0 --on-the-fly --buffers 256 --eject -v2

Звуковой CD посредством файлов wav с 12ти кратной скоростью записывают так:

$ wodim -v -eject -pad -dao speed=12 dev=/hdc defpregap=0 -audio *.wav

CD из файла bin/cue-имидж записывают так:

$ cdrdao write --speed 24 --device ATA:1,0,0 --eject filenam.cue

CD из имидж ISO-файла записывают так:

$ wodim dev=/dev/hdc driveropts=burnfree,noforcespeed fs=14M speed=10 -dao -eject -overburn -v something.iso

ISO-файл из папки и всех под каталогов создают так:

$ genisoimage -o myImage.iso -r -J -l directory

Можно использовать growisofs, чтобы записать DVD, например, ISO-файл:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso

Несколько файлов на DVD записывают так:

$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...

Если на DVD еще имеется место, то можно добавить файлы так:

$ growisofs -M /dev/dvd -R -J anotherfile andanotherfile...

Чтобы закрыть сеанс:

$ growisofs -M /dev/dvd=/dev/zero $
