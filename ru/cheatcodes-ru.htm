Cheat коды

Синтаксис использования данных из поля "значение" таблицы: cheat_код=значение . Если, например, желаемое разрешение экрана - 1280x1024, то в командную строку grub необходимо ввести screen=1280x1024, для использования русского языка, lang=ru.

Особые Cheat коды для sidux LiveCD

Cheat коды Значение Описание
blacklist имя модуля временно отключить модули до запуска udev
desktop kde выбор менеджера окон: kde
fluxbox выбор менеджера окон: fluxbox
fromiso Смотрите в главе "fromiso"

lang be, bg, cs, da, de, de_CH, el, en, en_AU, en_GB, en_IE, es, fr, fr_BE, ga, hr, hu, it, ja, nl, nl_BE, pl, pt (pt_BR), pt_PT, ro, ru, zh Sets your language. It will also set locale, keyboard layout (console and X), timezone and a debian mirror.
When the extended form lang=ll_cc or lang=ll-cc is given, ll will be used for language selection and cc for keyboard, mirror and timezone selection (e.g. lang=fr-be). The defaults for English are en_US with America/NewYork as the timezone and for German, de with Europe/Berlin as the timezone. Example: lang=pt_PT_US tz=Pacific/Auckland

X related cheatcodes

Either the xrandr or xmodule cheatcode should also be used if using other X related cheatcodes with a radeon, intel or mga graphics card.

Cheat коды Значение Описание
dpi auto or XX Set desired Dots Per Inch for your display. You can work out your screen's dpi from the width in pixels divided by the diagonal size in inches and multiplying this by 1.25 for a 4:3 screen, 1.18 for a 16:10 screen or 1.147 for a 16:9 screen. For a 24" 1920x1080 screen it would be 1.147*1920/24 giving dpi=92 or a 15" 1600x1200 screen would be 1.25*1600/15 giving dpi=133.
hsync 80 (например) устанавливает горизонтальную X-частоту развертки монитора в килогерцах
noml вынуждает конфигурацию X.org обходиться без "modelines" для использования RandR 1.2 драйверов.
noxrandr дезактивирует применение расширений RandR 1.2 новыми драйверами X.org и использует старые техники для опроса свойств монитора
screen 1280x1024 устанавливает определенное пользователем разрешение для X (1260x1024 или другие разрешения дисплея)
vsync 60 (например) устанавливает вертикальную синхронизацию X
xdepth значения: 8 15 16 24 устанавливает глубину цвета, которую использует X.org (не все драйверы поддерживают 1 и 4)
xkblayout (обычно us, de) устанавливает расположение клавиш на клавиатуре, которое используется X.org
xkbmodel (обычно 104, 105) устанавливает тип клавиатуры, который используется X.org (число обозначает количество клавиш)
xkbvariant обычно nodeadkeys устанавливает вариант клавиатуры, который используется X.org
xmode (обычно 800x600) устанавливает размер экрана по указанным данным, 1024x768, 1600x1200 etc
xmodule or xdriver ati, fbdev, i810, intel, mga, nv, radeon, savage, vesa использует выбранный модуль X
xrandr вынуждает конфигурацию X.org использовать расширения RandR 1.2
xrate XX вынуждает предпочитаемую частоту повторения при драйверах, которые поддержаны RandR 1.2. Эта опция должна использоваться в сочетании с загрузочной опцией xmode. Подробная документация находится здесь.
xhrefresh 75 (например) устанавливает горизонтальную частоту в X (монитора) в килогерцах
xvrefresh 60 (например) устанавливает вертикальную частоту  в X (монитора) в герцах
