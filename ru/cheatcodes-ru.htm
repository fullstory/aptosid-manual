Cheat коды

Синтаксис использования данных из поля "значение" таблицы: cheat_код=значение . Если, например, 1280x1024 желаемое разрешение экрана, то необходимо ввести screen=1280x1024 в командную строку grub, или для xmodules xmodule=vesa, или для acpi acpi=off

Cheat коды Значение Описание
acpi off отключает ACPI полностью
oldboot активирует только элементы ACPI, в которых нуждаются для загрузки
force включает ACPI
apm off отключает Advanced Power Management
desktop kde выбор менеджера окон: kde
fluxbox fluxbox
gnome gnome
icewm icewm
xfce xfce
dpi auto or XX setzt die gewünschten Pixel pro Zoll für hren Monitor (z. B. mein 15" 1600x1200 laptop hat 1600/(15*(4/5)) = 133 also dpi=133)
failsafe устанавливает процесс загрузки на надежные значения
fromiso some/path/sidux.iso Please refer to booting "fromiso"
hostname whatever-your-host-name-is hostname, изменяет имя хоста системы на Live-CD
hsync 80 (z.B.) устанавливает горизонтальную X-частоту развертки монитора в килогерцах
irqpoll использует опрос IRQ
lang au, be, bg, br, ch, cn, cz, de, dk, es, fi, fr, gr, gb, il, hr, ie, ga, it, jp, nl, pl, pt, ru, si, sk, tr, tw, us Set your default language. It will also influence selection of default locale, keyboard layout (console and X), timezone and debian mirror.
For sidux releases where the relevant i18n packages have been installed, the preferences wil apply.
The form "lang=ll-cc" is honored, where both ll and cc can take any of the supported 2-char values. "ll" will be used for language selection and "cc" for keyboard and timezone selection (e.g. "lang=fr-be"). This decouples the notions of language and geographical location.
md5sum проверяет, в порядке ли загруженный CD
mem 128MB (например) использует указанную величину памяти
noacpi не использует поддержку acpi (смотри переключатель ACPI)
noagp не использует поддержку AGP (Accelerated Graphics Port)
noapic не использует опрос APIC
noddc не использует  опрос данных монитора
nodhcp не использует DHCP (DHCP пробует автоматически создать Ethernet соединения)
nodma не использует поддержку для DMA (Direct Memory Access)
noeject не удаляет CD из дисковода
nofloppy не использует опрос floppy
nointro пропускает выход index.html при запуске Live-CD
noisapnpbios не проводит  ISA-" Plug and Play " опроса при запуске
nomce отключает сообщения при незначительной погрешности
noml возвращает все Monitor-Modelines и осведомляется, которые из них должны использоваться
nocpufreq не активирует Speedstep/Powernow
nosmp не использует Symetric Multi-Prozessor (несколько центральных процессоров или центральный процессор с Hyper-Threading)
notouchpad не использует поддержку Touchpad
nowheel вынуждает PS/2-Protokol для мыши PS/2
pci noacpi не использует ACPI для устройств PCI
quiet не происходит выход на дисплее
screen 1280x1024 устанавливает определенное пользователем разрешение для X (1260x1024 или другие разрешения дисплея)
toram копирует CD в RAM и стартует с копии в RAM
vga normal больше о кодах VGA здесь: VGA коды
vsync 60 (например) устанавливает вертикальную частоту монитора в килогерцах
xmodule ati использует общую модуль ATI
fbdev использует общую модуль фрейм буфера
i810 использует общую модуль Intel
mga использует общую модуль Matrox
nv использует общую модуль Nvidia
radeon использует общую модуль ATI RADEON
savage использует общую модуль S3 Savage
vesa использует общую модуль Vesa
xhrefresh 75 (например) устанавливает горизонтальную частоту в X (монитора) в килогерцах
xvrefresh 60 (например) устанавливает вертикальную частоту  в X (монитора) в герцах
