Пароль суперпользователи root на Live-CD

Внимание: Когда всегда Вы работаете с правами root, нужно знать, что делать. Для навигации в интернете никакие права root не нужны.

На LiveCD никакой пароль root не установлен. Имеются несколько возможностей, запускать программу с правами root:

(Временный) пароль root устанавливают так:

Открывают консоль:

sidux@0[sidux]$ sudo passwd
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
sidux@0[sidux]$

Этот пароль может использоваться до конца Live сеанса.

Программа как root запускается так

sudo
oder
su-me
Нажимаjйте Alt+F2 и Enter и вводите
sudo

перед желаемым именем программы, например:

sudo gparted
Программу из терминала root запускают так:

Затем Вы открываете терминал или консоль и вводите последующему синтаксису:

sidux@0[sidux]$ su
root@0[sidux]# gparted ( (или желаемую программу)

Чтобы покинуть терминал, вводится:

exit

после этого еще раз:

 exit

вместе с этим терминал должен закрыться.

Установка ПО в режиме LiveCD

apt-get update
apt-get install your-preferred-package

Примечание: Когда Вы выключите компьютер в режиме LiveCD, никакие изменения не будут сохранены.

Разделы NTFS с ntfs-3g делают фиксирующимися

Внимание: Хотя драйвер ntfs-3g объяснен теперь стабильным, мы рекомендуем никакое использование без ресерв-копирования данных, прежде всего, не на рабочих платформах! Использование на собственный риск, потеря данных не исключена!

В консоль вводят следующее: Смотрите разбиение жесткого диска, для выяснения наименования разделов жесткого диска в Linux.

sidux@9[sidux]$ su
root@0[sidux]# apt-get update && apt-get install ntfs-3g
root@0[sidux]# umount /media/xdxx
root@0[sidux]# mount -t ntfs-3g /dev/disk/by-uuid/xxyyzz[etc] /media/xdxx
Zum Verlassen der Konsole gibt man nur ein: exit

Теперь NTFS-раздел должен  быть присоединен с письменным доступом, и данные могут быть там сохранены. И еще раз наше предупреждение:Зтот образ действия упомянут для ситуаций, для которых не имеется никаких других возможностей решения. Для ежедневного рутинного использования этот образ действий не рекомендован!
