Сетевой поиск

Из-за некоторых ограничений функции поиска используйте общие английские компьютерные слова, чтобы искать в российском руководстве, например partition вместо разделение. Для офлайного поиска используйте Recoll как описано в заголовке 'Офлайновый поиск'.

Введите одно или несколько слов в поле поиска и нажмите кнопку Поиск (или клавишу Return).

Варианты поиска

Если 'Совпадение всех слов' выбрано, только ти документы будут показаны, которые содержат все условия поиска. Если Match ANY word выбрано все документы будут показаны, в которых содержится хотя бы одно из условий поиска.

Альтернативно Вы можете поместить знак плюс (+) непосредственно перед одним или более словами, чтобы только получить те файлы, которые включают все те слова. Слова с  знаком минус (-) изменяют результат так, чтобы только документы были перечислены, который не содержит ни одного из тех слов.

Результаты поиска сортируются по релевантности. Наиболее релевантные документы перечисленны в первую очередь. Значимость зависит от количества и расположенияслов в документах.

Офлайновый поиск

Группа руководства sidux разработали некоторые специальные конфигурации в сочетании с разработчиком Recoll в качестве составной части пакета по умолчанию. Recoll является личным полнотекстовый инструмент поискa для Unix/Linux. Он основан на Xapian конечном приложении, чтобы получить всеобъемлющий и удобный графического QT интерфейс. Большое спасибо разработчику Recoll.

Recoll называется Personal Search Tool в Debian меню KDE.

Шаг 1

В настоящее время руководство доступно через apt для следующих языков:
da, de, el, en, es, fr, hr, it, ja, nl, pl, pt-br, ro, ru.

apt-get update
apt-get install sidux-manual-ru
Шаг 2
apt-get install recoll
Шаг 3
  • Recoll on the first start will ask you if you want to build the database, click cancel, where upon an indexing configuration box will appear.
  • In the Global parameters - Top directories box, click on the + (plus) sign and go to /usr/share/sidux-manual and click on your language and click ok. Next click on the - (minus) sign and delete any other folders including ~ (tilde).
  • Finally File>Update Index and search the topic you need in the sidux-manual.

После обновления sidux руководство по apt, необходимо Recoll повторно инициализировать (как пользователь $ в консоли):

recollindex -z

Затем: Kmenu -> Debian -> Applications -> Data Management -> Personal Search Tool и в файле Recoll -> Update Index.

The 'Preview' link will highlight all instances of your search word in the file, whereas 'Open' will open Konqueror, and use the Find on this page command for a micro search for the word you searched in Recoll.
