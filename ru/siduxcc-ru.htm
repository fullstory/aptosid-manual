sidux центр управления

sidux центр управления - внешний интерфейс для того, чтобы управлять sidux и состоит из модулей. Он доступен на хорватском, датском, нидерландском, английском, французском, немецком, японском, польском, португальском, итальянском, румынском, русском, испанском и греческом языках. Приложение доступно в KDE и в раыличных версиях GTK.

siduxcc содержит 7 модулей со следующими функциями:

1. Модуль дисплея

Модуль предназначен

  • для установки разрешения и частоты
  • для изменения цветной глубины и DPI
  • для конфигурации X сервера
  • для инсталляции графической карты
Kменю -> sidux -> sidux центр управления -> дисплей
2. Модуль интернета

Модуль предназначен

  • для выбора браузера sidux
Kменю -> sidux -> sidux центр управления -> интернет
3. Модуль ядра

Модуль предназначен

  • для инсталляции нового ядра
  • для удаления старого ядра
  • для инсталляции модулей ядра
Kменю -> sidux -> sidux центр управления -> ядро
4. Модуль локальной сети

Модуль предназначен

  • для установки карты локальной сети
  • для установки имени хоста
  • для установки DNS-сервера
  • для инсталляции безпроводных карт WLan (ndiswrapper)
  • для инсталляции безпроводных карт WLan (fwdetect)
Kменю -> sidux -> sidux центр управления -> локальная сеть
5. Модуль служб

Модуль предназначен

  • для управления службами (как старт, стоп, перезапуск, перезагрузка, удаление/добавление runlevel, Apache, Samba, SSH...)
  • для старта служб во время загрузки
Kменю -> sidux -> sidux центр управления -> службы
6. Модуль сообщения
  • Shows you information about your system
Kменю -> sidux -> sidux центр управления -> системное сообщение

sidux hermes

sidux-hermes (dist-upgrade sentinel) is a kde-tray application that warns you if there are any dist-upgrade problems. It scans the News Section of sidux.com for the keyword 'Warning' and shows a red stoplight should it find a match. There is also a gtk version.

sidux-hermes-qt4

It is recommended to install the qt4 version. It has the most features and it works on all desktops (KDE3, KDE4, Gnome, Xfce4).

sidux-hermes-qt4 has options like kernel information and the ability to download files ready to update (dist-upgrade to be carried out in init 3 later). 

apt-get install sidux-hermes-qt4
sidux-hermes-gtk2

Use the gtk2 version if you want to have an application without qt dependencies.

apt-get install sidux-hermes-gtk2
sidux-hermes cli

sidux-hermes contains the backend of all hermes versions. It will be automatically installed when you install the qt4 or gtk2 versions.

apt-get install sidux-hermes
