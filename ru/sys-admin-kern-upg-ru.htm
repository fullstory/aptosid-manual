Обновление ядра

Ядра sidux'a находятся в apt. Выбирайте близлежащий зеркальный сервер и адаптируют Ваш $sources.list:

Ядра доступны в следующих формах:

  • sidux-686 - Ядро для процессоров архитектуры i686 моно/двух и более ядерных
  • sidux-amd64 - Ядро для 64бит sidux
Steps to do it manually in a konsole:

1. Вводите как root в консоли:

apt-get update

2. Для установки самой актуальной версии ядра:

apt-get install linux-image-2.6-sidux-686 linux-headers-2.6-sidux-686

Если с новым ядром оказываются проблемы, Вы можете перезагрузить и выбрать старшее ядро.

Модули

Чтобы найти в каких модулях Вы нуждаетесь, следующая команда дает Вам список текущих доступных модулей. Скопируйте эту линию в Ваш терминал: 

apt-cache search 2.6.*.slh.*-sidux| awk '/modules/{print $1}'

Чтобы получить полное описание каждого модуля, скопируйте эту линию в Ваш терминал::

apt-cache search 2.6.*.slh.*-sidux

Reboot to use the new kernel

Чтобы устанавливать необходимые модули (например для virtual-ose и qc-usb):

apt-get install virtualbox-ose-modules-2.6.24-2.6.24.2.slh.7-sidux-686 (EXAMPLE)
apt-get install qc-usb-modules-2.6.24-2.6.24.2.slh.7-sidux-686 (EXAMPLE)

Чтобы проверять загружены ли модули в ядре, используют одну из следующих команд: 

ls /sys/module/
или
cat /proc/modules

ВАЖНО: Если Вы нуждаетесь в определенных модулях драйвера, например для wlan, Вы должны установить надлежащие пакеты .deb для них. Как пример, для сетевого адаптеpа Agere et131xр:

apt-get install et131x-modules-*

dpkg -i et131x-modules*.deb может только использоваться, если debs имеются в наличии уже локально. Но даже в том случае, когда они являются в кэше apt-cache, apt-get лучшую опцию.

Дополнительная информация Getting on with Ceni и для WIFI.

Другие методы инсталляций ядра

siduxcc

Вы можете также установить ядра с помощью центра управления siduxcc, который расположен в KDE-старт -> мenю -> Система -> sidux центр управления. siduxcc также предлагает несколько других системных приложений GUI администрации.

Удаление старого ядра (kernel remover)

После успешной установки нового ядра можно удалить kernel.zip и архив ядра, хотя рекомендуется его и более старое ядро сохранять ещё несколько дней, на случай непредвиденных проблем и необходимости перезагрузится с более старым ядром. Более старые ядра перечислены в меню менеджера загрузок Grub.

Старые ядра могут удаляться скриптом kernel-remover, который находится в пакете sidux-kernelhacking. Этот скрипт перечисляет те ядра, которые могут быть удалены.

apt-get update
apt-get install sidux-kernelhacking
kernel-remover

Альтернативно, Вы можете использовать siduxcc, который использует скрипт kernel-remover.
