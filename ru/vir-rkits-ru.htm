AntiVir PersonalEdition Classic для Linux

Следующая информация происходит из топика с форума

Чтобы инсталлировать antvir PE Linux, необходимы следующие шаги:

1. как root
# cd /tmp

2. Скачайте отсюда::
# wget http://free-av.com/personal/en/unix/antivir-workstation-pers.tar.gz

3. Распаковайте загруженный файл tar
# tar xvpzf antivir-workstation-pers.tar.gz

4. Перейдете в директорию
# cd antivir-workstation-pers-2.1.9-18
(или номер версии в данное время)

5. Запустите инсталляционный скрипт и следуйте за инструкциями по инсталляции
./install

6. Переместите старый antivir.conf
# mv /etc/antivir.conf /etc/antivir.conf.old

В конце выполните обновление версии
# antivir --update

rkhunter

Сканер по rootkit rkhunter - это приложение, которое служит для того, чтобы Ваша система была без вредоносных программ. Это приложение сканирует на наличие rootkits, backdoors и локальных Exploits, в это время проводятся следующие тесты как:

- Сравнение проверочной суммы MD5
- Сканирование на наличие использованных rootkits файлов по умолчанию
- Сканирование на наличие ошибочных прав для двоичных файлов
- Сканирование на наличие подозрительных Strings в LKM- и KLD-модулей
- Сканирование на наличие спрятанных файлов
- Опциональное сканирование внутри текстовых и бинарных файлов
- rootkit-hunter - это лицензированный проект под GPL и для каждого свободный к использованию

apt-get update
apt-get install rkhunter
rkhunter --update

rkhunter также осведомляется, должна ли устанавливаться т.н. работа Cron, это автоматически делает возможным сканированием, по прошествии определенного времени.

Как сканируют с rkhunter
rkhunter -c

chkrootkit

chkrootkit является приложением, которое ищет следы от rootkits.

apt-get install chkrootkit
Um mit chkrootkit zu scannen:
chkrootkit

chkrootkit prüft nach folgenden Regelsätzen:

ifpromisc.c
Проверяет, находится ли интерфейс в "promiscuous mode" (Promiskmodus).

chklastlog.c
Проверяет на наличие удалений в lastlog

chkwtmp.c
Проверяет на наличие удалений в wtmp

chkproc.c
Проверяет на наличие следов троянцев LKM

chkdirs.c
Проверяет на наличие следов троянцев LKM

strings.c
"quick and dirty" замену strings

chkutmp.c
Проверяет на наличие удалений в utmp
