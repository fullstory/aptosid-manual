Додавання non-free в список джерел і вбудоване програмне забезпечення (прошивка)

Cписок джерел

Як відомо ISO-образ aptosid містить тільки вільне ПЗ згідно DSFG, але існує можливість додати contrib non-free в файл / etc / apt / sources.list.d / debian.list 

mcedit /etc/apt/sources.list.d/debian.list

Запис в файл повинен виглядати таким чином. Bибір дзеркального сервера можливо часом змінювати. Прочитайте також Список поточного стану Debian серверів і дзеркал.:

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Після змін джерел серверів Вам потрібно виконати команду в терміналі apt-get update перед пошуком і установкою нових пакетів ('apt-get update' є для кожної установки рекомендованим).

Визначення невільної прошивки

З наступного командою встановлюються невільні прошивки, (якщо contrib non-free були додані в джерела):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

Необхідну прошивку можна визначити за допомогою наступного списку пристроїв або прошивок або з скриптом fw-detect, який знаходиться в пакеті aptosid-scripts).

$ fw-detect

Видача скрипта fw-detect описує команди, які потрібні щоб інсталювати і активувати прошивку:

Наприклад:

#Detected driver that requires firmware to operate
#Follow these instructions to obtain the correct firmware
# and activate the zd1211rw driver:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Щоб встановити прошивку з репозиторію git, введіть:

apt-get install git-core

non-free firmware debs on a stick

Якщо Ви потребуєте прошивці вже перед інсталяцією, вони повинні бути на USB-флешці. Щоб перезаписувати їх на другий комп'ютер, їх треба зберегти або як файл *. zip, або як архів *. tar.gz від сторінці http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/ та розпакувати в папці яка називається "firmware"в якому знаходяться після цього ряд файлів з закінченням *.deb.

Потім завантажте файл http://packages.debian.org/sid/firmware-linux-free *. deb та додайте цей до всіх інших *. deb в папці, до якої Ви витягли прошивку. Тоді передайте їх з флешкою на комп'ютер який потребує прошивку і:

dpkg -i firmware-linux-nonfree*.deb

Ми намагаємося надавати пакети для легально поширюваних прошивок від наших невільних сховищах, але не всі виробники дозволяють це.
