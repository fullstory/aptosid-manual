Зміст в ISO

aptosid забезпечує тільки Debian DFSG вільне програмне забезпечення на ISO-образі, будь ласка, перевірте цю посилання для отримання додаткової інформації про використання не вільних джерел для прошивки..

ISO повністю заснований на Debian Sid (від дати 2011-12-31) і стабілізузований з пакетами і скриптами вiд aptosid. Він поставляється з останній розширеній aptosid ядра, яке засноване на останніх vanilla kernel 3.1.6. ACPI і DMA включені за замовчуванням.

Повний список ПО входить в кожен варіант поточного випуску, називається manifest знаходиться на кожному дзеркалі завантаження:: aptosid дзеркала,завантаження та запис.

Випуск ISO варіації

aptosid представлений у п'яти варіантах ISO-образів заснованих на Debian Sid. Його установка зазвичай займає від п'яти до двадцяти хвилин:

Варіації:
  1. KDE lite 32 bit , ISO-образ приблизно 600MB:
    містить мінімальний набір програм оточення робочого столу KDE4. Додати відсутню Вам ПЗ можна за допомогою APT.
  2. KDE lite 64 bit , ISO-образ приблизно 600MB:
    містить мінімальний набір програм оточення робочого столу KDE4. Додати відсутню Вам ПЗ можна за допомогою APTt.
  3. XFCE 32 bit,ISO-образ приблизно 500MB:
    Повний набір робочого середовища XFCE, включені всі програми необхідні для роботи. Він має менший розмір ніж KDE4.
  4. XFCE 64 bit , ISO-образ приблизно 500MB:
    Повний набір робочого середовища XFCE, включені всі програми необхідні для роботи. Він має менший розмір ніж KDE4.
  5. KDE-full ISO-образ приблизно 2.5GB:
    містить обидві версії - 32 бітну і 64 бітну, повний набір оточення робочого столу KDE4, шістнадцять мов локалізації, а також додатки потрібні для роботи, такі як офісний пакет LibreOffice та інтернет браузер Iceweasel (Firefox).

Програми та інструменти

мінімальні системні вимоги KDE4 і XFCE

AMD64

  • Вимоги до процесора:
    • AMD64
    • Intel Core2
    • Intel Atom Series (2xx/ 3xx, N4xx/ D4xx/ D5xx; the N2xx series does not support EM64T)
    • VIA Nano
    • Будь x86-64 / EM64T процесор або новіший
    • Більш нові 64-бітові AMD Sempron і Pentium 4 CPUs(смотрітеіте на прапор "LM" в / Праці / CPUInfo або використовуєте скрипт infobash-v3).
  • Вимоги до ОЗУ:
    • KDE: ≥512 МБ ОЗУ (≥1 МБ рекомендується), ≥1 ГБ ОЗУ для застосування liveapt.
    • XFCE: ≥512 МБ ОЗУ
  • VGA відеокарта з роздільною здатністю принаймні 640x480 пікселів.
  • Оптичний привід або USB флешки.
  • ≥3 ГБ жорсткого диска, ≥10 ГБ рекомендується.
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)
    • UEFI+GPT (maximum HDD size <8 ZB)

i686

  • Вимоги до процесора:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (не K5/ K6)
    • Intel Atom Series (Z5xx/ Z6xx, N2xx)
    • VIA C3-2 (Nehemiah, не C3 Samuel або Ezra)/ C7
    • Любий x86-64/ EM64T процессор або новіший
    • Повний набір команд процесора i686.
  • Вимоги до ОЗУ:
    • KDE: ≥512 МБ ОЗУ (≥1 GB ОЗУ рекомендується), ≥1 GB ОЗУ для застосування liveapt.
    • XFCE: ≥512 MB RAM.
    • With i686, only 3.1 to 3.4 GB RAM is accessible, amd64 (x86-64/ EM64T) is strongly recommended for 64 bit capable hardware.
  • VGA відеокарта з роздільною здатністю принаймні 640x480 пікселів.
  • Оптичний привід або USB флешки.
  • ≥3 ГБ жорсткого диска, ≥10 ГБ рекомендується.
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)

Програми та інструменти

aptosid містить набір програм та інструментів, здатний задовольнити щоденні потреби користувача.

Вибір 2-х основних менеджерів ріжучої кромки робочого столу для доступу програм / додатків:
- KDE4 (англ. + нем.), один з самих настроюваних і зручних для користувача менеджерів робочого столу
- Xfce4 (англ. + нiм.), попроще ніж KDE4, але теж комфортабельний менеджер робочого столу.

Fluxbox (англ. + нiм.), легкий віконний менеджер входить у всі варіанти комплектації дисків.

Залежно від версії випуску в якості веб-браузера konqueror или iceweasel.

Office Productivity applications (again depending on the ISO variation range from LibreOffice, Abiword, Gnumeric, through to file managers like Dolphin and Thunar, (to name only a few).
