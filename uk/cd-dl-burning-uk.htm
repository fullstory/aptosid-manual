Визначення файли на кожного дзеркала

Кожнe aptosid дзеркало має наступні файли:

MD5SUM
MD5SUM.gpg
SHA256SUM
SHA256SUM.gpg
SOURCES
aptosid-20xx-xx-release-name-window-manager-arch-datetimestamp.arch.manifest
aptosid-20xx-xx-release-name-window-manager-arch-datetimestamp.iso

.manifest Файл містить список всіх пакетів, що містяться в конкретних образiв.

.iso є образ для завантаження.

MD5SUM and SHA256SUM Файли для перевірки цілісності образiв.

.gpg Файли підпису файлів, тому ви можете перевірити, що контрольні суми не були змінені, і що вони можуть бути використані для перевірки цілісності образiв.

Архів джерел aptosid *- source.tar призначений тільки для тих, які потребують статичної копії первинного коду для передач, щоб наповнювати вимоги ліцензування

md5sum та перевірка

ЦЕ ДУЖЕ ВАЖЛИВО: aptosid на Linux-LIVE-DVD/CD сильно стиснутий. З цієї причини потрібно звернути особливу увагу на запис ISO. Ми рекомендуємо для запису високоякісні CD (або DVD + RW), запис в режимі DAO-mode (disk-at-once) і не швидше ніж x8.

Це контрольна сума файла і використовується для перевірки цілісності файлів. Поточний md5sum з файлу в порівнянні з відомими контрольної суми. Таким чином, ви можете перевірити, якщо файл був змінений або пошкоджений, це доцільно для файлів, завантажених з Інтернету.

Якщо md5sum з завантаженого файлу відповідає один в md5-файл, ви можете бути впевнені, що файл був викачаний правильно. Під Linux ви отримуєте контрольну суму з: (займає одну хвилину, щоб обчислити)

$ md5sum file_to_check

Сума буде записана в консолi.
З md5sum (486 кб) md5sum також можуть бути підтверджені в ОС Windows.

ISO-образи aptosid завжди пропонуються з md5sum файлу і завжди повинні бути перевірені перед записом. Це гарантує, що, якщо виникають проблеми, вони не повинні шукати в завантажених файлів, а це веде форум вільний від проблем, які не можуть бути прикріплені вниз, тому що ISO-файл був пошкоджений в першу чергу. Ви повинні зробити це в консолi. Перейдіть в каталог з Iso-файлу і MD5-суму і видати команду мати md5-сума перевіряється.

Завантажте наступні файли з дзеркала:

MD5SUM
MD5SUM.gpg
SHA256SUM
SHA256SUM.gpg

В консолi:

$ md5sum aptosid.iso

Якщо вони не збігаються, ви отримаєте повідомлення про помилку:

"aptosid.iso: Error
md5sum: Warning: calculated checksum does not match!"

Якщо скачаний файл правильний, програма завершується без вести.

Якщо ви хочете, щоб мати можливість переконатися у справедливості .gpg files do the following as $USER:
приклад використання:

Важна информацiя: aptosid на Linux-LIVE-DVD/CD сильно стиснутий. З цієї причини потрібно звернути особливу увагу на запис ISO. Ми рекомендуємо для запису високоякісні CD (або DVD + RW), запис в режиміDAO (disk-at-once) і не швидше ніж 8x.

Само собою зрозуміло, CD можна записувати і в Windows. Завантажений файл повинен бути записуваний як ISO-файл. Winrar (або інша програма архівування), пов'язаний з ISO за замовчуванням, відкриває ISO-образ як простий архівний файл. Але ISO-образ використовується для запису CD.

Запис ISO-образу пiд Linux

Важлива інформація: aptosid на Linux-LIVE-DVD/CD сильно стиснутий. З цієї причини потрібно звернути особливу увагу на запис ISO. Ми рекомендуємо для запису високоякісні CD (або DVD + RW), запис в режимі DAO (disk-at-once) і не швидше ніж 8x.

На комп'ютері з уже встановленим Linux можна записати LiveCD усіх встановлених в системі програмою для запису дисків. У aptosid - k3b - стандартна програма для запису дисків. У пункті меню необхідно вибрати "Tools" -> ...Записати образCD (або DVD)" и выбрать записываемый ISO-образ (например, aptosid-Имя.iso).Встановіть режим DAO (Disk At Once) як метод для запису дисків.

K3b спочатку розраховує суму MD5 ISO-образу (займає деякий час). Збіг розрахованого значення MD5 із значенням перевірочної суми MD5 файл, збережений при завантаженні образу і знаходиться в тій же самій папці (наприклад, aptosid-Імя.iso.md5) означає, що завантаження пройшла успішно і файл може бути записаний простим клацанням на кнопці "Почати ".

Натиснувши правою кнопкою миші на рядку, що містить розраховану суму, виберіть пункт "compare checksums", це викличе додаткове вікно, в яке можна вручну ввести відому MD5 суму для порівняння.

Проблеми з пропаленням головним чином трапляються через зовнішнього інтерфейсу. Ви можете пропалювати безпосередньо з консолі за допомогою burniso.
