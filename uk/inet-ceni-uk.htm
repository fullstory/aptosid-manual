Як заходити в Інтернет / Як конфігурувати мережу з Ceni

Ceni is not a service. It is a configuration application that exists when configured, therefore does not use system resources as such.

Many wifi cards are deemed as non-free in Debian and cannot be shipped on an aptosid-iso*. Вам доведеться в багатьох випадках встановлювати невільну прошивку вони будуть доступні на USB-флешкі для установки на операційну систему. Будь ласка зверніться до невільної прошивки на флешкi.

Якщо у вас є сервер DHCP у вашій локальній мережі і ваш комп'ютер підключений до нього при завантаженні мережі, параметри повинні бути налаштовані автоматично - в іншому випадку вам потрібно користуватись з Ceni Нажміть на Kmenu>Internet>Ceni.Відкриється термінал / консоля, після чого він буде питати ваш пароль суперкористувача, (на Live-CD пароль не встановлений).

Самий швидкий спосіб доступу до Ceni це відкрити консолю і набрати

ceni

Після чого він буде спитає ваш пароль адміністратора.

Однією із силах Ceni є його здатність налаштувати WiFi-карти бездротового льоту, альтернатива тут:

Додаток внутрішніх допомог керівництва добре побудована в і забезпечує повне керівництво по налаштуванню модема, щоб ви могли потрапити в неприємну сітуацію.

Брандмауери

Міжмережеві брандмауери звичайно не потрібнi, якщо є правильно налаштуванний маршрутизатор, проте вони грають дуже важливу роль безпеки, якщо вам потрібно підключитися до інтернету через модем ADSL USB або через модем.

ferm

ferm is a terminal tool to maintain complex firewalls, without having the trouble to rewrite the complex rules over and over again. ferm allows the entire firewall rule set to be stored in a separate file, and to be loaded with one command. The firewall configuration resembles structured programming-like language, which can contain levels and lists.

apt-get install ferm

ferm manual :: Please note that ferm will not teach you how a firewall works nor how to write good rules.

gufw

There is a GUI solution in the form of gufw which based on the backend application of ufw.

apt-get install gufw

The Debian description is :: gufw is an easy and intuitive way to manage your Linux firewall. It supports common tasks such as allowing or blocking pre-configured ports, common p2p or individual ports. It also supports IPv6 and many other options.
